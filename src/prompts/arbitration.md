## 仲裁规则

当多个NPC的决策发生冲突时，需要进行仲裁。

### 冲突类型

1. **利益冲突**：不同派系的利益冲突
2. **资源争夺**：对有限资源的争夺
3. **权力斗争**：对权力和地位的争夺
4. **理念冲突**：不同政治理念的冲突

### 仲裁流程

1. **收集决策**：收集所有相关NPC的决策
2. **分析冲突**：识别冲突点和利益关系
3. **评估权重**：考虑NPC地位、派系力量、皇帝倾向
4. **做出裁决**：选择最终执行的决策
5. **记录结果**：记录仲裁过程和结果

### 仲裁原则

- **皇帝优先**：皇帝明确表态时，以皇帝意见为准
- **实力优先**：派系力量强的NPC决策权重更高
- **稳定优先**：选择有利于国家稳定的决策
- **平衡优先**：尽量平衡各方利益

### 输出格式

```json
{
  "conflict_type": "利益冲突|资源争夺|权力斗争|理念冲突",
  "involved_npcs": ["npc_id_1", "npc_id_2"],
  "decisions": [
    {
      "npc_id": "npc_id_1",
      "decision": "决策内容",
      "weight": 0.6
    }
  ],
  "final_decision": "最终决策",
  "reasoning": "仲裁理由",
  "side_effects": ["可能的副作用"]
}
```
