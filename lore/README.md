## 核心设定摘要

### 世界观基调
- **朝代**：靖朝（永德年间）
- **风格**：权谋向互动小说，类似《大明王朝1566》
- **基调**：写实向，道教是政治工具，不涉及真正法术

### 核心冲突
1. **皇帝 vs 国师**：理性 vs 神秘、证据 vs 信仰
2. **皇帝 vs 各派系**：权力与利益的博弈
3. **皇帝 vs 自我**：现代人格与古代世界的不兼容
4. **国师 vs 自我**：与时代、师父、自身观念的不兼容

### 核心角色
- **张猛**：情感驱动的武将（德米特里型）
- **谢渊**：极度理性的谋士（伊万型）
- **李静斋**：理想主义的年轻官员（阿廖沙型）
- **国师玄明**：贝阿朵莉切式四重人格（核心对手）

### 政治派系
- 清流派、帝党派、军队派、宦官派、务实派

## 注意事项

1. **禁用词汇**：避免使用"魔法"、"魔女"等与古代设定冲突的词汇
2. **用词规范**：道教相关使用"道术"或"神迹"
3. **四重人格名称**：天真的道士、残酷的道士、游戏的道士、脆弱的道士
4. **朝代年号**：统一使用"靖朝"和"永德"年号
