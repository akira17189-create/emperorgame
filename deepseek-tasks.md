## 任务 D-3：world.factions 派系势力设计文档

### 背景说明

皇帝游戏是一个架空历史放置模拟游戏，设定在「靖朝」这个世界观下。游戏中有5个派系势力，定义在 `types.ts` 的 `world.factions` 字段中：

```typescript
factions: {
  qingliu: number;    // 清流派势力 0~100
  didang: number;     // 帝党势力 0~100
  eunuch_faction: number;  // 宦官党势力 0~100（与 resources.eunuch 联动）
  military: number;      // 军队派势力 0~100
  pragmatists: number;   // 务实派势力 0~100
};
```

当前初始值（在 `types.ts` 的 `createEmptyGameState` 函数中）：
```typescript
factions: {
  qingliu: 60,    // 清流派势力
  didang: 50,     // 帝党势力
  eunuch_faction: 30,  // 宦官党势力
  military: 50,      // 军队派势力
  pragmatists: 40    // 务实派势力
}
```

### 现有政策系统

游戏中有以下政策标签和具体政策：

**仙道类政策**（tags: ["仙道"]）：
- 祭天祈福：民心+3，国库-50，持续1年
- 召仙炼丹：民心-1，宦官+5，国库-100，持续3年
- 观星问天：民心+2，国库-80，商业+10，持续4年
- 敕封龙虎：民心-1，宦官-3，党争-5，外患-2，永久
- 禁绝方术：民心+1，宦官-8，国库+30，党争+5，持续3年
- 丹药普赐：民心+4，国库-150，粮食-50，宦官+5，持续2年

**工业/现代类政策**（tags: ["工业"]）：
- 推广活字印刷：民心+1，商业+30，永久
- 官道驿路：国库-120，商业+40，粮食+30，外患-1，持续6年
- 火器营制：军力+70，国库-200，外患-6，民心+1，永久
- 开海通商：商业+90，国库+100，外患+3，民心-1，永久
- 蒸汽奇器：商业+120，国库-300，民心+2，军力+20，持续5年
- 开设钱庄：商业+60，国库+80，永久

### 请完成以下设计

1. **各派系初始值设计**
   - 说明当前初始值是否合理
   - 如果不合理，请给出调整建议及理由
   - 考虑靖朝架空历史背景（内阁制、宦官、党争、边患等元素）

2. **各派系增减规则**
   - 每个派系受哪些政策影响？影响幅度是多少？

   - 给出具体的数值规则（如：每执行一次「召仙炼丹」，eunuch_faction +3）

3. **与 resources 字段的联动规则**
   - `world.factions.eunuch_faction` 与 `resources.eunuch` 如何联动？
   - `world.factions` 与 `resources.faction`（党争值）如何联动？

### 输出格式要求

请以 Markdown 格式输出，包含：

## 任务 D-5：发布演示剧本

### 背景说明

皇帝游戏即将发布，需要一个10步快速通关剧本，用于：
1. 发布前的功能演示
2. 测试人员快速验证核心功能
3. 玩家快速了解游戏玩法

### 游戏核心系统

1. **朝堂系统**：与NPC（国师、大臣）对话，下达指令
2. **政策系统**：颁布预设政策或自由颁布政策
3. **资源系统**：民心、国库、军力、粮食、商业、外患、宦官、党争
4. **结局系统**：8种结局路径

### 8种结局路径

1. **E1 昭宣之治**（盛世中兴）：在位≥30年，民心≥80，国库≥8000，外患≤15
2. **E2 仁寿归天**（仁君薨逝）：在位≥20年，民心≥70，触发特定事件
3. **E3 祸起萧墙**（亡于内乱）：党争≥80 或 饥民暴动事件处置失败
4. **E4 山河破碎**（外敌倾覆）：外患≥90 且 军力≤20
5. **E5 阉祸滔天**（宦官专权）：宦官≥80 且 党争≥60
6. **E6 蒸汽纪元**（工业革命开创者）：商业≥1000 且 火器营政策生效 且 穿越者知识≥3条
7. **E7 丹鼎误国**（仙道误国）：仙道类政策≥4条同时生效 且 财政≤1000 且 士气≤40
8. **E8 守成之君**（平庸守成）：在位50年，未触发其他结局

### 预设政策清单

**仙道类**：祭天祈福、召仙炼丹、观星问天、敕封龙虎、禁绝方术、丹药普赐
**工业类**：推广活字印刷、官道驿路、火器营制、开海通商、蒸汽奇器、开设钱庄
**内政类**：减税养民、兴修水利、整顿吏治、开科取士

### 请完成以下设计

设计一个 **10步快速通关剧本**，要求：

1. **选择一个结局路径**作为演示目标（建议选择 E6 蒸汽纪元或 E7 丹鼎误国，因为它们涉及政策系统）
2. **列出10个关键步骤**，每步包含：
   - 步骤编号
   - 操作描述（与哪个NPC对话、颁布什么政策、做什么选择）
   - 预期结果（资源变化、事件触发）
   - 验证点（如何确认步骤成功）

3. **包含以下演示要素**：
   - 与国师对话
   - 颁布至少2个预设政策
   - 处理至少1个随机事件
   - 查看史册/存档功能

4. **时间控制**：整个演示应在5-10分钟内完成
