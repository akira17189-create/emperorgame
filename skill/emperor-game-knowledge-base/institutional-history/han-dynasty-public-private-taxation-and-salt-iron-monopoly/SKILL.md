# 汉代公私分税制度与盐铁国营政策

## 适用场景
- 解释汉代“山林池泽归皇室、田租归国家”的财政划分逻辑
- 分析盐铁商人与政府关系恶化的原因
- 判断何时触发资源国有化政策（如军费激增、大司农财源枯竭）
- 对比《盐铁论》中民间与官方对垄断政策的立场

## 执行步骤

1. **确认制度前提**：
   - 封建制度已瓦解，耕地归农民私有
   - 山林池泽被视为皇室公产（非国家公共财产）
   - 战国以来山泽开放征税，形成少府收入来源

2. **识别财政结构**：
   - 大司农收入 = 耕地田租 → 政府公共财政
   - 少府收入 = 盐、铁、渔猎、伐木等山泽税 → 皇室私人用度

3. **评估经济变迁影响**：
   - 盐铁因属全民必需品，利润剧增
   - 少府收入逐步超过大司农，打破原有平衡

4. **判断政策触发条件**：
   - IF 山海池泽税收（少府） > 田租（大司农）
   - AND 政府面临重大财政危机（如汉武帝讨匈奴、通西域）
   - THEN 启动资源国有化程序

5. **执行国有化措施**：
   - 皇帝先捐出少府收入支援国家
   - 号召盐铁商人乐捐，若响应不足则收回经营权
   - 设立官营机构自行煮盐冶铁，利润全归政府
   - 扩展至酒类官卖（因其耗粮且属奢侈品）

6. **理解政策性质与争议**：
   - 此为早期“国家社会主义”实践，主导中国至清代
   - 汉昭帝时召开盐铁会议，《盐铁论》记录正反辩论

> 注意：本技能不适用于文帝免税期间或耕地税收问题。
