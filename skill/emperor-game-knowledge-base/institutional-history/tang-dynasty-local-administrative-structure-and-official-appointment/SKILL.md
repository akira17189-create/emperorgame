# 唐代地方行政层级与官员任用制度

## 适用场景
- 用户询问唐代州县如何划分等级
- 需要解释唐代地方官员为何频繁流动、缺乏稳定性
- 比较唐代与汉代地方治理模式差异
- 分析中央集权对地方行政效率的影响

## 核心执行流程

1. **确认行政层级结构**
   - 基础为二级制：县 → 州
   - 安史之乱后，观察使等使职介入，实际演变为三级制

2. **判断州县等级**
   - **县分级依据户数**：
     - 上县：≥6000户
     - 中县：3000–5999户
     - 下县：<3000户
   - **州分级依据户数**：
     - 上州：≥100,000户
     - 中州：20,000–99,999户
     - 下州：<20,000户

3. **核查官员任用机制**
   - 州县长官无权自行辟署属官
   - 所有地方官员均由中央吏部统一任命

4. **分析升迁与治理影响**
   - 升迁需逐级（下→中→上），路径冗长，实质晋升困难
   - 官员普遍存在“五日京兆”心态，不安其位
   - 导致行政效率低下，与汉代形成鲜明对比

5. **对比汉代制度（关键差异点）**
   - 汉代郡守可自辟属官，三年一考绩
   - 汉代阶级少、升转灵活，官员“各安于位”

> 注意：本制度不适用于汉代或唐代以前的地方自治模式分析。
