# 唐代科举制度运作与分析

## 何时使用
- 用户询问唐代士人入仕路径
- 需要解释科举如何打破门第限制
- 比较中国古代与西方文官选拔机制
- 分析科举制度的公平性与流弊

## 核心操作流程

### 1. 确认报考资格
- **适用对象**：一般士人，尤指寒门子弟
- **身份限制**：不得为商人或工人（因其“专为私家谋利”）
- **资格要求**：须“身家清白”（无犯罪记录等）
- **报名方式**：实行“怀牒自列”，即自行向地方政府报名，无需地方察举或中正品评

### 2. 执行两阶段考试
- **第一阶段（礼部试）**：
  - 主持机构：尚书省礼部
  - 主要科目：进士、明经等
  - 考核重点：才学
  - 结果：及格者称“进士及第”，获得做官资格
- **第二阶段（吏部试）**：
  - 主持机构：吏部
  - 考核内容：仪表、口试、行政公文处理能力（即“干练”）
  - 结果：通过后方可实际任用

### 3. 分析制度意义与问题
- **历史进步性**：
  - 打破门第垄断，实现“开放政权”
  - 相较汉代乡举里选，范围更广、自由度更高
  - 体现“考试代替选举”的中国传统用人逻辑
- **结构性矛盾**：
  - 报考人数远超录取名额，导致“士十于官，求官者十于士”
  - 政府被迫扩编，设员外官、候补官
  - 知识分子过度集中于仕途，造成“政治脂肪”与政权臃肿

### 4. 进行中西对比
- **中国模式**：“一切用人全凭考试和铨叙”，强调客观标准
- **西方模式**：政务官靠选举，事务官靠考试

> 注意：本技能不适用于九品中正制时代（魏晋南北朝）的选官分析。
