# 宋代国防地理劣势与战略被动性分析

## 何时使用
- 用户询问宋代为何长期采取守势而非主动进攻
- 需解释北宋“积弱”现象的地缘政治根源
- 分析开封作为都城的战略风险
- 探讨宋代骑兵短缺与边防体系脆弱性的关联

> **注意**：本技能不适用于南宋江淮防线分析，亦不适用于讨论主动进攻型战略（如熙河开边）。

## 执行步骤

1. **确认地理前提条件**：
   - 燕云十六州已失于辽（今河北北部、山西北部、辽宁西部、内蒙古南部）
   - 都城设于开封（黄河平原，无天然屏障）
   - 缺乏优质产马地（东北蓟北之野、西北甘凉河套均被敌控）
   - 国家财政依赖南方漕运

2. **评估地理防御缺陷**：
   - 指出华北平原无险可守，敌军骑兵三日可达开封
   - 对比洛阳（有京索山、雁门关为二线）或长安（更利防守）的潜在优势
   - 强调拒马河（涿州附近）为唯一主要防线，突破即危及京师

3. **分析军事资源制约**：
   - 说明内地养马效率低下（1匹马占地≈25人粮田，且易病亡）
   - 解释“保马法”失败原因（民间赔马负担重，反成弊政）

4. **揭示防御手段的被动性**：
   - 描述澶渊之盟后不敢明设边防的困境
   - 列举“种水田、开沟渠、植榆杨”等非正式阻骑措施
   - 指出辽军冬季袭扰破坏宋朝边防建设的战术

5. **总结战略恶性循环**：
   - 因地理劣势 → 不敢进攻 → 长期养兵 → 兵将分离 → “进不可攻，退不可守”
