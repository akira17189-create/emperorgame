# 宋代君权提升与相权低落的表现分析

## 何时使用
- 用户研究宋代政治制度演变
- 分析君权与相权关系的历史转折点
- 探讨五代乱世后政权重建中的“尊王”策略
- 比较唐宋两代宰相地位与朝仪差异

## 如何执行
1. **确认历史背景**：判断是否处于宋代初期（尤其是太祖、太宗时期），且政权刚从五代乱世中建立。
2. **识别核心动因**：
   - 五代皇帝频繁更替导致皇权威信受损
   - 宰相（如冯道）长期任职反显稳定，削弱君主权威
   - 宋太祖出身武将，需强化皇权合法性
3. **分析朝仪制度变化**：
   - 对比唐代：宰相可坐论政、赐茶，体现“三公坐而论道”传统
   - 观察宋代：宰相主动“逊让不坐”，以示对新君忠诚
4. **解读深层心理与社会因素**：
   - 晚唐五代进士出身官员存在“小家样读书人”的自卑心理
   - 不同于唐代门第士族或西汉朴讷之士的气度
5. **评估历史后果**：
   - 初衷为“尊王”以拨乱返治
   - 后世遗忘本意，仅见“皇帝尊严”与“宰相卑微”的固化印象

> 注意：本技能不适用于唐代及以前朝代的朝仪制度分析。
