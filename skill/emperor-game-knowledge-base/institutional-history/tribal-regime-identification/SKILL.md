# 部族政权识别与分析

## 何时使用
- 用户提及元代、清代或其他非汉族主导的政权结构
- 需判断某政权是否以特定部族整体利益为核心
- 分析政策背后是否存在“私心”而非“公义”动机
- 对比士人政权（如汉唐宋明）与异族统治政权的制度差异

## 执行步骤

1. **确认前提条件**：
   - 是否存在一个明确的统治部族（如满洲、蒙古）
   - 该部族是否整体支持并实际掌控政权

2. **应用触发条件**：
   - 若政权由某一特定部族集体掌握并为其利益服务，则判定为部族政权

3. **识别核心特征**：
   - 政权由整个部族支撑，而非皇帝个人独裁
   - 权力垄断于该部族成员，形成特权阶级
   - 所有政策优先维护部族统治安全，而非国家整体利益

4. **辨析与士人政权的区别**：
   - 士人政权：通过科举选拔官员，无世袭特权，理论上代表全民
   - 部族政权：特定民族垄断权力，制度表面延续传统，实则夹带私心

5. **观察运作表现**：
   - 表面维持传统政治理论（如重用读书人）
   - 实际通过“法术”凌驾制度（如满汉双设官职、秘密上谕）
   - 政治过程不公开、不透明，民众参与受限

6. **输出警示结论**：
   - 明确指出该政权属于部族政权
   - 警示其政策可能出于部族私利，而非公共福祉

> 注意：本技能不适用于汉族主导的士人政权（如汉、唐、宋、明）。
