# 区分制度与法术

## 何时使用
- 分析某一朝代或政权的政治结构是否具备制度性
- 评估某项治理措施是出于公心（国家整体利益）还是私心（特定集团利益）
- 比较不同历史时期治理模式的正当性与可持续性
- 用户提及“清代是否为制度”“汉唐与宋明治理差异”等具体历史判断场景

## 如何执行
1. **确认适用对象**：仅适用于存在公共治理结构且有明确职责划分的政治体系、历史政权或治理措施；不适用于纯粹个人行为或临时权宜之计。

2. **应用触发条件**：
   - 若某项政治安排**出自公心且具普遍适用性** → 判定为**制度**
   - 若**出自私心且无固定节限** → 判定为**法术**

3. **执行四步区分法**：
   - **动机判断**：考察主导动机是“公心”（以国家整体利益和职责为出发点）还是“私心”（以特定部族、集团或个人利益为出发点）
   - **稳定性检验**：制度具有稳定性、可预期性；法术多为临时、权宜手段
   - **普遍性检验**：制度具有一般性和可重复适用性；法术常为特例，不可复制
   - **透明度检验**：制度需公开透明；法术常涉秘密或非正式操作

4. **输出结论**：明确指出该政治安排属于“制度”或“法术”，并简要说明依据。

> 注意：本技能基于钱穆《中国历代政治得失》等传统政治哲学框架，适用于对中国帝制时代治理逻辑的历史分析。
