## 紫檀木建材规格

### 板材要求

```
单块墙板规格：
- 厚度：1寸（3.2cm）—— 保证结构强度和隔音
- 宽度：2尺（64cm）—— 减少拼缝，提升整体感
- 高度：2丈（6.4m）—— 通高无接缝，彰显奢华

用量估算（四面墙）：
- 长墙（9丈）：每面约需 14-15 块板
- 短墙（5丈）：每面约需 7-8 块板
- 总计：约44-46块整板
```

### 紫檀木特性

| 特性 | 说明 | 设计利用 |
|------|------|---------|
| 密度 | 极高（沉水） | 墙体厚重，隔音优异 |
| 硬度 | 极高 | 结构稳固，耐久数百年 |
| 色泽 | 深紫褐至黑紫 | 内部营造沉稳奢华氛围 |
| 香气 | 淡淡檀香 | 天然香氛，无需熏香 |
| 价格 | 寸木寸金 | 仅限内部使用，外部不可见 |

### 乌木衣架规格

- 材质：乌木（仅次于紫檀的顶级硬木）
- 形制：整排沿墙设置
- 功能：陈列丝绸女装，兼具装饰与实用

## 施工技术要点

### 紫檀墙板安装

1. **基础处理**：夯土地基上铺设石板防潮层
2. **立柱定位**：四角及中部设暗柱（外包紫檀）
3. **板缝处理**：榫卯咬合，缝隙以紫檀木屑调胶填充
4. **表面打磨**：细砂纸逐次打磨至镜面光泽

### 隔音强化

- 墙体双层结构：外层紫檀+内层夯土芯
- 或采用"夹墙"做法：双板中空，填充丝绸/棉絮
