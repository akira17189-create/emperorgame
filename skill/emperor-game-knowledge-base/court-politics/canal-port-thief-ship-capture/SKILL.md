# 运河码头贼船强制缉拿程序

## 适用场景

当满足以下条件时启动本程序：
- 发现贼船或高度怀疑的违法船只
- 目标船只位于运河码头区域
- 存在船只解缆逃离的风险
- 已获得上级缉拿授权或拥有紧急处置权

## 前置条件

- 官军配备长枪、火铳等武器
- 明确目标船只位置（如"靠左边那十几条粮船"）
- 码头区域有可供包围的空间条件

## 执行步骤

### 1. 兵力配置与装备检查

- 派遣由队官（挎刀指挥人员）率领的官军小队
- 武器配置必须包括：
  - 长枪：用于近距离控制和威慑
  - 火铳：用于远距离威慑和必要时射击
- 确保武器处于可用状态，队官携带指挥刀便于现场调度

### 2. 快速战术接近

- 以跑步方式快速接近码头目标区域，保持战斗队形
- 根据情报明确具体目标位置
- 行动需迅速以防止目标船只解缆逃离

### 3. 包围与控制

- 队官大声吆喝明确指令："走！快点！就是靠左边那十几条粮船，围住，不要让他们跑了！"
- 实施物理包围战术，封锁船只所有可能离港路径（前后甲板、缆绳固定点）
- 持长枪士兵负责近距离封锁，持火铳士兵负责外围警戒和威慑

### 4. 强制清场与疏散

- 高声呼喝警示平民："闪开！"、"抓贼船的！都闪开了！"
- 对码头上的运工、商人及货物实施强制疏散，无论是否阻碍行动
- 对未立即避让者采取物理冲撞措施强行通过，不保证人员货物安全

### 5. 后续控制

- 完成包围后实施登船检查或等待上级指令
- 维持现场秩序，防止围观群众干扰

## 约束与警告

- **仅适用于确认或高度怀疑的贼船/违法船只**
- **不适用于正常商船检查**（需用常规巡检程序）
- 需避免误伤朝廷官船或漕运粮船
- 人员密集时可能造成附带损伤（运工、货物被撞倒）
- 此程序具有高度强制性，允许对正常商业活动造成中断和物理损害
