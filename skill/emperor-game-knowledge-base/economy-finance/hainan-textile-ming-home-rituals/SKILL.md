# 海南纺织传统与明代家居入室礼仪

## 适用场景
- 雪天归家，需执行入室清洁程序
- 讨论纺织技术起源及黄道婆历史
- 明代家庭居室空间布局认知
- 海南籍人士家庭传统实践

## 执行步骤

### 1. 入室前清洁程序

**触发条件**：归家时遇雪

1. 取下门框上挂着的笤帚
2. 先为同行家人掸去头上和身上的雪
3. 再自我清洁，掸去自身雪迹
4. 确保不将雪水带入室内

### 2. 脱鞋赤脚规范

**约束条件**：必须赤脚入室

1. 完成掸雪后，脱下鞋袜
2. 赤脚进入屋内
3. 此为海南籍人士的习惯或特定家庭传统

### 3. 纺织技术溯源

**适用对象**：纺织从业者、讨论技术起源时

1. 认知海南地区在棉纺织技术上的历史优势
2. 理解黄道婆在海南学习纺织技术后传播至内地各省的历史路径
3. 传承海南纺织文化认同

### 4. 织机操作认知

**关键指标**：
- 织机类型：木制织机
- 进度标识：机头露出约三寸布头
- 生产效率：半天时间完成一匹布

### 5. 空间布局认知

**明代家庭居室功能分区**：
- 东面墙：摆放织棉布的木机
- 西面：书房/更衣处
- 厨房：独立于主屋

## 关键变量

| 变量名 | 类型 | 描述 |
|--------|------|------|
| 清洁工具位置 | string | 门框上挂着的笤帚 |
| 纺织历史渊源 | string | 黄道婆在海南学习后传入内地 |
| 织布进度标识 | length | 机头露出布头约三寸 |
| 入室足部状态 | categorical | 赤脚(脱鞋脱袜) |

## 预期结果

- 保持室内清洁（无雪水）
- 传承海南纺织文化认同
- 明确家庭空间功能分区（东墙织机/西屋书房）

## 注意事项

- 本技能适用于特定地域传统（海南）
- 雪天必须先掸雪，赤脚入室是强制要求
- 纺织技术历史需结合黄道婆传播路径理解
