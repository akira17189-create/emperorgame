# 历史案例：高拱辩“奸”

## 背景
发生在玉熙宫外间大殿的对峙场景。高拱面临严世蕃等人的政治指控，被指责为“奸臣”。

## 具体应用

### 指控字词
“奸”

### 拆解逻辑
- **结构分析**：“奸”字由“女”字旁和“干”字组成。
- **语义关联**：将“女”字旁与女性关系联系起来，暗示该字义与男女关系混乱有关。

### 事实对比
- **自身情况**：高拱陈述自己仅有一个“糟糠之妻”（指共患难的发妻），以此证明自身在男女关系上的清白与专一。
- **对手情况**：指出对手严世蕃拥有众多女性伴侣，例如“就在昨天你才娶了第九房姨太太”。

### 结论输出
“这个‘奸’字，恐怕加不到我高拱身上。”

## 效果
通过字义与事实的巧妙结合，不仅驳斥了指控，还利用对手的私德瑕疵进行了有力反击，化解了政治危机。
