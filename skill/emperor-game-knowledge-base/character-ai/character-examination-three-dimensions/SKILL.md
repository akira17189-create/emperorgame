# 故事人物审视的三个维度

## 适用场景
当需要分析或理解故事人物的呈现方式、人物在叙事结构中的定位时，使用此框架进行分类审视。

## 三个审视维度

### 维度一：不同故事型态中的卡司
- **关注点**：从故事类型的角度出发
- **分析内容**：审视人物在卡司（角色阵容）中的定位与构成
- **关键问题**：
  - 该人物在故事类型中扮演什么角色？
  - 人物在角色阵容中的定位是什么？

### 维度二：人物在一个故事里采取的行动
- **关注点**：从情节发展的角度出发
- **分析内容**：审视人物在故事内部所实施的具体行动
- **关键问题**：
  - 人物在故事中做了什么？
  - 这些行动如何推动情节发展？

### 维度三：从读者与观众的观点来看故事人物
- **关注点**：从受众接收的角度出发
- **分析内容**：审视读者与观众如何感知和看待故事人物
- **关键问题**：
  - 受众如何理解这个人物？
  - 人物给受众留下了什么印象？

## 使用步骤
1. 确定分析的故事型态
2. 从卡司定位角度分析人物
3. 从行动表现角度分析人物
4. 从受众视角角度分析人物
5. 综合三个维度得出人物全面画像
