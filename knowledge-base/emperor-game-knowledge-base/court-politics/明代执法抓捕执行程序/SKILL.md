# 明代执法抓捕执行程序

## 适用场景

本程序适用于以下抓捕任务：
- 锦衣卫执行皇帝圣旨抓捕高级官员
- 提刑司、镇抚司围府抓捕政治人物
- 县衙差役缉拿欠债百姓
- 需要保护特定对象的拦截行动

## 执行层级与授权

### 高层政治抓捕（锦衣卫/提刑司）

**授权来源**：
- 皇帝圣旨（最高授权）
- 司礼监掌印太监旨意（子时发布）
- 内阁票拟批红

**执行单位**：
- 锦衣卫（指挥使朱七等）
- 提刑司
- 镇抚司

### 基层执法抓捕（县衙）

**授权链条**：
- 浙江巡抚(赵中丞) → 县丞(二老爷/田有禄) → 差役班头(赵班头) → 差役执行
- 必要时可调配牢卒协助

**执行单位**：
- 县衙差役
- 狱卒（临时调配）
- 县丞（现场指挥）

## 核心执行步骤

### 第一步：确认目标与性质

明确抓捕对象：
- 主要目标：严世蕃、罗龙文、鄢懋卿等
- 保护对象：高拱、张居正等清流官员
- 基层对象：持有生丝的欠债百姓

确认任务性质：
- "干犯天条"（政治罪名）
- 欠织造局官粮需以生丝抵偿
- 执行皇帝圣旨

### 第二步：现场控制与围堵

**围府抓捕**：
- 立即包围目标府邸
- 严格约束：一个人一样东西都不许放走
- 提刑司、镇抚司各分小队执行

**现场对峙处理**：
- 进入现场后先向各方见礼（如朱七向严世蕃拱手："严大人"）
- 无视目标人物的指令意图
- 保持中立态度，不与任何一方结盟

### 第三步：宣读并执行命令

**锦衣卫执行圣旨**：
1. 正式宣布："奉旨，着即将严世蕃押送回府，听旨发落。严大人，跟小的走吧"
2. 关键措辞分析：
   - "奉旨"——强调行动合法性来源为皇帝
   - "着即将"——命令的紧迫性，立即执行
   - "押送回府"——指定地点，非诏狱而先回府，留有余地
   - "听旨发落"——最终处置权在皇帝
   - "跟小的走吧"——表面谦卑，实则命令不可违抗

**县衙缉拿百姓**：
- 使用锁链锁人："锁了！都锁了！一个也不要让他们跑掉！"
- 物理控制：拽住衣领、掰着手臂
- 财物扣押：抢夺装生丝的包袱
- 人员处置：生丝送衙门，人押送大牢

### 第四步：应对抗拒与特殊情况

**应对官员抗拒**：
- 目标人物反应："还在惊疑"，"睁大了两眼直盯着"
- 重复并施压："严大人，小人们的任务是奉旨办差，请不要为难我们，跟我们走吧"
- 肢体语言配合：伸出蒲扇大的手掌向门外一让——既礼貌又具压迫感
- 处理抗议：如严世蕃边被押走边喊"我要见皇上！有奸臣，我要见皇上！"

**拦截保护行动**：
- 背景：严世蕃等人正领着刑部和大理寺的人去捉拿忠臣
- 行动：提刑司、镇抚司各分一个小队前往高拱和张居正府第
- 任务：把罗龙文、鄢懋卿拿了，送回到他们自己家里去看押起来

### 第五步：押送离场与人员遣散

**押送配置**：
- 前导锦衣卫开道
- 指挥使（朱七）贴身监视
- 后卫锦衣卫跟随
- 形成前后夹击之势，确保目标无法逃脱

**遣散相关人员**：
- 处理遗留官兵："是哪个衙门的就回哪个衙门去。大过节的瞎搀和什么"
- 暗示事件发生在节庆期间，官兵介入本属不当
- 队官传令整队撤离

### 第六步：为后续行动创造条件

- 锦衣卫离场后，清流官员（如张居正）立即接管现场
- 为保护对象创造行动空间（如张居正护送高翰文夫妇离京）
- 确保政治平衡不被打破

## 执行规范与禁忌

### 术语规范

明代称缉拿为"拿"而非"捉"：
- 体现"把人当做东西，去拿便是"的执法理念
- 被缉拿者被视为物件而非活人

### 执行场景

- 可在街市公开进行，无需避忌早市人群
- 导致"街面上立刻乱了"，"做其他生意的百姓都惊了，一个个拎着自己要卖的东西四处奔散"

### 节日惯例

**正月十五不抓人**：
- 传统上元宵节停止刑狱
- 官员可援引此惯例试图阻止搜捕
- 但权臣可无视此惯例（如严世蕃："正月初一老子还杀过人呢"）

### 权力展示

- 权臣自称"老子"而非官职称谓，体现跋扈姿态
- "来人！"的即时命令显示对军队的直接控制
- 无视节日禁忌，直接下令搜捕

## 关键变量

- **授权来源**：皇帝圣旨、司礼监旨意、巡抚命令
- **执行头目**：锦衣卫指挥使、提刑司官员、县丞、差役班头
- **协助人员**：锦衣卫、镇抚司、牢卒
- **缉拿对象**：政治人物、欠债百姓
- **扣押财物**：生丝包袱、其他财产
- **现场态势**：对峙、围堵、控制

## 特殊情况处理

### 保护对象拦截

当需要保护特定对象时：
- 提前派遣小队前往保护对象府第
- 拦截抓捕人员
- 将抓捕人员送回自家看押
- 避免保护对象被陷害

### 节日禁忌突破

当权臣无视节日惯例时：
- 以更极端的先例压制对方（如正月初一杀人）
- 表明权臣不受常规礼法约束
- 直接下令执行，无视抗辩

## 成功标准

- 目标人物被成功控制并押送至指定地点
- 保护对象得到有效保护
- 现场秩序得到控制，未发生大规模冲突
- 执行过程符合授权范围和程序规范
- 为后续政治行动创造有利条件
