# 涉及织造局机密账册的处置规程

## 适用场景

- 查抄获得账册，且内容涉及织造局或皇帝私人利益
- 发现财务记录与沈一石等织造局相关人员有关
- 需要处理可能涉及皇权私产的敏感文件

## 执行步骤

### 1. 识别账册性质
确认账册内容是否涉及"织造局"或皇帝的私人利益（如沈一石账册）。

### 2. 确定保密级别
此类账册属于"除了皇上谁也不能看"的最高机密，任何非授权查阅都是"犯了大忌"。

### 3. 禁止停留与查阅
严禁将账册留在任何官员（包括内阁首辅）处过夜或查阅，必须拒绝接收。

### 4. 立即转运
必须立刻将账册运出私宅，送往宫中朝房。

### 5. 呈送时机
赶在皇帝召见（如卯时）之前，将账册直接呈送宫中，向皇帝如实禀报。

### 6. 核心原则
织造局代表皇权私产，触碰此即触碰皇帝，必须物理隔绝非皇权接触。

## 关键约束

- 不得私自留存账册
- 不得向非皇帝人员展示账册内容
- 必须在皇帝召见前完成呈送

## 执行结果

将账册送往宫中朝房，等待呈送皇帝。
