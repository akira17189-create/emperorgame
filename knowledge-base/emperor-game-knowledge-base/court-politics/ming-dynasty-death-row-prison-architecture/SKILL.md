# 明代死囚牢房建筑规制与空间布局

## 适用场景
当满足以下条件时使用此技能：
- 需要设计或分析明代枭司衙门（如锦衣卫等特务机构）的死囚牢房
- 场景涉及地下牢狱结构（地牢）
- 需要关押死囚或重犯的高安全级别监狱
- 进行明代历史建筑复原或文学创作考据

## 建筑类型特征

### 基本结构
- **建筑形式**：地牢（地下牢狱结构）
- **照明方式**：火把人工照明，无自然采光
- **建筑材料**：全部采用巨大石材砌筑

## 空间布局规制

### 整体分区
```
┌─────────────────────────────────┐
│         北面区域                │
│    宽宽的通道（狱卒巡逻区）      │
├─────────────────────────────────┤
│         南面区域                │
│  粗铁栏杆分隔的独立牢房         │
└─────────────────────────────────┘
```

### 详细布局
1. **北面通道**
   - 宽阔的通行区域
   - 供狱卒巡逻和通行使用
   - 便于对南面牢房进行监视

2. **南面牢房区**
   - 使用粗粗的铁栏杆分隔
   - 每间牢房独立
   - 铁栏杆既实现隔离又保持可视性

## 功能设计逻辑

### 安全控制
- **南北对峙布局**：通道与牢房南北相对，便于狱卒全方位监视控制
- **铁栏杆隔离**：既防止囚犯逃脱，又保持可视性便于管理

### 防逃设计
- **巨石结构**：墙面、地面全部使用巨大石材，防止囚犯破墙越狱
- **地牢设计**：地下结构增加逃脱难度
- **隔音效果**：地下结构天然具备隔音功能

## 设计检查清单

在应用此建筑规制时，确认以下要素：
- [ ] 确认建筑对象为明代枭司衙门（非普通州县监狱）
- [ ] 采用地下结构设计
- [ ] 北面设置宽阔通道
- [ ] 南面设置铁栏杆分隔牢房
- [ ] 全部使用巨大石材建造
- [ ] 采用火把照明方式

## 约束条件
- 此规制仅适用于明代特定官署牢狱（如锦衣卫等特务机构）
- 不适用于普通州县监狱
- 需满足火把照明和石材建筑的前提条件
