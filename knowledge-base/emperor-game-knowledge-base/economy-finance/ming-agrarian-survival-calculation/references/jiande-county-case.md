# 建德县案例详解

## 背景

《大明王朝1566》中王用汲与郑泌昌的辩论场景，展示如何以数据驳斥"百姓还有余力"的论调。

---

## 基础数据

| 项目 | 数值 | 来源 |
|-----|------|------|
| 在籍人口 | 27万人 | 建德县册 |
| 总田亩 | 44万亩 | 官方统计 |
| 其中桑田 | 15万亩 | 经济作物 |
| 其中稻田 | 29万亩 | 粮食作物 |

---

## 正常年份计算

### 亩产与总产量

- 丰年标准：每亩产谷 **2.5石**
- 歉年标准：每亩不到 **2石**
- 取中间值估算：按丰年2.5石计算（实际可能更低）

稻田总产量 = 29万亩 × 2.5石 = **72.5万石**

### 人均摊算

```
人均稻谷 = 72.5万石 × 120斤/石 ÷ 27万人
         = 8700万斤 ÷ 27万
         = 322斤/人/年
```

> 注：原文"不足300斤"，取保守值或考虑部分桑田折算。

### 脱粒后净得

```
人均白米 = 322斤 × 0.83 ≈ 267斤
         ≈ 250斤（原文"不到250斤"）
```

### 日均口粮

```
日均白米 = 250斤 × 16两/斤 ÷ 365天
         = 4000两 ÷ 365
         ≈ 10.96两/天
```

**修正理解**：原文"不足7两"可能指：
- 实际亩产按歉年2石计算：250斤 → 约200斤白米 → 约8.8两/日
- 或考虑赋税预征、地主截留等，实际到手约7两

采用原文数据：**日均不足7两**

### 生存评估

> "老人孩童尚可勉强充饥，壮丁则远远不够"

- 7两 ≈ 218克白米（明代一两约31克）
- 现代标准：轻体力劳动需400-500克主食/日
- 明代壮丁徭役繁重，7两确实严重不足

---

## 灾害后：佃农模式计算

### 情景设定

- 分洪导致14万亩田被淹（约一半稻田）
- 百姓被迫卖田，次年租田耕种
- 身份转变：自耕农 → 佃农

### 稻田佃农（五五分租）

```
净得稻谷 = 2.5石 × 0.5 = 1.25石
         = 1.25 × 120 = 150斤

人均白米 = 150 × 0.83 ≈ 124.5斤

日均白米 = 124.5 × 16 ÷ 365 ≈ 5.46两
         ≈ 3.5钱（原文数据，或有其他折损）
```

原文明确：**3.5钱/日**

### 桑田佃农

桑田分租比例更苛刻（非五五）：
- 假设四六分租：净得40%
- 桑田亩产折算：需查当时丝价与米价

原文结论：**每天还不定有2.5钱**

### 生存对比

| 身份 | 日均口粮 | 状态 |
|-----|---------|------|
| 自耕农（正常） | ~7两 | 勉强维持 |
| 稻田佃农 | 3.5钱 | 严重不足 |
| 桑田佃农 | <2.5钱 | 生存危机 |

> "3.5钱米当然不够"——王用汲

## 副业补充的局限性

原文关键限定："**靠山有水**的地方"

### 有条件地区
- 种茶叶、桑麻
- 产桐油、漆
- 河里捞鱼虾

**用途**：卖了缴纳赋税，剩余换油盐购粗粮勉强度日

### 无条件地区
- 纯平原农耕区
- 无山林河湖资源
- 副业收入 ≈ 0

**政策含义**：不能假设所有百姓都有副业补充。

---

## 辩论中的数据运用

### 郑泌昌的论点
> "百姓还有余力，明年再改种桑苗也不迟"

### 王用汲的反驳链条

1. **建立基线**：正常年景已不足7两，壮丁不够
2. **展示灾害后果**：一半田被淹，百姓卖田
3. **量化身份转变**：自耕农→佃农，口粮腰斩至3.5钱
4. **致命一击**：3.5钱当然不够，何谈"余力"？
5. **上纲上线**：明知不可为而为之，是不顾百姓死活

### 计算的政治功能

将抽象的"民生艰难"转化为具体的数字对比：
- 7两 → 3.5钱：降幅50%
- 3.5钱 vs "一天口粮"：直观的不可能

数据成为道德指控的武器。

## 历史度量衡备注

| 项目 | 换算 | 说明 |
|-----|------|------|
| 明代1石 | 约120斤 | 容积单位，因谷物密度而异 |
| 明代1斤 | 16两 ≈ 596.8克 | 库平制 |
| 明代1两 | 约37.3克 | 但民间常用31克说 |
| 本文"斤两" | 可能为地方习惯 | 建议以相对比例为准 |

**建议**：使用原文给出的"两/钱"数值进行定性判断，避免过度精确换算。

## 扩展应用场景

### 类似计算需求

1. **其他明代县域**：替换人口、田亩数据
2. **不同租率对比**：三七、四六、五五分的差异
3. **赋税改革评估**：一条鞭法前后的实际负担
4. **作物改种影响**：稻改桑、粮改棉的收益风险

### 现代类比

- 自耕农 → 有产工薪阶层
- 佃农 → 租房打工者
- 卖田 → 卖房/耗尽储蓄
- 7两到3.5钱 → 收入腰斩后的生存线

此模型的核心洞察：资产（土地）的丧失导致生存能力的断崖式下跌，而非线性下降。
