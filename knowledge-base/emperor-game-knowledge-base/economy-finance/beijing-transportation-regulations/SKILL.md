# 北京九门以内行轿走马规制

## 适用场景
- 位于明朝北京九门以内区域
- 在通衢大街（如前门外大街）活动
- 需要判定交通工具使用权限

## 轿子乘坐规制判定

### 规则
- **非有品级的官员**（无品级或平民）：禁止乘坐四抬以上的轿子
- **有品级官员**：可使用四抬及以上轿子

### 判定流程
1. 检查官员品级
2. 若无品级或平民 → 禁止乘坐四抬以上轿子
3. 若有品级 → 可使用四抬及以上轿子

## 骑马规制判定

### 一般规则
- 即使有马也不能骑乘，只能牵着马步行

### 例外群体
| 群体 | 条件 | 权限 |
|-----|------|------|
| 步军统领衙门人员 | 执行巡行街道任务 | 可骑马 |
| 巡街御史 | 巡行街道 | 可骑马 |

### 判定流程
1. 确认人员身份
2. 检查是否在执行巡行任务
3. 若是步军统领衙门或巡街御史且在执行任务 → 可骑马
4. 其他情况 → 只能牵马步行

## 驰马特权判定

### 适用场景
在繁华大街（如前门外大街）敢于驰马者必须符合以下条件之一

### 特权条件
| 条件 | 说明 |
|-----|------|
| 条件A | 持有兵部勘合的急递人员（官方快递/紧急公文传递） |
| 条件B | 极有来头的要害人物（如皇帝钦封的第十三太保、司礼监太监等皇帝亲信） |

### 判定流程
1. 确认街道类型（是否为通衢大街）
2. 检查是否持有兵部勘合
3. 检查是否为极有来头的要害人物
4. 若符合条件A或条件B → 特许驰马
5. 若都不符合 → 视为违反规制

## 违规识别

### 违规情形
在热闹繁华的大街（如前门外大街）上：
- 既非持有兵部勘合的急递
- 又非极有来头的要害人物
- 却驰马奔跑

### 违规后果
- 视为违反规制
- 导致人群纷纷避让
- 引起骚动

## 判定结果模板

### 合规情况
- **轿子**：符合品级规制
- **骑马**：符合巡行任务特权
- **驰马**：持有兵部勘合或为要害人物

### 违规情况
- **轿子超规制**：无品级乘坐四抬以上轿子
- **违规骑马**：非巡行人员骑马
- **违规驰马**：无特权驰马

### 特许标记
- **特许驰马**：持有兵部勘合或为要害人物
- **特许乘轿**：有品级官员

## 约束条件
- 持有兵部勘合的急递人员除外
- 极有来头的要害人物除外
- 步军统领衙门和巡街御史执行巡行任务时除外
