# 宫廷礼仪规范管理

## 适用场景
- 锦衣卫执行公务时需要确定行礼方式
- 举行特殊廷议需要布置场地和安排座次
- 官员受审需要确定合法姿态
- 需要区分公私场合的礼制差异

## 锦衣卫跪拜礼制规范

### 跪拜对象判定

**必须跪拜的对象：**
- 皇上（皇帝）——唯一上跪对象
- 司礼监长官（如陈公公等司礼监太监）
- 镇抚司长官（锦衣卫内部上级）

**禁止跪拜的对象：**
- 其他各品级官员（无论品级高低，包括户部主事等）
- 仅需举手行礼（作揖或拱手）

### 私跪例外处理

**判定条件：**
- 下跪因个人恩情（如救命之恩、恩公关系）
- 非因对方官职

**处理原则：**
- 性质认定为"私跪"，与职分无关，不违反礼制
- 陪同人员不应跟随下跪
- 正确做法：侧身低头站在一边，表示回避但不参与

## 特殊廷议场地布置规范

### 场地改造要求

1. **移除常规设施：**
   - 移除大案（审判桌）和所有椅子

2. **改用坐垫布置：**
   - 地上摆放坐垫（蒲团）
   - 北墙大案位置改为4个坐垫（上首尊位）
   - 大堂两侧摆满一排排坐垫

### 入场顺序与座次等级

**第一批（内阁首辅及大学士级别）：**
- 徐阶、李春芳、高拱、赵贞吉
- 在北墙上首的四个坐垫就坐

**第二批（监察官员与文学侍从级别）：**
- 都察院御史、通政使司给事中、翰林院国子监文学之臣
- 排成两行鱼贯步入，在两侧坐垫就坐

**特殊座位安排：**
- 左侧第一排首位：国子监司业李清源（曾率领群臣上疏）
- 左侧第一排末座：王用汲（连夜赶到京师）

**太监群体：**
- 陈洪等太监不进入大堂
- 在大堂门口由太监摆下的椅子上就坐

## 官员受审姿态等级规则

### 法律依据
《大明律》审讯官员条例

### 品级判定与姿态对应

| 官员品级 | 受审姿态 |
|---------|---------|
| 三品以上（尚书、侍郎、内阁大学士等） | 可以坐着受审 |
| 三品以下（六品主事、七品知县等） | 可以站着受审 |

### 适用前提条件（必须同时满足）
1. 官员尚未定罪
2. 官员尚未被革职
3. 处于正式审讯阶段

### 法律规定与礼仪惯例的区分

**法律规定：**
- 基于《大明律》的品级规则
- 三品以下站着是法定权利，非屈辱

**礼仪惯例：**
- 即使未犯罪，六品小官面对内阁阁员及六部九卿堂官时
- 按官场规矩需行跪拜大礼

**执行要点：**
- 官员可按惯例跪拜行礼
- 行礼后应自己站起来
- 符合"三品以下站着受审"的法律规定
- 不应继续跪着或趴在地上受审

## 身份识别与应对流程

1. **识别对方身份：**
   - 皇帝、司礼监/镇抚司长官、其他官员、私人恩公

2. **判断场合性质：**
   - 公务场合或私人场合

3. **确认是否存在私人恩情：**
   - 救命之恩等特殊私人关系

4. **选择行礼方式：**
   - 跪拜（仅限皇帝和特定长官）
   - 举手礼（其他官员）
   - 私跪（私人恩情，需旁人回避不跟随）
