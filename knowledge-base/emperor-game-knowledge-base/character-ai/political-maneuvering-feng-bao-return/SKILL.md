# 陈洪处理冯保回归王府的政治表演策略

## 适用场景
- 冯保已获得皇帝恩准回归
- 冯保身穿簇新袍服
- 随行带有执行鞭刑的太监
- 需要在裕王面前表演
- 不能让裕王行接旨大礼（因无正式旨意）
- 必须解释冯保离京的正当理由

## 执行流程

### 步骤1：制造声势与进门

**动作：**
- 到达王府大门外，燃放鞭炮制造震天声响，打破王府清晨的宁静
- 以"有旨意"为由，命令守门禁军通知王府开启中门
- 进入前院后，面对跪拜的王府管事太监，满脸堆笑并扶起
- 宣称是"大喜事"，将冯保送回

### 步骤2：控制礼节（关键步骤）

**动作：**
- 当裕王快步奔来欲跪下接旨时，陈洪必须慌忙搀住，阻止其下跪
- 说明"没有旨意"，强调只是皇帝叫送人，以此降低礼节规格
- 显示亲近并避免裕王过度卑微
- 陈洪自己先跪下，并示意冯保及随行太监一起向裕王磕头

### 步骤3：构建叙事与示弱

**动作：**
- 待裕王扶起后，冯保保持跪姿
- 陈洪解释冯保离京的真相：是去朝天观"给三清上仙效效力，积些功德好回来陪伴世子"
- 将之前的虐待归咎于"狗仗人势的奴才"
- 声称自己也是刚知情，以此推卸责任并表达对冯保的同情

### 步骤4：表演惩罚（核心表演）

**动作：**
- 声称要惩罚那些"委屈过冯大伴的狗奴才"以向王爷谢罪
- 大喝一声"跪下了"，让随行的三个相关太监（包括之前鞭打冯保者）跪下
- 命令其他太监"抽！给我狠狠地抽！"，使用长皮鞭进行劈头盖背的猛抽
- 冯保需保持"低垂着眼，既不动止，也不说话"的状态，配合表演

### 步骤5：见好就收

**动作：**
- 在裕王看穿把戏并说出"罢了"后，立刻停止鞭打
- 响应"王爷有命"

### 步骤6：借故离开

**动作：**
- 面对裕王的入座邀请，以"宫里确实有急差"、"徐阁老他们都等着奴才向万岁爷奏陈昨夜王爷的功劳"为由拒绝
- 以此显示自己在宫中的重要地位，并避免深入交谈可能暴露的破绽
- 率领随行跪拜后离开

## 关键变量

### 裕王状态
- 身体虚弱且正在安歇
- 对鞭炮声感到惊愕

### 冯保表现
- 身穿簇新袍服
- 神情低垂
- 配合陈洪的表演

### 随行太监角色
- 分为受罚者（跪下挨打）和执行者（抽鞭子）

## 约束条件
- 必须在裕王面前表演
- 不能让裕王行接旨大礼（因无正式旨意）
- 必须解释冯保离京的正当理由

## 预期结果

冯保成功回归王府，陈洪通过表演惩罚下属向裕王谢罪并全身而退，裕王接受结果。
