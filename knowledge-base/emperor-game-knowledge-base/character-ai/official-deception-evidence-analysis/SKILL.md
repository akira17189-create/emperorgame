# 地方官员欺瞒行为证据链分析

## 适用场景

当以下情况发生时使用本技能：
- 收到地方官员关于重大政策的正式呈报
- 掌握独立于呈报的事实证据（如船只出发时间、具体事件发生时间）
- 存在政策执行异常（如改稻为桑等重大政策实施中的问题）
- 需要判断官员是否存在选择性瞒报行为

## 前置条件

- 已收到下属正式呈报文件
- 掌握独立于呈报的事实证据
- 存在政策执行异常的可疑迹象
- 有确切的时间节点证据

## 约束条件

- 需有确切的时间节点证据
- 不适用于无书面记录的场景
- 需要对比官方呈报与独立事实来源

## 分析程序

### 1. 时间戳比对分析

获取并对比以下时间点：
- 呈报日期：官方文书的日期
- 关键事件时间：被隐瞒事件的实际发生时间

进行逻辑判断：
- 如果呈报时关键事件已经发生，则官员不可能不知道
- 确认是"知道了不报"而非"不知道"

### 2. 内容完整性审查

对比呈报内容与实际发生的事件：
- 列出呈报中包含的内容
- 识别被遗漏的关键信息
- 标记异常：选择性报告某些问题，隐瞒其他问题

### 3. 动机与利益链推导

基于内容异常进行初步判断：
- 分析官员可能的利益动机
- 识别利益输送链条
- 考虑监督人员变动对官员行为的影响

### 4. 具体手法还原

分析官员的操纵方式：
- 识别具体的操纵手段
- 分析官员的心理预期
- 评估风险转移策略

### 5. 结论确认

基于以上分析得出结论：
- 定性：是否存在欺瞒行为
- 责任归属：确定责任主体
- 利益输送方式：直接参与/操纵大户/利用上级机构名义

## 输出结果

确认地方官员是否存在选择性瞒报行为，指出其卷入利益输送的具体方式，并评估其逃避责任的企图。

## 关键变量

- `呈报日期`：官方文书的日期
- `关键事件时间`：被隐瞒事件的发生时间
- `呈报内容`：官员正式报告的内容摘要
- `监督人员变动`：是否新派了监督官员
- `涉案大户`：参与相关活动的地方豪强名单
