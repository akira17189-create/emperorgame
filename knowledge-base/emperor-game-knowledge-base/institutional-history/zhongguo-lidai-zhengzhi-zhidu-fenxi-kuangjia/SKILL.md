# 中国历代政治制度分析框架

## 适用场景
- 阅读或讲授《中国历代政治得失》时需把握其整体论述结构
- 对汉、唐、宋、明、清五朝政治制度进行比较研究
- 需聚焦政府制度（非人事）且限定于四大制度领域的学术分析

## 执行步骤

1. **确认分析范围**
   - 仅限汉、唐、宋、明、清五个朝代
   - 聚焦制度而非人物评价
   - 排除学校、教育等非相关制度
   - 经济制度仅讨论田赋相关内容

2. **按四大制度领域依次展开分析**

   ### (1) 政府组织（职官制度）
   - 分析中央与地方政府的职权分配结构
   - 注意：此为“第二义”问题（政权归属更根本），但因叙述便利先行讨论

   ### (2) 考试与选举制度
   - 考察人才进入政府的机制（如察举、科举）
   - 理解其核心意义：“在政府和社会间打通一条路”，使社会能掌握、预闻并运用政治
   - 认识其理论优先级：本应最先讨论，因其关乎“政权交付给哪些人”这一根本问题

   ### (3) 赋税制度（仅限田赋）
   - 聚焦土地相关的财政安排
   - 参照唐代杜佑《通典》将“食货”列为首门的传统，理解其重要性
   - 严格限定于田赋，不扩展至其他财政或经济制度

   ### (4) 国防与兵役制度
   - 分析保卫政府的武力组织方式
   - 把握其功能定位：“养育此政府的是经济，保卫此政府的是武力”

3. **坚持历史实证原则**
   - 依据历史事实比较制度优劣
   - 引用当时人的意见说明制度得失
   - 避免以现代标准简单评判古代制度

> 提示：若需深入探讨某一制度的具体运作细节（如科举流程、两税法内容等），应调用相应专项技能或查阅参考资料。
