# 《唐六典》行政法规分析技能

## 何时使用
- 需要了解唐代尚书省六部的实际组织结构、人员配置与职务分工
- 需评估《唐六典》对宋、明、清行政制度的影响
- 需对比现实行政法典（《唐六典》）与理想政制蓝图（如《周礼》）
- 需从制度史角度解读中国古代行政体系的思想基础

## 执行步骤
1. **确认适用前提**：确保讨论对象基于唐代尚书省六部制度，且非理想化制度设计（如《周礼》）。
2. **提取核心内容**：聚焦《唐六典》中关于各部门职权、人事安排与运作规程的具体条文。
3. **强调现实属性**：明确指出《唐六典》是唐玄宗时期依据现行法规编纂的实证性行政法典，非乌托邦构想。
4. **揭示制度思想**：说明该典籍如何将唐代政治思想具体化、制度化，体现“制度必有理论支撑”的原则。
5. **延伸历史影响**：指出其被宋、明、清奉为行政典范，构成千余年官僚制度的参照基准。
6. **避免误用**：不得将其视为单纯史料记录，而应作为制度与思想双重载体进行解读。

## 关键区分
- **《唐六典》**：唐代实际施行的行政法规，具操作性与历史真实性。
- **《周礼》**：先秦理想政府模型，无实施依据，仅反映政治理想。

> 提示：本技能不适用于分析未受唐代制度影响的政权或纯理论性制度设计。
