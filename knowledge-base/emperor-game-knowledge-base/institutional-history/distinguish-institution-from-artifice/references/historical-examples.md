# 历史例证参考

## 典型制度案例
- **西汉**：郡国并行制虽有缺陷，但整体以天下为公，设官分职明确，属制度。
- **唐代**：三省六部制、科举制体现职责分工与公共理性，为典型制度。

## 典型法术案例
- **东汉**：外戚宦官专权，权力运作依附私人关系，缺乏制度节限。
- **宋代**：重文轻武、强干弱枝虽有制度形式，但核心出于防范武将之私心，部分措施沦为法术。
- **明代**：废丞相、设厂卫，皇权集中于私意，制度空壳化。
- **清代**：所有政治安排（如军机处、满汉分治、文字狱）皆服务于满洲部族私利，无真正制度，仅为法术。

## 关键概念界定
- **公心**：以国家长治久安、百姓福祉为出发点的政治动机。
- **私心**：以维护特定族群（如满洲）、家族（如皇室）或集团（如宦官）利益为核心目标。

## 边界情形处理
- 若某措施表面公开但实质服务私利（如清代科举），仍判为法术。
- 若制度被严重扭曲但仍保留公共框架（如晚唐藩镇），可视为制度退化，但起源仍属制度。
