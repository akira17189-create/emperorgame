# 平铺社会与人才政治的结构性矛盾分析

## 适用场景
- 分析宋代以后中国传统社会结构特征及其政治后果
- 探讨科举制度对人才流向与社会力量组织的影响
- 理解为何高度平等的社会反而缺乏抵抗中央崩溃的能力
- 对比中西社会力量形成路径差异

## 执行步骤

1. **确认适用条件**：仅适用于废除封建贵族、实行节制资本、开放政权（考试入仕）的常态治理时期；排除元、清部族特权时期及战乱状态。

2. **识别社会平铺特征**：判断是否具备以下三点：
   - 无世袭贵族阶层
   - 无资本高度集中
   - 政权向平民开放（通过科举等机制）

3. **分析人才流向后果**：
   - 聪明才智之士普遍选择从政而非工商业
   - 官僚系统臃肿，“无用的脂肪太多”
   - 生产领域人才匮乏，经济基础薄弱

4. **评估社会组织力缺失**：
   - 社会“平铺散漫”，如沙粒般缺乏凝聚中心
   - 中央一旦崩溃，无法形成有效地方抵抗或替代秩序
   - 对比西方：因早期排斥平民参政，才智转向工商业，形成工厂、政党等组织力量

5. **考察制度繁密化加剧问题**：
   - 文书制度层级繁多、程序冗杂
   - “开会忙”与行政内耗束缚人才施展
   - 制度越复杂，人才越难“转身”创新

6. **提炼根本矛盾**：社会平等（人人可入仕）与政治效能（需组织力与专业分工）不可兼得。

7. **提出行动建议**：
   - 简化政治与行政制度，减少人才束缚
   - 在保持社会平等基础上，重建新型社会组织中心（非依赖临时学生运动）
   - 参考孙中山“权能分开”思想：政权属民，治权由有能力者专业行使

> 注意：本技能不用于评价现代制度，仅用于历史结构性分析。
