# 分析清代部族政权下科举制度的本质

## 何时使用
- 用户探讨清代科举制度的公平性或开放性
- 用户分析满汉官僚结构差异
- 用户研究部族政权（如清代）与传统中原王朝在选官机制上的根本区别
- 用户引用邹容等近代思想家对清代政治的批判

## 如何执行
1. **确认政权性质**：验证所讨论政权是否为部族政权（如清代满洲统治），排除元代以前的传统中国考试制度。
2. **核查官缺分配数据**：检查中央机构（如六部）中满缺与汉缺的实际数量比例（通常满缺为汉缺三倍以上）。
3. **识别关键部门排他性**：确认理藩院等要害部门是否完全排除汉人任职。
4. **分析缙绅录信息偏差**：注意官方名录是否刻意隐去满缺数据，制造汉人参政假象。
5. **评估职位实质**：判断汉人通过科举获得的职位是否多为“额外主事”“清贫翰林”等虚职，升迁是否“补缺难于登天”。
6. **检验晋升路径差异**：确认汉人是否必须经由翰林院出身方可晋升尚书侍郎，而满人则无此限制且可文武兼资。
7. **判断制度意图**：综合上述证据，判定该制度是否旨在“设法虚糜之，戢其异心”，即通过虚职安抚汉人知识分子以维稳，而非真正开放政权。

> 注意：此分析不适用于非部族政权下的选官机制，亦不可将清代科举与汉唐宋明制度等同视之。
