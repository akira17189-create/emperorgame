# 行省制度历史分析

## 何时使用
- 用户研究元代地方行政体系
- 探讨外族（如蒙古、女真）统治下中央与地方权力关系
- 分析中国历史上行政区划不合理现象的制度根源
- 比较不同朝代地方治理模式（尤其排除唐宋等汉族本土政权）

## 如何执行
1. **确认适用对象**：仅适用于元代地方行政、金代地方治理、明代承宣布政使司；不适用于汉族主导的唐宋地方自治或现代民主地方治理体系。
2. **识别触发条件**：若政权由外族建立且对地方缺乏信任，则很可能采用行中书省制度。
3. **解析制度本质**：
   - 行中书省不是地方政府，而是“行动的中央政府”或“宰相府的派出所”，由中央中书省（都省）直接派出。
   - 其设立目的并非行政效率，而是军事控制与政治镇压，刻意割裂完整地理区域以防止地方联合反抗。
4. **追溯历史脉络**：该制度源于金代，制度化于元代，体现“异族征服者不敢分散政权，必须完全集中于中央”的逻辑。
5. **输出核心结论**：强调该制度揭示了外族统治下中央集权对地方自治的系统性压制，并警示其遗留的行政区划不合理问题。

> 注意：避免将行省误解为现代意义上的省级行政区；其本质是中央权力对地方的侵入机制。
