# 汉代察举制异化为门阀世族形成机制

## 适用场景
- 分析汉代政治制度演变
- 解释魏晋门阀社会的起源
- 探讨古代教育资源不平等对社会流动的影响
- 评估制度设计初衷与实际结果的背离

## 执行步骤
1. **确认制度基础**：核实是否为汉代察举制背景，核心为地方长官每年举荐“孝子”与“廉吏”，后合并为“孝廉”资格。
2. **识别关键约束条件**：
   - 察举名额按郡分配（如每二十万户口仅举一人）
   - 教育资源高度稀缺且依赖家族传承（竹帛书写、抄写成本高、师资稀少）
   - 地方长官（如二千石官员）掌握察举权
3. **追踪异化路径**：
   - **资源稀缺化**：名额限制使孝廉成为稀缺政治资本
   - **教育世袭化**：书籍获取困难导致“书本世袭”，形成“世代经学”家族
   - **权力闭环化**：家族成员任郡守获得察举权 → 察举亲信或后代 → 被举者日后报恩 → 循环巩固家族地位
   - **地域固化**：各郡固定几家垄断察举，形成“郡望”
4. **输出制度悖论结论**：制度初衷为“全国各地人民皆有跑进政府希望”，实际结果却是以“书籍资本”替代“黄金资本”，制造新型世袭不平等（“黄金满籯，不如遗子一经”）。

## 触发条件
仅在同时满足以下条件时启用本技能：
- 分析对象属于汉代选举制度、孝廉察举或地方仕进通道
- 察举名额按行政区域（郡）分配
- 教育资源获取存在显著家族垄断

> 注意：本技能不适用于印刷术普及后、民主选举制度或教育资源广泛可及的社会情境。
