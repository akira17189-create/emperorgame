# 分析清代地丁合一政策的真实性质

## 何时使用
- 用户分析清代“永不加丁赋”是否属于真正的仁政；
- 用户探讨清代农民实际税负变化；
- 用户质疑“地丁合一”是否为制度创新；
- 用户研究清代社会治理与民众责任意识的关系。

## 执行步骤
1. **确认历史前提**：
   - 明代一条鞭法已将差役摊入田赋；
   - 清初沿用万历则例，差役早已并入地租。

2. **梳理清代关键措施**：
   - 康熙五十年（1711年）诏令“永不加丁赋”，以当时2462万口为基准固定丁税总额；
   - 仍每五年编审人口，仅维持丁税基数不变；
   - 实际赋税通过田赋持续增长。

3. **判断政策实质**：
   - “地丁合一”并非制度创新，而是恢复明代旧制；
   - 政策本质是制度惰性延续，非真正减负；
   - 长期导致无产者对国家缺乏责任意识。

4. **揭示政治意图**：
   - 表面讨好下层民众，实则配合压制士人阶层；
   - 通过社会分化削弱整体抗清力量。

> 注意：本技能不适用于分析一条鞭法之前的赋役制度，也不适用于非农业人口税收问题。
