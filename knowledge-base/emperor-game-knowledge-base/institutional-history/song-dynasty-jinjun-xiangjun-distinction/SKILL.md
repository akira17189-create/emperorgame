# 宋代禁军与厢军的区分及选拔制度

## 何时使用
- 分析宋代初期（北宋太祖至仁宗时期）军队构成
- 判断某士兵属于禁军还是厢军
- 解释宋代兵制为何导致军队数量膨胀但战斗力低下
- 探讨“兵样”在士兵选拔中的作用

## 执行步骤

1. **确认军队类型**：宋代军队分为两类——禁军（中央精锐）与厢军（地方非战斗部队）。

2. **识别禁军特征**：
   - 职能：负责边防与京师守卫
   - 选拔标准严格：
     - 初期使用活人作为“兵样”参照体格
     - 后改用木制人形模型（“兵样”）分发各地
     - 符合标准者选送中央编入禁军

3. **识别厢军特征**：
   - 驻于各州府“城厢”周边
   - 不参与作战，仅承担修路、搬运、官府差遣等杂役
   - 多由未被选入禁军的老弱残兵组成

4. **追溯制度成因**：
   - 源于唐末五代藩镇割据遗留问题
   - 当时军队被视为罪犯（“配军”），面上刺字以防逃亡
   - 宋太祖无法彻底裁撤旧军，只能择优组建禁军，余者留为厢军

5. **评估制度后果**：
   - 形成“养兵而不能打仗”的结构性矛盾
   - 军队数量持续膨胀：太祖约20万 → 太宗66万 → 仁宗125万

> 注意：本技能仅适用于北宋初期至中期军事制度分析，不适用于汉唐兵制或南宋后期防御体系。
