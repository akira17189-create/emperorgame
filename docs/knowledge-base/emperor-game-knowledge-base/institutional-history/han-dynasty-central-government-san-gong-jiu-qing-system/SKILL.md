# 汉代中央政府三公九卿制度结构

## 何时使用
- 分析汉代中央政府最高层组织架构
- 解释“三公九卿”在国家治理中的角色
- 探讨皇室财政（少府）与国家财政（大司农）的分离机制
- 理解从“以家为国”到“化家为国”的政治制度演变

> **注意**：仅适用于西汉至东汉前期（不适用于秦以前或魏晋以后的官制演变期）。

## 执行步骤

### 1. 识别三公构成与核心职能
- **丞相**：行政首长，文官之首，负实际政务全责；依不成文法，须先任御史大夫方可升任。
- **太尉**：军事首长，武官之首，仅掌军事，不预闻其他政事。
- **御史大夫**：副丞相，掌监察，辅助丞相监督所有政治设施。

### 2. 列出九卿及其原始职能（名义上属皇家家务）
- **太常**：管皇家祭祀与教育（因古代学术在宗庙）。
- **光禄勋**：原为皇宫门房，后管侍卫及郎官（青年入仕初阶）。
- **卫尉**：皇宫卫兵司令，掌门卫屯兵。
- **太仆**：皇帝车夫，兼管国家车马武装。
- **廷尉**：掌司法刑狱，后演变为全国司法首长。
- **大鸿胪**：主外交与宾客接待，类礼宾司或外交部。
- **宗正**：管皇帝同姓本家及异姓亲戚事务。
- **大司农**：管政府财政，收入支销国家公费（主要来源为全国田赋）。
- **少府**：管皇室财政，收入供皇室私用（如盐铁工商税），不得使用大司农资金。

### 3. 明确关键隶属关系
- 九卿**全部隶属于宰相**，体现“皇室经济也由宰相支配”。
- 此结构反映“化家为国”：原皇家家务官转为国家公职。

### 4. 应用历史演变逻辑
- 封建时代：“以家为国”，宰相为天子管家。
- 郡县时代：“化家为国”，宰相成为政府首长，家务部门转为国家行政部门。

## 输出目标
清晰说明汉代如何通过三公九卿制度实现政务专业化，并维持皇室与政府财政的制度性分离。
