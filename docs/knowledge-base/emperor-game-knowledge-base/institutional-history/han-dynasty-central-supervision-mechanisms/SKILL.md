# 汉代中央对地方的监察机制

## 适用场景
- 分析汉代中央如何控制地方行政
- 解释刺史的职能与权限边界
- 阐述上计制度在地方考核中的作用
- 区分汉代监察体系与其他朝代（如唐、明、清）地方监督机制

## 执行步骤

### 1. 应用上计制度
- 要求每郡于每年九月至十月间向中央提交“计簿”（年度行政统计表册）。
- 计簿内容必须涵盖财政收支、经济状况、教育成效、刑事案件、民事纠纷、盗贼情况及灾荒应对等核心行政指标。

### 2. 部署刺史监察
- 将全国划分为十三个监察区，每区派遣一名刺史。
- 确保每名刺史监察不超过九个郡。
- 严格限定监察范围为“六条考察”，不得干预六条以外事务。
- 明确刺史仅为六百石低阶官员（远低于二千石太守），仅作为中央“耳目”，不替代地方行政。

### 3. 确认监察隶属关系
- 刺史直接向御史中丞汇报。
- 御史中丞隶属于御史大夫（副丞相）。
- 御史大夫最终向宰相负责。

### 4. 补充皇宫内部监察
- 启用十五名侍御史，专门弹劾中央及皇宫内部事务，不涉及地方。

### 5. 明确责任归属
- 地方行政责任由太守承担。
- 刺史仅履行监督职责，无权代行行政职能。

## 注意事项
- 本机制仅适用于汉代，不适用于唐代节度使、明代巡抚或清代总督等后世制度。
- 刺史若越出“六条考察”范围即属违规。
- 上计与刺史制度共同构成低成本、高效率的双重监控体系。
