# 汉代士人政府形成机制

## 何时使用
当需要分析以下问题时启用本技能：
- 汉代官员的主要来源渠道
- “士人政府”如何取代贵族世袭或军功入仕
- 太学、孝廉、贤良、郎官等制度如何构成完整入仕链条
- 判断某时期政府性质是否属于“读书人的政府”

## 执行步骤

1. **确认适用前提**：
   - 封建世袭制已废除
   - 太学已设立（武帝之后）
   - 郡国察举制度正在运行
   - 时间范围限于西汉武帝至东汉末，排除九品中正制时期

2. **梳理四阶段入仕路径**：
   - **第一阶段：入太学学习**
     进入国立大学（太学）接受儒家经典教育。
   - **第二阶段：毕业分科任用**
     - 甲科：补为“郎”，进入光禄勋下属郎署，担任皇宫侍卫。
     - 乙科：回本郡“补吏”，任地方掾属（须为本地人）。
   - **第三阶段：经察举晋升**
     - **孝廉**：郡国每年按定额（后期约二十万户举一人）举荐德才兼备者，到中央仍任郎官。
     - **贤良**：遇新帝即位、灾异等特殊时机，由朝廷下诏举荐，经“策问”（政府提问）与“对策”（书面作答）后任用。
     - **特殊选举**：如出使、治河、军职等，可自荐或被特举。
   - **第四阶段：中央考试授官**
     郎官服务数年后，通过中央考核，正式分发任职。

3. **评估制度效果**：
   - 郎官群体从贵族子弟为主转变为太学出身的知识青年为主。
   - 政府官吏几乎全部由此路径产生，形成“士人政府”。
   - 自昭宣时期起，宰相多为读书人，非凭血统、军功或财富。

4. **注意历史演变**：
   - 初期存在“任子制”（高官子弟直接为郎），垄断仕途。
   - 武帝后，太学甲科与孝廉制打破垄断，东汉时“仕途只有孝廉察举的一条路”。
   - 后期孝廉渐失“孝子廉吏”原义，仅作为参政资格；中央加试，形成“教育—实习—选举—考试”闭环。

5. **输出判断**：
   若官员主要来自太学→郎官/补吏→孝廉或贤良→中央考试这一路径，则判定为“士人政府”。
