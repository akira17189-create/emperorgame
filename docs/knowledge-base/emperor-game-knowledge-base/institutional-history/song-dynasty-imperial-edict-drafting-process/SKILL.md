# 宋代诏书拟定程序与君相权力分配

## 何时使用
- 用户研究宋代政治制度、中央决策机制或皇权演变
- 需要对比唐代“熟拟”与宋代“箓子”制度差异
- 探讨“宋代宰相失职”是否源于制度设计
- 分析皇帝在最高政令中的实质性参与程度

## 执行步骤

1. **识别制度背景**
   - 确认讨论对象为宋代（尤其宋初至神宗朝前）的皇帝诏书拟定程序
   - 排除唐代熟拟制度及神宗朝后部分恢复旧制的特殊情况

2. **对比唐宋诏令流程**
   - 唐代：中书省“熟拟”定稿，皇帝仅“亲览纸尾批几句”并“印画”同意，实质为“宰相出旨，皇帝同意”
   - 宋代：宰相先提交“箓子”（多方案意见节要），通过“面取进止”请皇帝裁决，再依旨意正式拟旨

3. **确认权力归属**
   - 宋代皇帝拥有“参加意见之权”，最高政令决定权归皇帝
   - 宰相角色从决策者转为执行者

4. **验证关键例证**
   - 宋初“堂帖”被禁，改用箓子指挥政务
   - 太宗因地方官抗议箓子处分，下令“公事须降敕处分”，强化皇帝专属命令权
   - 神宗虽恢复“割子”（即箓子），但君重相轻格局已定

5. **得出结论**
   - 宋代宰相权力削弱是制度设计结果，非单纯帝王专制所致
   - 最高政令必须经皇帝实质性参与决策
