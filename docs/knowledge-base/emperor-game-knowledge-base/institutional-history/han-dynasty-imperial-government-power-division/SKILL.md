# 汉代皇室与政府的职权划分原则

## 何时使用
- 分析汉代中央政治体制结构
- 辨析“皇帝专制”论断在汉代的适用性
- 比较皇权与相权的实际运作范围
- 研究中国古代政府组织演变

## 核心原则
1. **角色定位**：皇帝为国家元首，象征国家统一；宰相为政府首脑，承担全部实际政务责任。
2. **制度体现**：
   - **皇室机构（六尚）**：皇帝设“六尚”秘书处，其中五尚（尚衣、尚食、尚冠、尚席、尚浴）仅管理皇帝私人生活；尚书最初仅四人，掌管文书，地位不高。
   - **政府机构（十三曹）**：宰相设“十三曹”秘书处，规模庞大、职权广泛，覆盖全国政务。
3. **权力对比**：十三曹处理全国实际政务，而皇室仅有一个笼统的尚书处，表明“一切实际事权，照法理，该在相府，不在皇室”。
4. **历史意义**：此划分体现“中国人一向意见，皇室和政府是应该分开的”，可用来反驳“秦汉以来皆为皇帝专制”的误解。

## 执行步骤
1. 确认分析对象为汉代中央政治体制（非秦以前或明清等朝代）。
2. 识别涉及的机构：区分“六尚”（皇室）与“十三曹”（政府）。
3. 对照十三曹职能，判断具体政务归属：
   - 西曹：府史署用人
   - 东曹：二千石高官及军吏任免
   - 户曹：祭祀与农桑
   - 奏曹：章奏处理
   - 词曹：民事诉讼
   - 法曹：邮驿与交通时限/量限（科程）
   - 尉曹：军队物资转运
   - 贼曹：盗贼案件
   - 决曹：刑事审判
   - 兵曹：兵役管理
   - 金曹：货币与盐铁
   - 仓曹：仓储谷物
   - 黄阁：总务协调
4. 得出结论：若实际政务由宰相十三曹主导，则汉代并非皇帝专制，而是皇室与政府分权体制。

> 注意：本技能不适用于秦以前封建时代或后世皇权高度集中的朝代（如明清）。
