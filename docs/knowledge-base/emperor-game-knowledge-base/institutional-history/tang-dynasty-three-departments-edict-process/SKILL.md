# 唐代三省制下的诏敕制定与封驳流程

## 何时使用
- 用户询问唐代如何合法发布最高命令（敕）
- 需验证某项唐代政令是否符合法定程序
- 分析中书省、门下省在决策中的角色与制衡机制
- 探讨政事堂会议对诏敕合法性的作用

> **注意**：本流程仅适用于三省制正常运作、皇帝接受程序约束、且非宦官专权时期的常规政令制定。

## 执行步骤

### 1. 起草阶段（中书省）
- 中书舍人（通常七八人）各自拟稿，称为“五花判事”
- 中书令或中书侍郎选定一稿并修润后呈送皇帝

### 2. 皇帝确认（画敕）
- 皇帝仅在敕文上“画一敕字”表示同意，不参与实质内容拟定

### 3. 封驳审查（门下省）
- 门下省（侍中、侍郎及给事中）对敕文进行复核
- 若认为不当，可行使“封驳权”，批注退回（称“涂归”）
- 未经门下副署的敕令不具备法律效力

### 4. 政事堂协调与盖印
- 中书、门下两省在政事堂举行联席会议，预先协调意见以减少涂归
- 正式诏敕必须加盖“中书门下之印”，表明已通过政事堂审议
- 仅有加衔“同中书门下平章事”者方可出席政事堂，视为真宰相

### 5. 执行移交（尚书省）
- 经完整流程的诏敕移交尚书省执行
- 尚书省仅有执行权，无权修改或否决政令内容

## 合法性要件
确保政令合法必须完成以下完整链条：
**中书起草 → 皇帝画敕 → 门下封驳审查 → 政事堂盖印 → 尚书执行**

任何环节缺失（如绕过门下、未盖政事堂印）均构成程序违法，如唐中宗“斜封墨敕”即因违反此流程而遭士人鄙视。
