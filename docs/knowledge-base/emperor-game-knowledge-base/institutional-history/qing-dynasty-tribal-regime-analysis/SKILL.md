# 清代部族政权分析技能

## 何时使用
- 分析清末改革路径选择（如康有为变法主张 vs 孙中山革命理论）
- 比较清代与汉唐宋明等中国传统政权的政治结构
- 探讨辛亥革命后制度重建失败的历史根源
- 讨论“主权”归属是否掌握在特定族群集团手中

## 执行步骤

1. **识别政权性质**
   - 若政权立场背景为“全国人民、全社会”，无特殊集团私利，则属于中国传统政权（如汉唐宋明）。
   - 若主权实际由“满洲部族”这一特殊统治集团掌握，皇帝仅为表象，则属于清代部族政权。

2. **评估改革路径可行性**
   - 在部族政权下，仅靠说服皇帝推行变法（如康有为路径）无效，因真正权力在部族集团。
   - 必须通过革命打破部族集团垄断（如孙中山主张），之后再进行制度建设（“革命之后还得要变法”）。

3. **避免历史误判**
   - 切勿将“满清制度”等同于“中国全部历史传统”，否则会导致推翻政权的同时否定整个历史文化，削弱新制度的社会共信基础。

4. **注意适用边界**
   - 本技能不适用于汉唐宋明等非部族政权分析。
   - 不适用于仅以“皇帝专制”为框架的简化解释。

## 关键变量
- **部族集团**：指满洲人作为一个整体掌握国家主权的统治集团。
- **共尊共信心**：民众对制度的普遍信任与尊重，是制度稳定运行的心理基础。
