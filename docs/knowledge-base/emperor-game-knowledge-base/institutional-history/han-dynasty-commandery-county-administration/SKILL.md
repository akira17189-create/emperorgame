# 汉代郡县两级地方行政体系分析

## 适用场景
- 分析汉代地方行政架构
- 评估汉代吏治效率与制度设计
- 对比不同朝代地方治理体系（仅限于确认对象为汉代郡县制时）

## 执行步骤

1. **确认行政层级结构**：全国划分为郡与县两级，县为基层单位，郡辖10–20县。
2. **核实规模数据**：
   - 郡总数约200余个
   - 县总数在1100–1400之间
3. **识别关键官职与任命方式**：
   - 郡长官为太守，俸禄二千石，地位与中央九卿相当
   - 县长官为县令，由中央直接任命
4. **分析官员流动机制**：
   - 九卿可外放为郡太守，太守可内调为九卿
   - 此机制实现“升转极灵活”，强化中央与地方联系而不显等级隔阂
5. **应用人事原则**：
   - 地方长官（太守、县令）必须由中央任命
   - 其下属掾属必须为本地人，由长官自行辟署（即自主任命）
6. **评估治理效能**：结合“适中行政规模 + 灵活官阶流动 + 本地掾属制”判断汉代地方治理是否高效且贴近民情。

> 注意：本技能不适用于周代分封制或元明清行省制等非汉代郡县体系。
