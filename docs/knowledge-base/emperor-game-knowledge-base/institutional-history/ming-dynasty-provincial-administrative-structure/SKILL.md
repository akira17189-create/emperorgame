# 明代省级行政三司与监司官体系分析

## 适用场景
- 研究明代地方行政制度演变
- 分析省级辖区过大引发的治理调整
- 探讨“分守道”“分巡道”等监司官的设置逻辑与实际影响
- 对比汉代郡县二级制与明代四级结构的治理效能

## 执行步骤
1. **确认背景条件**：验证是否满足触发条件——地方行政区域过大且事务繁杂，且处于明代（不适用于元代行省制或汉代郡县制）。
2. **识别三司架构**：明确明代省级平行设立的三大机构：
   - 承宣布政使司（藩司）：主管民政与财政
   - 提刑按察使司（臬司）：主管司法与监察
   - 都指挥使司：主管军事
3. **追溯分司设置**：因省区过大，从三司派出分司：
   - 布政使派出“分守道”，分管若干府州行政
   - 按察使派出“分巡道”，分管若干府州司法监察
4. **构建行政层级**：形成“县 → 府/州 → 分司（道台）→ 省司”的四级结构。
5. **评估结构性后果**：
   - “管官的官多，管民的官少”
   - 县官作为唯一“亲民官”地位卑微，疲于应付上级
   - 地方行政效率低下，县政衰败
6. **输出核心结论**：揭示该体系在制度设计上导致的治理冗余与基层弱化问题。

> 注意：本技能仅适用于明代省级行政分析，不可用于其他朝代或中央官制研究。
