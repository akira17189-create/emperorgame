# 通过鼓声节奏预判事态升级

## 适用场景
- 封闭空间（如琴房）
- 能够听到清晰的鼓声节奏
- 具备古曲《风吹荷叶煞》的知识储备
- 不适用于无规律的噪音或非音乐性的敲击

## 初始阶段监听
- 聆听鼓声节奏
- 若鼓声"安详"，敲击缓慢，每次敲击后有片刻停顿
- 接着是余音，描述为"像是微风吹过荷塘无边的莲叶"
- 则进入下一阶段观察

## 加速阶段识别
- 注意节奏变化
- 若敲击声"慢慢加快"
- 描述为"像是间歇的雨滴落在荷塘无边的莲叶上"
- 则识别为特定曲目

## 曲目匹配
- 将上述节奏模式识别为古曲《风吹荷叶煞》
- 该曲相传是称衡当年为曹操演奏的

## 事态预判
- 基于该曲目的结构逻辑
- 预判接下来的阶段必然是"狂风暴雨般的宣泄"
- 在当前语境下（封闭的琴房），这暗示着迫在眉睫的剧烈事件或危险
- 如自杀或纵火

## 行动决策
- 立即下令"把门撞开"以试图干预或进入

## 危险验证（边缘情况处理）
- 若门纹丝不动且伴随热浪冲出
- 随后门缝喷出火苗
- 确认琴房内已泼满油
- 且"狂风暴雨般的宣泄"已表现为火灾
- 此时应立即撤离并寻找水源救火

## 关键变量
- **鼓声节奏**：鼓声的快慢、停顿及余音特征
- **曲目特征**：《风吹荷叶煞》的旋律走向及情感宣泄模式

## 执行结果
立即执行破门行动以应对即将发生的剧烈危机。
