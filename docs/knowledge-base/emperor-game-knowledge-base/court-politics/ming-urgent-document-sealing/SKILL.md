# 明代紧急公文封装与递送程序

## 适用场景
当出现以下情况时使用本技能：
- 重要公文需紧急递送
- 涉及多官员联署或重审案件
- 需确保公文密封性与权威性
- 需标明紧急程度

## 核心流程

### 第一步：公文整理与装封
1. 收集全部需封装文件
2. 由主审官员与协审官员共同确认文件完整性
3. 将文件装入皮纸公文信封

### 第二步：烤漆密封
1. 将凝固在铜签上的漆棒在火上烤熔
2. 将熔化的漆糊涂上信封封口处
3. **关键**：漆需烤熔但不可过热烧焦，保证粘性适中

### 第三步：印章加盖
1. **时机把握**：趁烤漆未硬盖印（半凝固状态最佳）
2. **印章选择**：使用主审官员印章
3. **联署规则**：原案用原案封印，重审用重审官印章

### 第四步：羽毛标识
1. **数量**：三根羽毛（最高紧急级别）
2. **位置**：粘在烤漆处
3. **时机**：必须在漆未硬之前完成

### 第五步：指定递送
1. 明确递送时限（如：一个时辰内）
2. 指定接收人
3. 采用急递方式

### 第六步：同步押解
如涉及人犯，同步安排押解至接收衙门，确保审讯连续性

## 关键变量
- **漆棒状态**：烤熔至可流动但不过热
- **印章使用权限**：主审官员优先
- **羽毛数量**：三根代表最高紧急级别
- **递送时限**：根据紧急程度设定

## 约束条件
- 烤漆需趁热盖章
- 羽毛粘贴需在漆未硬之前完成
- 印章使用需符合官员权限

## 防伪功能
- 强行拆封必然破坏漆印
- 印章与羽毛组合具有唯一性
- 接收人可通过检查漆印完整性确认未被拆阅
