# 正月十五不抓人惯例

## 适用场景

当出现以下情况时涉及本惯例：
- 正月十五元宵节期间需要执行抓捕
- 官员援引节日惯例试图阻止搜捕
- 权臣无视惯例坚持执行抓捕

## 惯例内容

**正月十五不抓人**：
- 元宵节，传统上停止刑狱的日期
- 官场柔性的时间禁忌
- 可作为护身符使用

## 惯例突破策略

### 高翰文的抗辩策略

- 援引官场惯例"正月十五不抓人"试图阻止搜捕
- 利用元宵节的社会禁忌作为护身符

### 严世蕃的反击逻辑

**直接否定惯例效力**：
- "正月初一老子还杀过人呢"
- 以自身更极端的行为先例压制对方

**表明权臣不受约束**：
- 权臣不受常规礼法约束
- 自称"老子"而非官职称谓，体现跋扈姿态

**即时命令**：
- "来人！"的即时命令显示对军队的直接控制
- 无视节日禁忌，下令"搜！把那个女人给我搜出来！"

## 惯例与暴力的冲突结果

- 官场柔性的时间禁忌（正月十五）
- 被权臣刚性的暴力逻辑（正月初一尚能杀人）彻底压倒

## 关键变量

- **正月十五**：元宵节，传统上停止刑狱的日期
- **正月初一**：元旦，一年中最禁忌刑杀的日子

## 权力展示

- 权臣严世蕃（小阁老）vs 待罪官员高翰文
- 权力不对称：权臣可以无视惯例，待罪官员只能依赖惯例保护

## 成功标准

- 权臣成功突破惯例，执行抓捕
- 待罪官员的惯例抗辩失效
- 搜捕行动继续进行

## 风险提示

- 惯例仅对遵守礼法的官员有效
- 权臣可以无视惯例，使用暴力
- 依赖惯例保护存在风险
