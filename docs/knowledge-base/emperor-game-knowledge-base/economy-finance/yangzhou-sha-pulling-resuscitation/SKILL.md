# 扬州扯痧急救法

## 何时使用

- 患者中暑症状（晕倒、虚弱）
- 可能伴有饥饿症状
- 有扬州/南方急救经验者在场
- 尤适用于江南体质者

## 核心特点

> 北人放血，南人扯痧
> 扬州技法：仅在颈部三处即可救人，技术精准高效

## 操作流程

### 准备阶段

1. **取水**：取凉水一碗
   - **必须用缸里存放的水**
   - **禁用井水**
2. **准备**：施救者挽起右手衣袖

### 执行阶段

| 顺序 | 部位 | 操作 | 目标效果 |
|-----|------|------|---------|
| 1 | 左颈部 | 湿指轻刮，夹扯三把 | 紫黑色痧痕一条 |
| 2 | 右颈部 | 同样方法扯数把 | 再一条黑紫痧痕 |
| 3 | 后颈脊椎处 | 转到背后扯数把 | 完成三处治疗 |

### 恢复阶段

1. 患者悠悠醒来，轻哼出声
2. 让患者坐着不要移动
3. 换一碗水给患者饮用
4. 继续休息观察

## 关键要点

| 要点 | 说明 |
|-----|------|
| 用水 | 缸里存放的水，非井水 |
| 手法 | 先刮后扯，食中二指操作 |
| 部位 | 仅颈部三处，非满颈满胸满背 |
| 标志 | 出现紫黑色痧痕为度 |

## 地域对比

| 地区 | 技法 | 范围 |
|-----|------|------|
| 湖广一带 | 扯痧 | 满颈满胸满背 |
| 扬州 | 扯痧 | 仅颈部三处，更精准 |
