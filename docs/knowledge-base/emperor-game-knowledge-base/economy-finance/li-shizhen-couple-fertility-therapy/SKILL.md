# 夫妻和谐生子疗法

## 何时使用

- 夫妻长期无子
- 脉象正常，排除生理性病变
- 夫妻互动疏离：不敢睁眼、不敢对视、在外人面前装作互不认识
- 有威信长辈在场配合监督
- 疑似长辈干涉子媳房帷

## 诊断要点

### 脉诊方法

- 丈夫伸左腕，妻子伸右腕，同时摆在桌上
- 医者两手六指同时搭二人寸关尺

### 关键观察

| 观察项 | 异常表现 | 正常应然 |
|-------|---------|---------|
| 女方容貌 | 端庄清秀，诗书家风 | — |
| 夫妻眼神 | 低垂着眼，谁也不正面看谁 | 自然交流 |
| 肢体语言 | 丈夫不叫夫人就座 | 相互关照 |
| 长辈因素 | 母亲在旁侧耳倾听 | 适度回避 |

### 诊断结论

**非病理原因，而是家庭伦理（命）所致**

> "医可治病，不可治命"

## 治疗流程

### 第一阶段：强制对视

1. 指令："你们各自望着对方的眼"
2. 若看向旁人，由长辈喝止："太医叫你们互相望着，看我干什么？"
3. 强调："望着，不要转睛"

### 第二阶段：诱导微笑

1. 指令："笑一笑"
2. 丈夫笑得僵硬：鼓励"再笑开些"
3. 妻子不敢笑：催促"要赶快，快笑"
4. 利用情境（丈夫奇怪的笑模样）诱导妻子自然发笑

### 第三阶段：大笑疗法

- 医生突然爆发出一阵大笑，声震屋宇
- 打破严肃僵硬气氛，消除紧张感
- 让夫妻从"人天交战"中解脱

### 第四阶段：回归生活

- 指令："你们该做官的做官去，该做饭的做饭去"
- 医生与长辈单独留下商议后续处方

## 预后

> "只要你们这样做了，两年以内我准保你抱上孙子"

## 核心原则

| 层面 | 原则 |
|-----|------|
| 医学边界 | 医可治病，不可治命 |
| 治疗本质 | 心理疏导与行为干预 |
| 关键变量 | 长辈干涉程度、夫妻配合度 |
