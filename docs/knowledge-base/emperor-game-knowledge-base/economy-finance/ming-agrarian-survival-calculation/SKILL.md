# 明代农业生存口粮计算

计算明代县域百姓在正常年份与灾后的粮食拥有量，评估生存状况。

## 何时使用

- 分析明代县域百姓粮食安全状况
- 评估自然灾害或土地政策对民生的影响
- 对比自耕农与佃农的生存差异
- 验证历史文献中关于"民不聊生"的描述是否成立

## 核心计算流程

### 步骤1：确定基础数据

收集以下四项数据：
- 在籍人口总数
- 稻田亩数、桑田亩数（区分土地类型）
- 亩产标准（丰年/歉年/正常年份）
- 地租分成比例（自耕农为0，佃农通常为0.5）

### 步骤2：计算人均粮食拥有量

执行以下换算链：

```
总产量 = 稻田亩数 × 亩产(石)
人均稻谷 = 总产量 ÷ 人口 × 120斤/石（明代一石约120斤）
人均白米 = 人均稻谷 × 0.83（脱粒折损率约17%）
日均口粮 = 人均白米 ÷ 365天
```

**生存阈值判断**：
- 日均 ≥7两：老人孩童可勉强充饥，壮丁仍不足
- 日均 3.5-7两：普遍不足，需副业补充
- 日均 <3.5两：生存危机

### 步骤3：计算佃农模式（如适用）

当百姓因灾害或卖田转为佃农时：

**稻田佃农（五五分租）**：
```
净得稻谷 = 亩产 × 0.5
日均白米 = (净得稻谷 × 120斤/石 × 0.83) ÷ 365
```
典型结果：约3.5钱/日

**桑田佃农**：
- 分租比例低于五五（通常四六或三七）
- 产出为蚕丝，需换算为粮食等价物
- 典型结果：低于2.5钱/日

### 步骤4：评估副业补充能力

检查区域是否"靠山有水"：
- **有条件**：茶叶、桑麻、桐漆、鱼虾 → 可换油盐粗粮勉强度日
- **无条件**：完全依赖口粮，生存线即为硬性边界

### 步骤5：输出生存评估

| 日均口粮 | 生存状况 | 政策含义 |
|---------|---------|---------|
| ≥7两 | 勉强维持，壮丁不足 | 需警惕徭役征发 |
| 3.5-7两 | 普遍匮乏，依赖副业 | 灾害即崩溃 |
| <3.5两 | 生存危机，民变风险 | 紧急赈济或减赋 |

## 关键公式速查

| 项目 | 公式 | 备注 |
|-----|------|------|
| 石转斤 | 1石 = 120斤 | 明代度量衡 |
| 脱粒折损 | 稻谷×0.83=白米 | 含碾磨损耗 |
| 斤两换算 | 1斤 = 16两 | 明代16两制 |
| 佃农净得 | 总产量×(1-租率) | 租率通常0.5 |

## 输出模板

```
【计算对象】XX县百姓
【人口/土地】X万人 / X万亩（稻田X/桑田X）
【正常年份】人均日粮X两 → 评估：XX
【灾害/佃农后】人均日粮X钱 → 评估：XX
【生存结论】（满足/勉强/危机）
【政策建议】（如适用）
```

## 约束与边界

- 仅适用于明代农业经济环境（赋税、度量、租佃制度）
- 未计入极端天气、战乱、瘟疫等额外变量
- 副业收入为定性估计，精确计算需具体物价数据
