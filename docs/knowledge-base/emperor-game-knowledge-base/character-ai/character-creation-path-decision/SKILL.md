# 单一故事人物塑造路径决策

## 适用场景
当开始处理单一故事人物、需要确定角色塑造的操作顺序时使用。

## 两种塑造路径

### 路径一：由外而内
1. **第一步**：先塑造外在特征
   - 外貌、服饰、行为习惯等可见特征
2. **第二步**：回头打造内在核心
   - 动机、价值观、性格本质等内在特质

### 路径二：由内而外
1. **第一步**：先从内在核心处理起
   - 确定人物的本质、动机、价值观
2. **第二步**：再往外添加外在特征
   - 根据内在核心设计相应的外貌、服饰、行为

## 决策依据
- 此决策独立于整体创作顺序
- 无论先打造设定再填卡司，或先发想卡司再打造世界，都不影响此选择
- 根据创作习惯和角色需求选择合适的路径

## 创作理念参考
人物是人类文明过去与现在的集合体，自书籍和报纸里取几段、从人性中拿一点、自华美的服饰里剪几块碎布拼凑而成。

## 使用步骤
1. 明确角色创造需求
2. 评估两种路径的适用性
3. 选择塑造路径（由外而内 或 由内而外）
4. 按选定路径执行角色塑造
