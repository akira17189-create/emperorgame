# Imperial Decree Examples

## Example 1: Feng Bao Exile Decree
- **Target**: Feng Bao (冯保)
- **Crimes**: Insubordination, secret communication, causing trouble
- **Sentence**: Hard labor at Chaotian Temple (朝天观)
- **Execution**: Immediate removal, no delays allowed
- **Messenger**: Chen Hong

## Example 2: Hai Rui Calligraphy Decree
- **Target**: Prince of Yu (裕王)
- **Action**: Copy Hai Rui's inscription, sign it, create plaque, hang at Liubiju
- **Purpose**: Test Prince's loyalty and relationship with Hai Rui
- **Constraints**: Must be done by Prince personally

## Example 3: Deathbed Political Arrangements
- **Economic**: Return Chu Prince's land to commoners (1,457,326 mu)
- **Personnel**: Pardon Hai Rui as "national weapon"
- **Cabinet**: Succession order: Xu Jie → Gao Gong → Zhang Juzheng
- **Principle**: Use when virtuous, dismiss when not

## Example 4: Pardon Transaction
- **Condition**: Hai Rui must admit error in public
- **Required admission**: Misunderstood Confucian teachings
- **Reward**: Pardon, promotion to Guozijian, participate in exam
- **Consequences for others**: All implicated persons saved if Hai Rui complies
