# 司礼监太监礼仪规范

## 使用场景
- 司礼监太监准备上朝或外出
- 司礼监值房内的日常管理
- 需要称呼司礼监掌印太监
- 处理灯烛等日常事务

## 值房空间规制

### 建筑规格
- 尺寸：进深一丈五尺，宽长五丈
- 改造：将原有三间房打通隔墙改为一间
- 取暖：两个白云铜大火盆，堆满寸长银炭
- 照明：屋梁上吊挂数盏红灯笼，与火盆上下辉映

### 核心人员座次
- 正中：司礼监掌印太监
- 两侧：四大秉笔太监（左右各二）\- 座椅：挨着北墙一溜儿五把紫檀木设垫的椅子

### 服侍人员配置
- 每位大太监配备两名小太监
- 腿下小太监：负责脱下暖鞋，换上上朝靴子
- 身后侧方小太监：负责给脖子围上白狐皮披肩

## 外出朝仪准备程序

1. **披风与袖筒准备**
   - 披风与白狐皮袖筒预先拿在手里
   - 给五人披披风并系上
   - 给五人套上狐皮袖筒

2. **抬舆准备**
   - 院子里预备五顶抬舆
   - 出发前掀开油布

3. **乘坐规制**
   - 五人走下台阶坐上抬舆
   - 各自太监将一块出锋的皮毡盖在其身体上
   - 四人一抬的抬舆出行

## 灯烛管理规则

### 正常流程
- 司礼监当值太监头儿带领当值太监手提灯笼两排站立
- 等候老祖宗到来
- 轿子从院门进入，太监齐声问候"老祖宗晨安"

### 情绪反常时的处理
1. **外院场景**
   - 吕芳自行撩帘出轿
   - 发出"压轿！压轿！"指令
   - 观察天色后突然发怒"还点着灯干什么！"
   - 当值太监头儿执行"熄灯！把灯笼都熄了！"

2. **内院场景**
   - 当值太监头儿跟进内院，令"把灯笼都灭了"
   - 吕芳停步质问"谁叫灭灯了？"
   - 惩罚指令："黑地儿里待着去！"

## 称谓禁忌规则

### 禁用称谓
- **"二祖宗"**：暗示存在"一祖宗"（指代前任吕芳），触犯陈洪忌讳
- **"老祖宗"**：暗示前任已死或陈洪即将退位，关联到"永陵"（陵墓）

### 正确称谓
- 必须使用"祖宗"，且前面不能加"二"、"老"等修饰词

### 违规后果
- 发配浣衣局或降职
- 被视为"不懂规矩"

## 权力互动礼仪

### 下级报喜程序
- 太监需"喘着气兴奋得满脸通红"闯入
- 在掌印太监面前"扑通跪下"
- 口称"恭喜老祖宗！贺喜老祖宗！"

### 群体行动秩序
- 两侧四大太监"急着想出门"但"都没举步"
- 必须"把目光全望向正中的吕芳"
- 等待吕芳率先行动
