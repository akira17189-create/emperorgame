# 历史背景与案例

## 历史来源
- **出处**：《大明王朝1566》第十九章
- **相关机构**：浙江巡抚衙门
- **时代背景**：明代嘉靖年间

## 典型案例

### 海瑞与王用汲案
- **场景**：知县海瑞、陪审官王用汲见巡抚/主审官
- **异常操作**：上级不着官服，在签押房接见，继续起居活动
- **下属反应**：王用汲悄悄望向海瑞，海瑞直直地坐在那里
- **意图分析**：刻意安排，进行心理施压和权力测试

## 相关制度

### 《大明会典》相关规定
- 官服与便服的礼仪规范
- 不同级别官员的会面礼节
- 正式场合的仪式要求

## 相关学派
- **泰州学派**：强调心性修养，对官场礼仪有独特见解
- **政治心理学**：明代官场中的权力运作机制

## 适用对象扩展
- 省级巡抚官员
- 下级知县
- 钦案陪审官
- 主审官
- 其他存在上下级关系的明代官员
