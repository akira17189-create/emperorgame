# 锦衣卫跪拜礼制详细说明

## 私跪案例：齐大柱跪海瑞

**背景：**
- 齐大柱为锦衣卫第十三太保
- 海瑞为六品主事
- 按礼制，锦衣卫不应向六品官员跪拜

**私跪认定：**
- 齐大柱因海瑞救命之恩而下跪
- 属于个人恩情，非因海瑞官职
- 其他锦衣卫见状，侧身低头站在一边，不跟随下跪

**法律依据：**
- 私跪与职分无关
- 不违反锦衣卫礼制规定

## 特殊廷议历史背景

**时间：** 嘉靖四十五年五月初五
**事件：** 海瑞上《治安疏》后朝廷组织的特殊驳斥会议
**目的：** 驳斥海瑞在奏疏里攻击皇上的言辞，然后进行论罪

**空间格局：**
- 北墙上首（尊位）：内阁大学士
- 两侧坐垫（臣位）：监察与文学侍从
- 门外（内廷监督位）：司礼监太监

**特殊性质：**
- 都察院大堂"从来没有像今天这样摆设过"
- 表明此次廷议的特殊政治敏感性和严肃性
- 区别于普通朝会或审判

## 官员受审姿态法律条文

**《大明律》审讯官员条例：**
- 三品以上官员：可以坐着受审
- 三品以下官员：可以站着受审

**海瑞案例：**
- 官职：六品主事
- 状态：尚未定罪，尚未革职
- 行为：跪拜行礼后自己站起来
- 合法性：符合"三品以下站着受审"的法律规定

**与礼仪惯例的对比：**
- 礼仪：六品小官面对内阁阁员及六部九卿堂官需跪拜
- 法律：三品以下站着受审是法定权利
- 执行：可按惯例跪拜行礼，但受审时应站立
