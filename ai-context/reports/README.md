## 使用说明

每个实现报告包含以下内容：
1. 实现概述
2. 已实现功能
3. 文件修改清单
4. 技术细节
5. 验收标准达成情况
6. 下一步建议
7. 风险控制

## 报告类型说明

### 实现报告 (Implementation Report)
详细的实现文档，包含：
- 技术实现细节
- 代码结构分析
- 算法和公式说明
- 文件修改清单

### 完成总结 (Summary)
简明的完成总结，包含：
- 任务完成情况
- 主要成就
- 注意事项
- 下一步计划
