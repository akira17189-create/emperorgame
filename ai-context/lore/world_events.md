# 靖朝世界事件系统设计文档

**设计理念**：事件是游戏世界的「活」的体现，让玩家感受到王朝在动态演化，而不仅仅是资源数字的变化。每个事件都有多结局分支，考验玩家的决策能力。

**事件分类**：自然灾害（5个）、仙道异变（6个）、工业革命事件（5个）、党争/宦官作乱（6个）、边疆危机（5个）、民间异动（5个），共计32个事件模板。

## 一、自然灾害事件（5个）

### 1. 黄河决堤
**事件描述**："连日暴雨，黄河水位暴涨。工部急报：开封段堤坝出现裂缝，随时可能决堤！下游百万百姓危在旦夕。"

**触发条件**：
- 概率触发：每年5%基础概率
- 条件加成：land_fertility>70时概率+10%，disaster_relief<30时概率+15%
- 季节限制：夏季触发概率翻倍

**玩家决策选项**：
1. 【紧急加固】立即调拨国库白银50万两，征发民夫十万紧急加固（fiscal -50万，population -3%，成功率70%）
2. 【疏散百姓】命地方官组织百姓撤离，减少伤亡（morale -15，population -1%，无财政损失）
3. 【祈求上天】率百官祭天，祈求河神息怒（taoist +10，fiscal -5万，成功率30%）
4. 【严惩失职】追究工部官员责任，杀一儆百（eunuch -10，loyalty_to_emperor -5）

**多结局**：
- **成功加固**：堤坝保住，皇帝声望+20，disaster_relief +15
- **决堤但疏散及时**：淹没良田万亩，food -20%，但百姓感恩，morale +10
- **祈求成功**：奇迹般雨停，taoist +30，但清流派斥为迷信
- **全面溃败**：黄河改道，food -40%，population -10%，threat +25

### 2. 北方大旱
**事件描述**："连续三月无雨，北方赤地千里。田间禾苗枯死，井水干涸，灾民开始聚集，恐生变乱。"

**触发条件**：
- 概率触发：每年8%基础概率  
- 条件加成：weather_this_year="干旱"时概率+20%
- 地区限制：只在北方省份触发

**玩家决策选项**：
1. 【开仓放粮】立即开放北方所有官仓，赈济灾民（food -30%，morale +20，fiscal -10%）
2. 【组织抗旱】命工部兴修水渠，打深井（engineering +15，fiscal -25%，长期效果）
3. 【减免税赋】免除灾区三年税赋，休养生息（fiscal -15%，commerce +10%）
4. 【调兵维稳】调边军入关，严防民变（military_cost +10%，threat -5）

**多结局**：
- **有效赈灾**：灾情缓解，皇帝获称"仁君"，legitimacy +25
- **抗旱成功**：建立抗旱体系，land_fertility +10（永久增益）
- **激起民变**：赈灾不力，饥民暴动，threat +30，需要平叛

### 3. 江南水灾
**事件描述**："梅雨连绵，长江水位告急。江宁府急报：江水已漫过堤岸，城内积水三尺，商铺民居被淹。"

**触发条件**：
- 概率触发：每年6%基础概率
- 条件加成：夏季+雨季时概率+25%
- 地区限制：只在南方省份触发

**玩家决策选项**：
1. 【全力救灾】调集全国物资，派遣钦差坐镇指挥（fiscal -30%，efficiency +20）
2. 【疏导泄洪**】炸开下游分洪区，牺牲局部保全局（population -2%，threat +15）
3. 【追究责任】严惩治水不力的官员（corruption -10，loyalty_to_emperor -10）
4. 【祈求龙神**】举行盛大祭龙仪式（taoist +15，成功率40%）

**多结局**：
- **救灾得力**：损失最小化，commerce恢复快，获得"明君"称号
- **泄洪争议**：下游百姓怨恨，但保住了江宁，morale分裂
- **祭龙灵验**：奇迹般雨停，道教势力大增，taoist +25

### 4. 关中地震
**事件描述**："地动山摇！西安府急报：昨夜子时地龙翻身，房屋倒塌十之三四，死伤枕藉，余震不断。"

**触发条件**：
- 概率触发：每年3%基础概率（稀有事件）
- 无季节地区限制
- 触发后3年内不会再触发同类型事件

**玩家决策选项**：
1. 【紧急救援】调拨国库，派遣太医，全力救灾（fiscal -40%，population -3%但morale +15）
2. 【重建都城**】借机重新规划西安，修建更坚固的房屋（engineering +25，fiscal -50%但长期受益）
3. 【罪己诏】下诏罪己，认为是自己失德招致天谴（legitimacy -10，但morale +20）
4. 【镇压谣言**】严禁传播"天谴"言论，违者处斩（threat +10，loyalty_to_emperor -15）

**多结局**：
- **仁君形象**：救灾得力，百姓感恩，legitimacy +30
- **新城奇迹**：重建后的西安更繁荣，commerce +20%
- **天谴传言**：民间传言皇帝失德，legitimacy -25，需要强力辟谣

### 5. 蝗灾肆虐
**事件描述**："遮天蔽日！河南、山东多地突遭蝗灾，蝗虫过境，寸草不生，秋粮绝收已成定局。"

**触发条件**：
- 概率触发：每年7%基础概率
- 条件加成：food>80时概率+15%（粮食丰富吸引蝗虫）
- 季节限制：秋季触发

**玩家决策选项**：
1. 【组织灭蝗】动员百姓捕杀，官收蝗虫（每斤一文钱）（fiscal -10%，food损失减少50%）
2. 【减免税赋**】受灾州县免赋三年（fiscal -20%，morale +25）
3. 【祭拜虫神**】设坛祭祀，祈求蝗神离去（taoist +10，成功率30%）
4. 【严令补种**】命农民立即补种耐寒作物（agriculture +15，但劳动力紧张）

**多结局**：
- **有效控制**：损失控制在15%以内，agriculture经验+10
- **祭祀灵验**：蝗虫莫名离去，道教声望大涨
- **全面绝收**：应对不力，food -40%，可能引发饥荒

## 二、仙道异变事件（6个）

### 6. 灵气潮汐
**事件描述**："钦天监急报：昨夜观星，发现天地灵气异常波动，似有潮汐之势。灵气浓郁处，草木一夜开花；稀薄处，井水突然干涸。"

**触发条件**：
- 概率触发：每年4%基础概率
- 条件加成：taoist_resources>50时概率+20%
- 与"观星台修建"政策联动：有此政策时事件更详细

**玩家决策选项**：
1. 【命国师研究**】让国师探究灵气潮汐规律（taoist +20，可能获得"灵气预测"能力）
2. 【设聚灵阵**】在灵气浓郁处设阵，收集灵气（taoist_resources +30，但可能破坏地脉）
3. 【布告安民**】告知百姓此乃吉兆，不必惊慌（morale +10，但若处理不当有风险）
4. 【严禁议论**】以妖言惑众罪禁止讨论灵气（threat +5，taoist -10）

**多结局**：
- **掌握规律**：获得"灵气预测"技能，可预知下次潮汐
- **聚灵成功**：taoist_resources大增，但可能引发地震
- **引发恐慌**：百姓以为天谴，morale -20

### 7. 异兽现世
**事件描述**："湖广深山有猎户报：见一异兽，鹿角、牛身、马尾，周身散发五彩光芒。乡民或言祥瑞，或言妖孽，莫衷一是。"

**触发条件**：
- 概率触发：每年3%基础概率
- 条件加成：forest_coverage>60时概率+15%
- 稀有事件，触发后5年内不会重复

**玩家决策选项**：
1. 【活捉进献**】命当地官员组织围捕，献于御前（可能获得"祥瑞"加成）
2. 【封山保护**】划为禁地，严禁捕猎（taoist +15，但猎户不满）
3. 【遣道士降服**】派得道高人前往，或降或驱（taoist +25，成功率60%）
4. 【视为妖孽**】调兵剿灭，以安民心（military -5，taoist -20）

**多结局**：
- **获得祥瑞**：异兽驯服，legitimacy +30，国运+1
- **道士降服**：异兽成为护山神兽，taoist_resources +20
- **剿灭引发天谴**：异兽死后天降异象，灾祸连连

### 8. 丹药异变
**事件描述**："炼丹房突发异变！一炉'长生丹'炼制时突然爆炸，丹炉碎片四溅，三名方士重伤。更奇的是，幸存的丹药竟发出诡异绿光。"

**触发条件**：
- 必须有"炼丹术推广"政策
- 概率触发：该政策实施后每年10%概率
- taoist50时概率降低

**玩家决策选项**：
1. 【继续研究**】命太医救治伤者，继续研究变异丹药（medicine +20，但可能再次爆炸）
2. 【封存丹药**】将变异丹药封入玄铁箱，深埋地下（安全但无收益）
3. 【命死囚试药**】用死囚测试药效，观察反应（morale -15，可能获得神奇药方）
4. 【取缔炼丹**】以此为由取缔所有炼丹活动（taoist -30，但消除隐患）

**多结局**：
- **研制成功**：获得新型丹药，emperor.health +30
- **试药惨剧**：死囚变异成怪物，引发恐慌
- **炼丹禁止**：道教势力受挫，但朝廷更稳定

### 9. 风水逆转
**事件描述**："南京孝陵守陵官急报：昨夜雷雨，陵前石像生突然移位，护陵河水流逆转。风水师言：此乃龙脉异动，大凶之兆！"

**触发条件**：
- 概率触发：每年2%基础概率（极稀有）
- 条件加成：fengshui>70时概率反而+10%（风水太好引动地气）
- 必须建有皇陵

**玩家决策选项**：
1. 【急召国师**】命国师连夜赶赴，勘察龙脉（taoist +20，需要时间）
2. 【重新安葬**】开陵移棺，另择吉地（legitimacy -25，但可能化解）
3. 【设坛镇压**】在陵前设七星坛，镇压地气（taoist +15，成功率50%）
4. 【严密封锁**】严禁消息外传，违者斩（threat +10，但可能压不住）

**多结局**：
- **国师化解**：找出原因并解决，fengshui +30
- **迁陵风波**：引发朝野争议，legitimacy受损但长久安全
- **镇压失败**：地气爆发，引发地震

### 10. 梦境启示
**事件描述**："皇帝连续三夜做同一怪梦：一白须老者立于云端，手持玉简，似有所言却无声。醒来后，玉简内容竟依稀记得几分……"

**触发条件**：
- 概率触发：每年5%基础概率
- 与皇帝特质相关：有"慧眼""灵感"特质时概率翻倍
- 无季节限制

**玩家决策选项**：
1. 【命翰林解梦**】让翰林学士集体研究梦境（knowledge +15，可能解读成功）
2. 【问道于国师**】请国师以道法解梦（taoist +20，成功率更高）
3. 【置之不理**】视为寻常梦境，不予理会（无影响）
4. 【记录存档**】详细记录梦境，留待日后验证（可能触发后续事件）

**多结局**：
- **成功解读**：获得重要预言，可避开未来灾祸
- **错误解读**：错误引导决策，导致损失
- **后续事件**：数月后梦境应验，证明是预知梦

### 11. 秘境开启
**事件描述**："蜀山采药人发现一山洞，内有光华闪烁。入内探查，竟别有洞天：奇花异草，灵泉流淌，似非凡间之地。"

**触发条件**：
- 概率触发：每年3%基础概率
- 条件加成：exploration>40时概率+15%
- 地区限制：只在山地省份触发

**玩家决策选项**：
1. 【派军探查**】调精锐部队前往，探明虚实（military -5，可能获得宝物）
2. 【遣道士探索**】让道教高人前往，或能发现仙缘（taoist +25，成功率60%）
3. 【封闭洞口**】以巨石封死，免生事端（安全但无收益）
4. 【设为禁地**】划为皇家禁苑，严禁外人进入（taoist_resources +10但占用土地）

**多结局**：
- **发现仙府**：获得上古修仙传承，taoist +50
- **遭遇凶兽**：探查队伤亡惨重，但带回珍稀材料
- **秘境消失**：探索无果，秘境自行封闭

## 三、工业革命事件（5个）

### 12. 蒸汽机突破
**事件描述**："工部狂喜来报：经过三年研究，蒸汽机原型机终于成功运转！此机可抵百名壮劳力，若能推广，天下将为之一变。"

**触发条件**：
- 必须有"蒸汽机研发"政策
- 研发进度达到80%以上
- technology>60时加速触发

**玩家决策选项**：
1. 【全力推广**】拨巨资在全国推广，建立蒸汽工坊（industry +40%，fiscal -35%）
2. 【谨慎试点**】先在京畿试点，观察效果再推广（稳妥但进展慢）
3. 【限制使用**】只允许官营工坊使用，严禁民间仿制（控制技术但限制发展）
4. 【继续改进**】命工匠继续优化，追求更高效能（technology +20，延迟推广）

**多结局**：
- **工业飞跃**：迅速进入蒸汽时代，industry翻倍
- **试点成功**：稳步推进，避免了很多问题
- **技术泄露**：蒸汽机技术流传到民间，商人开始仿制

### 13. 铁路争议
**事件描述**："第一条实验性铁路在京郊建成。火车轰鸣而过，速度惊人。然而沿线百姓惊恐万分，称此'铁龙'破坏风水，惊扰祖坟。"

**触发条件**：
- 必须有"官道修建"政策且完成度>50%
- 触发概率：政策实施后第3年30%概率
- fengsh50时概率更高

**玩家决策选项**：
1. 【强力推行**】以圣旨压服反对声音，继续修路（efficiency +30，但threat +15）
2. 【调整路线**】避让风水要地，绕道修建（成本+20%，但阻力减小）
3. 【宣传教育**】组织百姓参观，消除恐惧（morale +10，但需要时间）
4. 【暂停工程**】迫于压力暂时停工（commerce -10%，但稳定民心）

**多结局**：
- **铁路时代**：成功建成铁路网，commerce +50%
- **民变四起**：强行推行引发暴动，需要平叛
- **风水灾难**：破坏龙脉，引发天灾

### 14. 工人罢工
**事件描述**："江南纺织厂千余名工人集体罢工，抗议工时长、薪资低。工人聚集厂外，与厂卫对峙，局势一触即发。"

**触发条件**：
- industry>40且持续增长
- morale<50时概率+20%
- 有"皇家工坊"政策时更易触发

**玩家决策选项**：
1. 【武力镇压**】调兵驱散，逮捕领头者（threat -10，但morale -25）
2. 【协商加薪**】命厂主适度提高薪资，改善条件（fiscal -5%，morale +15）
3. 【立法保障**】制定《工坊法》，规定最低薪资和工时（legitimacy +20，但厂主反对）
4. 【分化瓦解**】收买工头，瓦解罢工组织（corruption +10，短期有效）

**多结局**：
- **劳资和谐**：建立现代劳资关系，industry稳定增长
- **镇压成功但埋隐患**：工人表面屈服，但仇恨积累
- **立法开创先河**：成为第一个有劳工法的王朝

### 15. 技术盗窃
**事件描述**："锦衣卫密报：有洋商重金收买工部匠人，窃取蒸汽机图纸。涉事匠人已抓获，但图纸是否已流出尚不可知。"

**触发条件**：
- technology>50且持续增长
- corruption>30时概率+15%
- 与外国有贸易往来时更易触发

**玩家决策选项**：
1. 【严惩不贷**】涉事匠人凌迟，洋商驱逐出境（technology保护，但外贸受损）
2. 【将计就计**】故意泄露错误图纸，误导洋人（可能获得反向情报）
3. 【加强保密**】建立匠籍制度，严格控制技术（innovation -10，但技术安全）
4. 【技术交换**】以此为契机，与洋人谈判技术交换（可能获得新科技）

**多结局**：
- **保住优势**：技术领先优势保持
- **误导成功**：洋人研发走上歧路，我方优势扩大
- **技术外流**：洋人迅速仿制，技术优势丧失

  // 决策选项
  options: EventOption[];
  
  // 结局（根据选项和随机因素生成）
  possibleOutcomes: EventOutcome[];
  
  // 与其他事件的关联
  relatedEvents: {
    unlocks?: string[];  // 可能解锁的事件
    prevents?: string[]; // 可能阻止的事件
    chainTo?: string;    // 连锁事件
  };
  
  severity: 1 | 2 | 3 | 4 | 5;  // 严重程度
  duration: number;  // 预计持续时间（Tick数）
}

export interface EventOption {
  id: string;
  text: string;
  costs: ResourceCost[];
  successRate: number;  // 0-1
  requiredResources?: Record<string, number>;  // 所需资源阈值
  characterTraitModifiers?: Record<string, number>;  // 角色特质影响成功率
}

export interface EventOutcome {
  type: 'success' | 'partial' | 'failure' | 'catastrophe';
  conditions: string;  // 达成条件描述
  effects: {
    resources: Record<string, number>;
    factions: Record<string, number>;
    flags: string[];  // 设置的游戏标志
  };
  narrative: {
    immediate: string;
    followup?: string;
    chronicle: string;
  };
}

## 八、事件系统与现有引擎衔接

1. **触发检查**：每个Tick开始时，tick.ts调用事件触发器
2. **条件评估**：根据当前状态计算每个事件的触发概率
3. **事件呈现**：触发的事件通过UI呈现给玩家，附带决策选项
4. **决策处理**：玩家选择后，计算结局（考虑成功率和随机因素）
5. **效果应用**：应用结局效果到游戏状态
6. **叙事生成**：生成事件叙事，并入史册
7. **连锁反应**：可能触发后续事件或影响其他系统

**事件重要性分级**：
- **1-2级**：小事件，自动处理，只生成简叙事
- **3级**：需要玩家决策，但后果较小
- **4-5级**：重大事件，可能改变游戏走向，需要慎重决策

**避免事件疲劳**：
- 同一时期最多一个4-5级事件
- 同类型事件有冷却时间
- 季节性事件只在对应季节触发
- 考虑玩家的决策负担
