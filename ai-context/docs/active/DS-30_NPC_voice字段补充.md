# DS-30：核心NPC voice字段补充

## 说明
voice字段用于定义NPC的个性化语言风格和口头禅，帮助Mimo生成符合角色性格的对话。每个voice字段应包含以下要素：
1. 说话风格（正式/随意/文雅/粗俗等）
2. 常用词汇/口头禅
3. 句式特点（长短、结构）
4. 情绪表达方式
5. 派系特征语言

## voice字段使用规则

### 1. 对话生成规则
- **直接对话**: 当NPC与玩家直接对话时，严格遵循voice字段的语言风格
- **间接引用**: 当NPC的话被第三方转述时，保留核心风格特征
- **情绪变化**: 根据情境调整语气，但基本语言特征不变

### 2. 派系特征优先级
1. 个人性格特征 > 派系共同特征
2. 特殊身份特征（如太监、女性伪装）> 一般官员特征  
3. 职业特征（如武将、商人、医生）> 政治派系特征

### 3. 情境适应性
- **正式场合**: 语言更加规范，减少口头禅
- **私下场合**: 语言更加随意，个性特征更明显
- **情绪激动时**: 特征词汇和句式更加突出
- **紧张/害怕时**: 可能出现口吃、重复等临时特征

### 4. 避免事项
- 避免所有NPC说话风格雷同
- 避免voice特征过于夸张，失去真实感
- 避免特征词汇过度重复，造成审美疲劳
- 避免voice与角色行为严重不符

## 给Mimo的使用说明

### 对话生成流程
1. 识别对话NPC的身份和派系
2. 查找对应的voice字段描述
3. 提取关键特征：说话风格、口头禅、句式特点
4. 根据对话情境调整语气和内容
5. 生成符合角色特征的对话

### 示例应用
**场景**: 皇帝询问陈四海关于减税政策的影响

**不正确的生成**:
> "陛下，减税会让百姓高兴，但国库会减少收入。"

**正确的生成（根据voice字段）**:
> "陛下，据臣统计，减税三成将使今年税收减少一百二十万两。从账面上看，国库确实会吃紧。但若能刺激商贸，长期来看或有回报。只是这短期缺口...需要从别处找补。"

### 验证标准
1. 对话是否符合角色的派系立场？
2. 对话是否体现了角色的职业/身份特征？
3. 对话中是否包含角色的特征词汇或句式？
4. 对话语气是否符合情境（正式/私下）？
5. 多个NPC同时出现时，对话风格是否有明显区分？
