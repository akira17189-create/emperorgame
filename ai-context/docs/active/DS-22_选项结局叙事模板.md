# DS-22：选项结局叙事模板（用于LLM生成）

## 模板使用说明

1. **占位符说明**：
   - `{{政策效果数值}}`：游戏引擎计算的政策效果数值（如+10、-5等）
   - `{{事件名称}}`：当前事件的名称（如"黄淮大旱"）
   - `{{选项标签}}`：玩家选择的选项标签（如"开仓赈灾"）
   - `{{派系名称}}`：相关派系名称（如"清流派"、"务实派"等）
   - `{{地点}}`：事件发生地点（如"河南"、"江南"等）
   - `{{时间跨度}}`：事件发生后的时间跨度（如"三个月后"、"半年后"等）

2. **模板结构**：
   - 第一段：描述事件发生后，选项实施的具体过程
   - 第二段：描述实施过程中的困难和挑战
   - 第三段：描述实施效果和各方反应
   - 第四段：总结性叙述，连接下一个事件

### 注意事项

1. **保持风格一致性**：所有叙述应符合游戏世界观和叙事风格
2. **体现派系差异**：不同派系对同一事件应有不同反应
3. **连接游戏机制**：叙事应与游戏数值变化（{{政策效果数值}}）相呼应
4. **预留扩展空间**：每个模板都应能根据具体事件灵活调整
