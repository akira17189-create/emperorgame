## 使用说明

1. **Mimo将使用此矩阵自动生成NPC对政策的反应**
2. 计算政策综合态度时，取政策所有标签的态度平均值
3. 例如政策有【内政+1、财政-1】两个标签，则综合态度 = (1 + (-1)) / 2 = 0（中立）
4. 综合态度决定NPC的立场（支持/反对/中立）和发言激烈程度
5. 具体台词仍需要结合NPC个人性格特征生成

## 各派系核心立场

### 清流派
- **强烈支持**：廉政、言论自由、民生
- **强烈反对**：仙道、穿越技术
- **特点**：理想主义，重道德轻实务

### 帝党派
- **支持**：内政稳定、军事、外交
- **中立偏支持**：仙道（讨好皇帝）
- **特点**：务实保守，维护皇权稳定

### 军队派
- **强烈支持**：军事
- **支持**：财政（为军费）、工业（为军械）
- **强烈反对**：仙道
- **特点**：直接实际，重实力轻空谈

### 宦官派
- **强烈支持**：仙道（可控制皇帝）
- **强烈反对**：廉政、言论自由
- **支持**：财政、基建、商业（可捞油水）
- **特点**：利益导向，维护自身权力

### 务实派
- **强烈支持**：财政、农业、民生、工业、基建、商业、穿越技术
- **强烈反对**：仙道
- **特点**：数据驱动，注重实际效益
