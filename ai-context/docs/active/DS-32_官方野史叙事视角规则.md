# DS-32：官方/野史叙事视角规则

## 核心概念

### 两种叙事视角
1. **官方视角**：代表朝廷/皇帝的正式立场，用于奏折、诏书、史官记录
2. **野史视角**：代表民间/反对派的非正式立场，用于流言、笔记小说、民间传说

### 叙事差异维度
| 维度 | 官方叙事 | 野史叙事 |
|------|----------|----------|
| **事实选择** | 选择性报道有利事实 | 选择性报道不利事实 |
| **因果解释** | 归因于天意/忠臣 | 归因于人祸/奸臣 |
| **价值判断** | 符合儒家正统 | 质疑官方说辞 |
| **语言风格** | 庄重典雅 | 生动鲜活 |
| **传播渠道** | 官方文书 | 口耳相传/地下刊印 |

## 一、自然灾害类事件叙事规则

### 官方叙事模板
**结构**: 天象示警 → 圣主应对 → 君臣努力 → 灾情缓解 → 皇恩浩荡

**关键要素**:
- 强调"天意难测"（为皇帝卸责）
- 突出"陛下英明决断"
- 表彰"忠臣良将"
- 弱化灾情严重程度
- 强调赈灾效果

**示例**: 黄淮大旱
> "永德三年夏，黄淮少雨。陛下忧心忡忡，夜不能寐，召集群臣商议。下旨开仓赈灾，命户部调粮三十万石，遣能臣干吏督办。虽天灾无情，然皇恩浩荡，百姓感念圣德，未有大规模流亡。此乃陛下仁德感天，群臣协力之果也。"

### 野史叙事模板
**结构**: 天怒示警 → 朝廷失德 → 官员腐败 → 百姓遭殃 → 民怨沸腾

**关键要素**:
- 强调"天怒人怨"（指责皇帝）
- 揭露"官员贪腐"
- 描绘"民间惨状"
- 质疑"赈灾实效"
- 传播"灵异征兆"

**示例**: 黄淮大旱
> "听说了吗？黄淮那边三年没下雨了！为啥？朝廷失德呗！宫里天天炼丹修仙，哪管百姓死活？赈灾粮？哼，发到手里只剩三成，剩下七成都进了贪官口袋！村口老王家三个儿子，饿死俩，剩一个卖给了人牙子。夜里还能听到鬼哭呢，说是不甘心..."

## 二、仙道异变类事件叙事规则

### 官方叙事模板
**结构**: 祥瑞现世 → 圣主感应 → 国师解兆 → 顺应天意 → 国运昌隆

**关键要素**:
- 定性为"祥瑞吉兆"
- 强调"陛下圣明感应"
- 突出"国师专业解读"
- 连接"国运昌盛"
- 压制"负面解读"

**示例**: 仙鹤衔书
> "乙未年三月初七，有仙鹤翔于太庙，衔素帛而降。国师玄明观之，曰：'此乃天降祥瑞，主陛下圣德感天，国祚绵长。'陛下谦逊不受，言'此皆祖宗庇佑，群臣辅佐之功'。命将帛书藏于太庙，岁时祭祀。群臣称贺，万民欢腾。"

### 野史叙事模板  
**结构**: 妖异现世 → 朝廷掩盖 → 道士装神 → 皇帝迷信 → 祸国殃民

**关键要素**:
- 定性为"妖异凶兆"
- 揭露"朝廷掩盖真相"
- 指控"道士装神弄鬼"
- 讽刺"皇帝昏庸迷信"
- 预言"大祸将至"

**示例**: 仙鹤衔书
> "知道太庙那事儿吗？什么仙鹤，分明是有人训练的白鹤！帛书上写的啥？'龙潜于渊'——那不是咒皇上吗？国师硬说是吉兆，哄鬼呢！听说宫里那位信得不得了，天天对着那破布磕头。等着瞧吧，不出三年，准出大事！"

## 三、党争类事件叙事规则

### 官方叙事模板
**结构**: 奸佞误国 → 忠臣进谏 → 陛下明察 → 惩恶扬善 → 朝纲肃清

**关键要素**:
- 定性为"忠奸斗争"
- 突出"陛下圣明裁决"
- 强调"法度严明"
- 弱化"派系斗争"
- 宣扬"政治清明"

**示例**: 司礼监专权
> "司礼监掌印太监王福全，倚仗内廷便利，越权干政，私拟奏折。都察院御史风闻奏事，陛下震怒，下旨彻查。查实其罪十三条，依律革职查办。陛下谕曰：'内臣干政，祖宗大忌。此后司礼监仅司文书传递，不得干预朝政。'朝野称快，纲纪为之一新。"

### 野史叙事模板
**结构**: 权力斗争 → 黑幕重重 → 皇帝偏袒 → 派系倾轧 → 朝政混乱

**关键要素**:
- 揭露"权力斗争本质"
- 曝光"黑幕交易"
- 质疑"皇帝公正性"
- 描绘"派系倾轧惨烈"
- 感叹"朝政黑暗"

**示例**: 司礼监专权
> "王公公那事儿？嘿，哪是什么忠奸斗争，就是分赃不均！听说他吞了江南盐税三成，东厂那边眼红，才把他捅出来。皇上？皇上早就知道！为啥现在才办？因为王公公最近跟太后走得太近，威胁到皇上了呗！办了一个王福全，还有张福全、李福全，换汤不换药！"

## 四、边疆危机类事件叙事规则

### 官方叙事模板
**结构**: 外敌挑衅 → 陛下决断 → 将士用命 → 大捷凯旋 → 天威远播

**关键要素**:
- 强调"外敌无理挑衅"
- 突出"陛下英明决策"
- 表彰"将士忠勇"
- 宣扬"天朝威仪"
- 弱化"战争代价"

**示例**: 北疆游牧扣边
> "鞑靼犯边，掠我子民。陛下闻讯，龙颜大怒：'蕞尔小丑，敢犯天威！'即命蓟辽总督徐达率军五万迎击。徐将军身先士卒，将士用命，三战三捷，斩首八千，鞑靼远遁三百里。捷报传来，陛下犒赏三军，曰：'有此虎贲，何愁边疆不宁？'"

### 野史叙事模板
**结构**: 边将挑衅 → 朝廷失策 → 战事不利 → 隐瞒伤亡 → 百姓遭殃

**关键要素**:
- 揭露"边将挑衅真相"
- 批评"朝廷决策失误"
- 曝光"战事真实情况"
- 指控"隐瞒伤亡数字"
- 描绘"百姓双重苦难"

**示例**: 北疆游牧扣边
> "什么鞑靼犯边，分明是徐达那厮杀良冒功，把人家放牧的百姓当鞑靼杀了！人家来报仇，他又打不过，只好求援。朝廷派了五万人去，死了多少？八千？那是斩首数！咱们自己死了至少两万！尸首都运不回来，就地埋了。家里等着儿子丈夫回来的，等到的是啥？一纸'阵亡通知'和几两抚恤银！"

## 五、工业事故类事件叙事规则

### 官方叙事模板
**结构**: 意外事故 → 朝廷重视 → 全力救援 → 妥善处理 → 改进防范

**关键要素**:
- 定性为"意外事故"
- 强调"朝廷高度重视"
- 表彰"救援人员"
- 说明"善后措施"
- 承诺"改进防范"

**示例**: 矿山塌方
> "江西铜矿因连日大雨，发生局部塌方。陛下闻讯，即命工部、户部协同救援，并拨专款抚恤。经三日全力施救，救出矿工三十余人，不幸遇难者厚加抚恤。陛下谕令：'人命关天，各矿区须加强安全防范，杜绝此类事故。'"

### 野史叙事模板
**结构**: 人为灾难 → 官员渎职 → 救援迟缓 → 掩盖真相 → 草菅人命

**关键要素**:
- 揭露"人为因素"
- 指控"官员渎职腐败"
- 曝光"救援不力真相"
- 揭露"死亡数字隐瞒"
- 控诉"草菅人命"

**示例**: 矿山塌方
> "塌方？那哪是天灾，是人祸！矿洞早就该加固了，矿主为了省钱就是不弄！出事了，地方官第一时间不是救人，是封锁消息！等朝廷知道，人都死透了！说是救出三十多个，其实只活了十几个。死了多少人？至少两百！家属去闹，一人发十两银子就打发了。人命啊，比草还贱！"

## 六、民间异动类事件叙事规则

### 官方叙事模板
**结构**: 刁民闹事 → 官员劝导 → 依法处置 → 恢复秩序 → 警示后人

**关键要素**:
- 定性为"刁民闹事"
- 强调"官员耐心劝导"
- 说明"依法处置依据"
- 宣扬"秩序恢复"
- 警示"不可效仿"

**示例**: 饥民暴动
> "河南数县，有刁民聚众闹事，冲击县衙。地方官员反复劝导无效，为保一方安宁，不得已调兵弹压。首恶已擒，胁从者遣散。陛下仁慈，仍命开仓放粮，救济灾民。谕曰：'民虽有困，然不可违法。各州县当以此为戒，妥善安抚。'"

### 野史叙事模板
**结构**: 官逼民反 → 百姓无奈 → 血腥镇压 → 冤魂不散 → 天道何在

**关键要素**:
- 强调"官逼民反"
- 描绘"百姓无奈处境"
- 控诉"血腥镇压暴行"
- 渲染"冤魂灵异"
- 质问"天理公道"

**示例**: 饥民暴动
> "暴动？那是活不下去了！三年大旱，朝廷一粒粮不发，县太爷家的粮仓堆得冒尖。老百姓去借粮，被衙役打得头破血流。不反？等死啊！领头的张秀才，多好的人，被抓住砍了头，挂在城墙上示众。夜里经过那里，还能听到他喊'冤枉'呢！这世道，哪有天理！"

## 七、叙事视角切换规则

### 1. 切换触发条件
- **玩家选择**: 根据玩家选择查看官方文书或民间传闻
- **派系关系**: 不同派系NPC会提供不同视角信息
- **情报来源**: 锦衣卫密报 vs 酒馆流言
- **时间推移**: 当时报道 vs 后世评价

### 2. 信息完整度规则
- **官方信息**: 全面但美化，隐藏负面信息
- **野史信息**: 片面但尖锐，夸大负面信息
- **真相位置**: 介于两者之间，需要玩家判断

### 3. 可信度评估体系
```
官方叙事可信度 = 基础50% + 皇帝声望×0.5 + 事件透明度×0.3
野史叙事可信度 = 基础30% + 民间不满度×0.6 + 传播广度×0.4
综合真相值 = (官方可信度×官方信息 + 野史可信度×野史信息) / (官方可信度+野史可信度)
```

### 4. 叙事冲突处理
- **直接矛盾**: 保留矛盾，让玩家判断
- **部分矛盾**: 取交集为事实，差异为观点
- **角度差异**: 保留多角度叙述
- **明显虚假**: 后续事件中揭穿

## 八、叙事风格差异规范

### 语言风格对比
| 特征 | 官方叙事 | 野史叙事 |
|------|----------|----------|
| **词汇选择** | 文言雅词 | 白话俗语 |
| **句式结构** | 工整对仗 | 松散随意 |
| **人称使用** | 陛下、臣、朕 | 皇上、官家、那老头 |
| **敬语系统** | 完整严谨 | 随意甚至不敬 |
| **修辞手法** | 用典、隐喻 | 夸张、比喻、谐音 |

### 情感表达对比
| 情感 | 官方叙事 | 野史叙事 |
|------|----------|----------|
| **赞扬** | "圣德巍巍"、"功勋卓著" | "真有两下子"、"还算有良心" |
| **批评** | "或有不当"、"尚需斟酌" | "狗官"、"王八蛋"、"不是东西" |
| **悲伤** | "朕心甚恸"、"深表哀悼" | "惨啊"、"可怜见的"、"造孽啊" |
| **愤怒** | "龙颜震怒"、"殊为可恨" | "气死我了"、"肺都炸了"、"他奶奶的" |
| **恐惧** | "朕心忧之"、"深以为虑" | "吓死人了"、"腿都软了"、"妈呀" |

### 传播方式对比
| 方式 | 官方叙事 | 野史叙事 |
|------|----------|----------|
| **载体** | 奏折、诏书、邸报、史书 | 流言、小曲、话本、墙头诗 |
| **传播者** | 官员、太监、驿卒 | 说书人、乞丐、商贩、妓女 |
| **速度** | 较慢（层层传递） | 很快（口耳相传） |
| **准确性** | 较高但被美化 | 较低但生动 |
| **持久性** | 持久（载入史册） | 短暂（易于遗忘） |

## 九、给Mimo的叙事生成指南

### 1. 双轨叙事生成流程
```
事件发生 → 收集基本信息 → 生成官方版本 → 生成野史版本 → 分别存储
```

### 2. 官方版本生成要点
- 使用正式文书格式
- 遵循儒家政治正确
- 突出皇帝英明决策
- 弱化或掩盖负面信息
- 结尾必须正面肯定

### 3. 野史版本生成要点
- 使用民间口语风格
- 体现阶级立场（百姓视角）
- 突出官员腐败无能
- 夸张或虚构细节增强传播性
- 结尾往往悲观或质疑

### 4. 叙事一致性维护
- **核心事实一致**: 时间、地点、人物等硬事实必须一致
- **解释角度不同**: 同一事实可以有相反的解释
- **细节选择差异**: 官方选择正面细节，野史选择负面细节
- **价值判断对立**: 善与恶、功与过、对与错的评价完全相反

### 5. 玩家引导策略
- **初期**: 主要接触官方叙事
- **中期**: 开始接触野史叙事，产生怀疑
- **后期**: 需要综合判断，接近真相
- **结局**: 根据选择，接受一种叙事或发现真相

## 十、应用场景示例

### 场景1：奏折 vs 酒馆流言
**事件**: 江南水患

**官方奏折**:
> "臣江苏巡抚某某谨奏：今夏梅雨过甚，长江水位略涨。臣已命加固堤防，疏散低洼民众。蒙陛下洪福，未酿大灾。现水势已退，百姓安居，田亩无损。"

**酒馆流言**:
> "听说了吗？长江决堤了！淹了十几个县！为啥决堤？堤坝是豆腐渣！修堤的银子，七成进了巡抚口袋！淹死多少人？少说几千！尸体都漂到出海口了！官府？官府在忙着捞油水呢，哪管百姓死活！"

### 场景2：圣旨 vs 童谣
**事件**: 加征赋税

**圣旨**:
> "奉天承运皇帝诏曰：今边疆不宁，军费浩繁。为保社稷，特加征天下田赋一成。此乃不得已之举，待海内承平，即行减免。钦此。"

**童谣**:
> "皇帝爷，缺钱花，加赋税，刮地皮。东家哭，西家号，卖儿卖女交皇粮。官老爷，肥流油，小百姓，瘦成猴。老天爷，快睁眼，劈死这些吸血鬼！"

### 场景3：史书记载 vs 笔记小说
**事件**: 御驾亲征

**《靖史·本纪》**:
> "永德五年，帝亲征漠北。龙旗所指，三军振奋。帝与将士同甘共苦，身先士卒。大破鞑靼于阴山，斩首三万，虏获无算。凯旋之日，万民夹道，山呼万岁。"

**《京华秘闻录》**:
> "嘿，那场亲征！皇上根本不会打仗，全靠徐达将军撑着。迷路三次，差点被俘。最可笑的是，有天晚上听见狼嚎，吓得尿裤子！斩首三万？杀的都是老百姓，割了脑袋充军功！凯旋？那是逃命回来的！"

## 附录：叙事视角组合策略

### 1. 纯官方视角
- 适用于初期教学关卡
- 玩家完全信任朝廷时
- 历史课模式

### 2. 纯野史视角
- 适用于地下反抗组织剧情
- 玩家与朝廷对立时
- 民间传说模式

### 3. 双轨并行
- 主要游戏模式
- 玩家需要判断真相
- 增强代入感和思考深度

### 4. 视角切换
- 根据派系关系自动切换
- 玩家选择调查方向时切换
- 重大事件揭露时强制切换

### 5. 视角融合
- 游戏后期解锁
- 玩家获得完整情报时
- 接近真相的叙述

---

**最终目标**: 通过官方/野史的双重叙事，让玩家体验历史的复杂性，理解"历史由胜利者书写"的深刻内涵，并在辨别真伪的过程中，形成自己的历史观和价值观。
