# 瘟疫期间错误处方纠正流程

## 适用场景
- 瘟疫期间灾民治疗
- 医疗专家判定现有治疗方案无效或存在风险
- 需要纠正行政官员基于医书但不适用的处方
- 存在潜在医疗事故风险

## 触发条件
医疗专家检查药材及药汤后判定不可用。

## 纠正流程

### 1. 现场检查
专家亲自检查熬药用的药材筐和正在熬制的药汤。

### 2. 质询与验证
- 询问负责熬药的人员郎中在哪里
- 尝试获取药汤进行验证

### 3. 否定错误方案
- 果断行动：将药泼在地上（阻止分发）
- 明确宣布："这药不能吃"

### 4. 应对行政辩解
若行政人员辩解称使用的是权威医书（如《千金方》）中的解暑清热方：
- 指出仅凭单一医书给大规模人群熬药治病的风险（"胆子太大"）
- 强调医书方剂需因时因地因人调整

### 5. 执行纠正
1. 要求行政人员立即安排一间独立的屋子
2. 召集所有负责抓药和熬药的手下人员
3. 专家重新开具处方
4. 指挥人员立刻按照新处方重新抓药

## 约束条件
- 不适用于专家确认处方正确的情况
- 不适用于非医疗人员的主观臆断
- 需有更高水平的医疗专家在场
