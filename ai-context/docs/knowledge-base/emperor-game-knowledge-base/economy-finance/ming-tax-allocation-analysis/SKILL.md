# 明代淞江棉纺业税收与利润分配分析

分析淞江棉纺业在官绅免税祖制下的利润分配结构与税收征收困境。

## 适用场景

- 计算朝廷从棉纺业实际获得的税收占比
- 分析官绅免税机制导致的财政流失
- 评估"六三一"分财方案的实际执行效果
- 诊断土地兼并与国库空虚的因果关系

## 核心机制

### 一、利润分配结构（"六三一"方案）

| 分配对象 | 比例 | 说明 |
|---------|------|------|
| 田主与棉商 | 60% | 土地与商业资本收益 |
| 朝廷 | 30% | 理论税收目标 |
| 百姓 | 10% | 实际负担沉重 |

> ⚠️ 关键矛盾：此方案被裕王质疑与"改稻为桑"无异，百姓实际所得远低于名义比例。

### 二、税收漏洞机制

**祖制限制**：
- 官绅田地免税，无论种植稻麦或棉花
- 官绅不直接贩运棉布，仅待棉商上门收购
- 官府只能在厘卡向棉商征收商税

**征收瓶颈**：
- 常规商税率：10%（十成抽一）
- 征收节点：仅能在流通环节（厘卡）征收
- 无法触及：官绅自产棉布的生产环节利润

## 分析流程

### 步骤1：确定理论利润基数

获取棉布产出的纯利总量（`total_profit`），单位为匹。

### 步骤2：计算理论税收与实际税收

```
理论朝廷税收 = total_profit × 30%
常规商税收入 = total_profit × 10%
实际征收税收 = 通过特殊手段（如产地直接收购收税）的实收数量
```

### 步骤3：评估税收流失程度

```
税收流失率 = (理论朝廷税收 - 实际征收税收) / 理论朝廷税收 × 100%
实际征收占比 = 实际征收税收 / total_profit × 100%
```

### 步骤4：分析制度性后果

- **土地兼并**：百姓不堪重赋，售田于藩王官绅
- **国库空虚**：扩种棉田无法充实朝廷财政
- **权力寻租**：需依赖特殊渠道（如李娘娘弟弟）才能提高征收效率

## 快速参考：典型数值

| 指标 | 数值 | 说明 |
|-----|------|------|
| 理论利润分成 | 30% | 朝廷应得比例 |
| 常规商税上限 | 10% | 仅通过厘卡征收 |
| 实际征收案例 | 25% | 5万匹/20万匹（需特殊手段） |
| 税收缺口 | 5个百分点 | 30%理论值 vs 25%实际值 |
