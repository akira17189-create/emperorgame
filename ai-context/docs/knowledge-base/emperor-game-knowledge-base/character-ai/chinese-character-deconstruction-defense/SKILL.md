# 汉字字义拆解辩护

## 适用场景
当面临政治对手的道德或忠诚度指控，且指控涉及特定汉字（尤其是带有贬义或道德色彩的汉字）时使用此技能。此技能适用于政治辩论、法庭辩护或公开对峙场景。

## 执行步骤

### 1. 拆解汉字结构
分析被指控汉字的字形组成，识别其偏旁部首和构成部件。

### 2. 建立语义关联
根据字形部件，建立与特定行为或道德标准的关联。通常利用偏旁部首的传统含义（如“女”字旁与男女关系）进行引申，构建该字义所隐含的负面行为特征。

### 3. 收集对比事实
- **自身事实**：列举自己在该语义关联方面的清白或正面表现（如家庭状况、个人品行）。
- **对手事实**：列举对手在该语义关联方面的负面行为或具体事实。

### 4. 逻辑归谬与反击
基于上述事实对比，论证该汉字的定义特征完全符合对手的行为模式，而与自身情况严重不符。

### 5. 输出结论
明确拒绝该指控，并断言该字义更适用于对手，从而完成反驳并反向攻击。

## 关键变量
- **指控字词**：对手使用的核心汉字。
- **自身事实**：用于证明自身清白的具体事例。
- **对手事实**：用于证明对手符合该字义的具体事例。

## 预期结果
通过字义与事实的巧妙结合，证明指控不成立，并暗示指控者自身才符合该指控。
