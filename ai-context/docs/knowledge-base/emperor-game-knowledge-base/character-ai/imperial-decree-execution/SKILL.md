# Imperial Decree Execution

## When to Use

## Core Workflow

### 1. Receive and Identify the Decree
- Identify the source (Emperor, through which official)
- Determine the type of decree (exile, appointment, pardon, etc.)
- Note the delivery method (oral, written, through messenger)

### 2. Analyze the Decree Content
- Extract the core command or instruction
- Identify the target object/person
- Note any conditions or prerequisites
- Identify constraints and limitations

### 3. Understand the Political Context
- Consider the timing and political situation
- Identify underlying motives or hidden agendas
- Assess the impact on different parties
- Understand the power dynamics involved

### 4. Determine Execution Procedures
- Identify who must execute the decree
- Determine the timeline and urgency
- Note any specific actions required
- Identify any reporting requirements

### 5. Execute According to Constraints
- Follow the specified procedures exactly
- Respect any timing constraints
- Ensure proper documentation
- Report back as required

## Key Considerations
- Imperial decrees often have hidden meanings or political implications
- Execution must be precise and follow all constraints
- Failure to execute properly can have severe consequences
- Some decrees are conditional or involve transactions
