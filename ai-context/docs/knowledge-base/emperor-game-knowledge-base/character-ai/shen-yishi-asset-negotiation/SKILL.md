# 沈一石资产交易与处置谈判

## 资产折算方法

计算五座作坊资产价值时（以沈一石作坊为例）：

1. **当年产量**：一万三千匹丝绸

2. **桑田资产折算**：
   - 面积：六万五千亩的五分之一=一万三千亩
   - 单价：每亩五十石，折合现银五十两
   - 价值：13,000亩×50两=650,000两=六万五千匹丝绸

3. **绸缎庄资产折算**：
   - 数量：杭州、苏州、南京、扬州共一百零七家，五座作坊分得二十家
   - 单价：一个铺面五千两银子
   - 价值：20家×5,000两=100,000两=一万匹丝绸

4. **粮债权折算**：
   - 总量：一百船粮食，每船一万八千石，共计一百八十万石
   - 五座作坊份额：三十六万石
   - 价值：折合上等丝绸三万六千匹

5. **初始硬账总计**：
   - 当年产量13,000匹+桑田65,000匹+绸缎庄10,000匹+粮债36,000匹=124,000匹
   - 需缴官里：十万匹
   - 注：管事称八万八千匹存在计算口径争议，需核实

6. **生丝成本扣除**（徽商反算）：
   - 织十万匹丝绸需生丝二十万两银子=二万匹
   - 真实成本：资产折算额-生丝成本-上缴定额=实际盈亏

## 谈判策略与优惠条件

与徽商谈判时，官府可逐轮追加优惠：

**第一轮优惠（免税+半价收丝）**：
- 身份认定：接手者成为织造局宫差、浙江官府宫差
- 作坊合并：原五座可再并入五座，共十座
- 免税政策：十座作坊从今年起一律免交赋税
- 生丝供应：每年所需生丝一律以官价（市价一半）由官府代收购
- 效果：今年亏损降至四万四千两

**第二轮优惠（减少上缴定额）**：
- 每年每人十座作坊向宫里缴五万匹（原十万匹）
- 风险：需确认江南织造局和宫里是否认可

**第三轮优惠（何茂才追加）**：
- 从明年起每人每年再减一万匹（即缴四万匹）
- 但再减的一万匹中，五千匹给巡抚衙门和布政使衙门作为"走脚跑腿"费用
- 实际净减：每人每年缴四万五千匹给宫里，五千匹给官府

## 违规识别

识别资产处置违规行为：

- **圣旨要求**：抄没沈一石家产充归国库，追缴赃款
- **违规事实**：织造局未执行抄没，而是转卖给胡宗宪亲谊（徽商）
- **关键证据**：赵贞吉（主审官）签署接手家产的约书
- **后果**：抄家等于没抄，追缴赃款等于没追，国库依然亏空
