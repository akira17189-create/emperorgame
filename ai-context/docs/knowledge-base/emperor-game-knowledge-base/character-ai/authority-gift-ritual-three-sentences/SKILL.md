# "三句话"赐恩仪式

## 何时使用

- 上级决定赏赐亲信下属
- 下属表现出极度敬畏
- 需要强化恩义纽带
- 非公开场合

## 环境营造

- **色调**：全红基调（红灯笼、红烛、红丝帐、红被、红坐垫）
- **氛围**：喜庆且带有压迫感
- **座次**：上级与赏赐对象同席，指示下属坐特定位置

## 仪式流程

### 第一环节：安全示警（河豚汤）

1. 端上珍稀食物（河豚汤）
2. 观察下属反应（若视如毒药不敢动）
3. 上级亲自试吃，强调特意准备
4. 提醒"慢慢喝"，缓解紧张

### 第二环节：建立饮酒契约

- 倒满一大杯酒
- 明确规则：**"也就三句话，喝一杯说一句。先把这杯喝了。"**

### 第三环节：执行"三句话"

| 顺序 | 内容 | 目的 | 预期反应 |
|-----|------|------|---------|
| 第一句 | 揭露下属内心隐秘欲望或言论 | 揭短/施压 | 极度惊恐、下跪磕头 |
| 第二句 | 询问上级平日待遇 | 索恩/确认 | 表达感激，强化恩义 |
| 第三句 | 宣布成全第一句中的欲望 | 赐予/成全 | 态度转变 |

### 第四环节：离场与交付

- 上级说完三句后起身离开，反手关门
- 下属醒悟后态度从恐惧转为得意

## 关键成功因素

- 环境压迫感与恩宠感的平衡
- 精准掌握下属隐秘欲望
- 三句话的递进节奏控制
