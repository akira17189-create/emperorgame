# 严党渐进式清算详细过程

## 时间线

### 嘉靖四十一年（1562年）

**严嵩倒台**
- 严嵩、严世蕃父子被弹劾
- 嘉靖帝赐严嵩致仕（退休）
- 保留年赏禄米一百石的待遇

**严世蕃处理**
- 严世蕃被论罪流放
- 多数严党官员依然在位
- "奢靡贪墨搜刮之风无稍遏减"

### 嘉靖四十二年（1563年）

**政治僵持**
- 严党势力依然强大
- 清流官员无法完全掌权
- 国库空虚，灾情频发

### 嘉靖四十三年（1564年）

**舆论发酵**
- 民间对嘉靖帝的不满增加
- "天下不耻嘉靖已甚"
- 清流官员准备再度上疏

### 嘉靖四十四年（1565年）

**清算升级**
- 徐阶、高拱、张居正策动御史再度上疏
- 五月嘉靖帝诛杀严世蕃
- 严党势力基本瓦解

## 六必居牌匾观测记录

### 牌匾信息
- 题写人：严嵩
- 位置：北京前门外粮食店街六必居酱菜铺
- 内容："六必居"三个大字
- 状态：被日光照得熠熠生辉

### 民众反应记录

**避道而行**
- 多数行人选择绕道
- 不愿从牌匾下经过
- 显示对严嵩的回避

**侧目而视**
- 部分行人侧目而视
- 表现出不满或鄙视
- 显示对时政的不满

**匆匆而过**
- 匆匆而过，不愿停留
- 显示对政治话题的回避

**停下议论**
- 少数人停下议论
- 表达对时政的不满
- "天下不耻嘉靖已甚"

## 政治心理分析

### 嘉靖帝的焦虑
- 对自身统治合法性的焦虑
- 对臣民忠诚度的怀疑
- 对舆论风向的担忧

### 测试的目的
- 评估"天下不耻嘉靖"的舆论强度
- 评估政治不满情绪
- 为后续政策调整提供依据

### 皇权的展示
- 保留罪臣符号显示皇权的绝对控制
- "朕想留就留，想取就取"
- 不受舆论影响

## 策略效果

### 正面效果
- 避免了政治动荡
- 逐步瓦解了严党势力
- 测试了舆论风向

### 负面效果
- "天下不耻嘉靖已甚"
- 民间对嘉靖帝的不满增加
- 清流官员对嘉靖帝的批评增加
