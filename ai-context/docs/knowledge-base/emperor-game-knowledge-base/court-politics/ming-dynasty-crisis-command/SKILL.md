# 明代海防与危机指挥

## 适用场景
- 浙江台州沿海海防指挥
- 倭寇进犯路径与失陷风险预测
- 重大洪涝险情（多县同时告急）
- 常规抢险手段失效的极端情况
- 总督/巡抚级地方军政长官亲赴一线指挥

## 沿海要地失陷连锁预测

### 首要目标风险评估
- **评估结论**：'今晚倭寇进犯的一定是桃渚'
- **失陷预测**：'桃渚要失陷'（高置信度判定，使用'一定'和'要'等确定性词汇）
- **评估主体**：戚继光（基于军事经验或情报分析）

### 连锁目标推演逻辑
- **前提条件**：IF '桃渚失陷'
- **推论结果**：THEN '下面倭寇会进犯新城'
- **地理依据**：基于浙江台州海防地理布局，桃渚与新城存在战略递进关系

### 防御响应升级
- **触发条件**：确认连环失陷风险（桃渚→新城）
- **指挥官反应**：面容严峻，'比海面上空那轮冷月还白'（表明情况危急）
- **隐含动作**：需立即调整防御部署，准备新城保卫战

## 总督紧急防汛前线指挥程序

### 险情确认与评估阶段
1. 在总督署大门外（暴雨雷电环境中）接收亲兵队长转述的河道监管报告
2. 确认关键情报：
   - 九个县每个县的堰口闸门都裂了口子
   - 沙包扔下去就冲走了，根本堵不住
   - 表明常规抢险手段完全失效，险情等级为"天地不仁"级别的自然灾害
3. 环境评估：
   - 天已全黑，大雨连幕，雷电交加（闪电将人影照得惨白）
   - 常规指挥体系已无法应对

### 立即决策亲赴一线
- **目的地选择**：淳安县（命令"去淳安！"）
- **交通方式决策**：
  - 拒绝乘轿（亲兵队长喊"把轿抬过来"被胡宗宪吼断"牵马！"）
  - 选择骑马以适应暴雨、泥泞道路和紧急行军需求
- **装备穿戴**：亲兵协助快速披油衣（油布雨衣）
  - 具体动作流程：胡宗宪两臂往下一伸，亲兵将油衣腋口对准双手往上一提，将油衣帽子往头上一罩，转到身前系好胸前系带
  - 亲兵队长扔弃油布大伞（无法打伞骑马）

### 军事抢险力量部署（通过亲兵队长传令）
- **传令配置**：亲兵队长带两个人立刻去大营（共3人传令小组）
- **指定指挥官**：戚总兵（戚继光）、谭参军（谭纶）
- **兵力调动**：领一千兵即刻赶到大堤
- **战术部署**：派兵分驻各个堰口抢险（针对九个县的多点分散部署，解决沙包冲走的紧急状况）
- **后续会合指令**：抢险部署完成后，二位指挥官赶赴淳安见总督（建立前线指挥汇合点，实现指挥体系前移）

### 指挥官现场行动与护卫
- 胡宗宪跨上黑马（亲兵队长搀扶着手臂往上一送送上马）
- 护卫配置：亲兵队全体上马（十几骑）紧随其后
- 行动顺序：胡宗宪两腿一夹，率先向雨幕中驰去，亲兵队紧跟着驰入雨幕（身先士卒的指挥原则）

## 关键变量
- **primary_target**：首要攻击目标（桃渚）
- **secondary_target**：连锁攻击目标（新城）
- **fallen_status**：首要目标是否失陷，是触发连锁预测的关键节点
- **confidence_level**：预测置信度（'一定'、'要'表示极高置信度）
- **affected_counties**：受灾县数量（九个县）
- **deployed_troops**：调动兵力规模（一千兵）
- **commanders**：指定指挥官（戚总兵、谭参军）
- **destination**：前线指挥所目的地（淳安）
- **messengers**：传令兵人数（亲兵队长+两个人）

## 执行结果
- 预警：桃渚失陷后，新城将面临直接威胁，需立即部署防御并准备连环作战
- 军事抢险力量（1000兵）已分驻九个县堰口进行抢险；指挥官（总督）已亲赴淳安一线建立前线指挥部；指定将领（戚总兵、谭参军）完成部署后赶赴淳安汇合
