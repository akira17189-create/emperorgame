# 明代监狱管理规程

## 使用场景
- 需要控制监狱人员准入
- 需要转移囚犯至审讯场所
- 发现非授权人员进入牢房
- 高级官员进入诏狱提审

## 淳安县大牢人员准入管制规则

### 前置条件
- 海瑞作为知县在场
- 发现非授权人员进入

### 执行步骤

1. **规则声明**
   - 明确规定牢房内只允许县衙的差役和牢卒进入
   - 禁止其他人员（如士兵）进入

2. **违规识别**
   - 当发现非授权人员（如四个兵）出现在牢房值房内时
   - 立即识别为违规

3. **质问与警告**
   - 质问违规人员"谁叫你们进来的！"
   - 指出其违规行为

4. **驱逐命令**
   - 直接命令违规人员离开（"出去！"）

5. **威慑升级**
   - 若违规人员无动于衷
   - 利用其上级权威施压
   - 命令下属（田有禄）"把徐千户叫来！"

6. **恢复秩序**
   - 在违规人员撤离后
   - 命令下属（牢头）关闭房门（"把门关上！"）
   - 切断外部干扰，重新掌控局面

### 约束条件
- 仅适用于淳安县大牢管理
- 需海瑞亲自执行或授权

## 北镇抚司诏狱囚犯转移与审讯前流程

### 前置条件
- 官员（黄锦）抵达北镇抚司诏狱小院
- 时间为午时后

### 执行步骤

1. **入场与观察**
   - 午时后，黄锦在两个锦衣卫陪同下进入小院
   - 黄锦观察环境：中间录房锁着，西边和东边屋门均关着

2. **下达指令**
   - 黄锦下令："人都在哪里？唤出来，到录房说话。"

3. **开启录房**
   - 一名锦衣卫打开录房锁
   - 侧立让黄锦进入，随后跟进

4. **召集囚犯**
   - 另一名锦衣卫对两间关着的屋子喊道："收拾了！收拾了！到中间录房来！"

5. **芸娘的应对（东屋/厨房）**
   - 芸娘从东边厨房出来，神态恹恹，发髻蓬乱
   - 锦衣卫对她说："熬到头了，收拾了东西先到录房来吧。"
   - 芸娘拿出布包袱，去收衣服
   - 取靠近高翰文衣衫的内衫时，手停住，怔怔看了一阵
   - 掀开高翰文衣服边幅取下自己的内衫，折好放回包袱

6. **关于高翰文的对话**
   - 锦衣卫问："那位呢？高大人哪。"
   - 芸娘答："应在西边屋里吧。"
   - 锦衣卫问："你们还一东一西，不住在一起？"
   - 芸娘表态："要带我去哪里，我这就跟你们走。我的事不干他的事，他的事也不干我的事。"
   - 锦衣卫追问："是你怕他牵累你，还是不愿自己牵累他？"芸娘沉默

7. **高翰文的应对（西屋）**
   - 录房内黄锦催促："怎么回事，还不带进来？"
   - 芸娘进入录房
   - 锦衣卫走到西屋，高翰文站在门内
   - 锦衣卫拱手："恭喜了。收拾了东西，我们送高大人出去了。"
   - 高翰文问："去哪里？"
   - 锦衣卫答："先去录房吧，到了录房就知道了。"

### 关键变量
- **黄锦**：司礼监掌印太监，负责此次提审或转移
- **芸娘**：囚犯，原杨金水身边人，现关押于东屋（厨房）
- **高翰文**：囚犯，原淳安知县，现关押于西屋
- **录房**：中间录房，用于审讯或谈话的场所
- **授权人员**：县衙的差役和牢卒
- **非授权人员**：如织造局派来的士兵等
