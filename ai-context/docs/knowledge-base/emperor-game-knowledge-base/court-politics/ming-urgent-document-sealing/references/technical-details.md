# 技术细节说明

## 烤漆材料与操作
- **材料**：漆棒凝固于铜签上
- **操作**：火烤熔化后使用
- **状态要求**：可流动但不过热，避免烧焦

## 羽毛标识系统
- **一根羽毛**：普通急递
- **两根羽毛**：紧急急递
- **三根羽毛**：最高级别急递

## 印章使用规则
- **主审官员优先**：重审案件使用重审官印章
- **协审官员**：一般不使用其印章，除非有特殊规定
- **权限原则**：原案用原案封印，重审用重审官印章

## 案例中的具体操作
- **文件清单**：内阁司礼监发回的原供、郑泌昌何茂才翻供的供词、蒋千户徐千户的供词、田有禄王牢头签名的字据
- **操作人员**：海瑞（主审）、王用汲（协审）
- **印章使用**：仅用海瑞印章
- **递送要求**：一个时辰内送达赵中丞
- **同步押解**：郑泌昌、何茂才押解至巡抚衙门
