# 官吏流品分化及其政治后果

## 适用场景
- 分析元明清政权中“官”与“吏”的结构性对立
- 解释胥吏舞弊、师爷专权等现象的制度成因
- 探讨“法愈密而弊愈深”的文书政治逻辑
- 对比汉唐官吏一体与元明后流品分化的差异

## 执行步骤

1. **确认触发条件**：判断是否存在以下任一情形：
   - 政权由不熟悉汉文的外族主导（如元代）
   - 制度明确限制胥吏晋升通道（如明代禁止吏胥考进士）

2. **追溯起源**：指出元代蒙古统治者依赖汉人书记处理政务，迫使读书人充任胥吏，开启“官”“吏”流品之分。

3. **识别制度固化点**：强调明成祖确立“吏胥不得为御史”“不准考进士”等禁令，彻底断绝胥吏上升路径，形成“清流”与“浊流”之别。

4. **分析社会后果**：
   - 胥吏因无望晋升而自暴自弃，普遍舞弊
   - 形成掌握铨选、处分、财赋、典礼、人命、讼狱、工程七项核心政务的专业集团
   - 出现“铨选可狡可迟，处分可轻可重，工程可增可减”等操作空间

5. **揭示权力倒挂机制**：
   - 科举出身官员缺乏实务能力，依赖师爷与胥吏
   - 胥吏掌控文书簿籍与例案掌故，可操纵长官决策
   - 新官到任需数月甚至一年半载方能熟悉衙门运作，期间易被掣肘

6. **引用典型案例**：提及“绍兴师爷”势力遍及全国，“天下治乱在六部，而六部胥吏尽绍兴人”，陈几亭称“绍兴乃天下治乱之根本”。

7. **归因于制度而非道德**：强调此非个人品德问题，而是“法病”——制度性歧视导致人才沉沦，催生“尚文之弊”与“文书政治”。

> **输出提示**：本技能最终应警示：制度性歧视如何系统性催生腐败温床，导致“法愈密而弊愈深”。
