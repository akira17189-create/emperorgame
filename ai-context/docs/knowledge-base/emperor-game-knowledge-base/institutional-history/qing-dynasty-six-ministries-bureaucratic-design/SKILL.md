# 清代六部与官员任用中的法术化设计

## 适用对象
- 清代六部（吏、户、礼、兵、刑、工）
- 尚书、侍郎等中央高级官员
- 地方督抚及以下官员

## 使用时机
当需要解释以下问题时启用本技能：
- 为何清代中央行政效率低下？
- 清代六部为何无法有效统合部门职能？
- 满汉双设如何导致官僚内耗？
- 皇帝如何通过人事制度强化个人控制？

## 执行步骤

### 1. 识别六部权力的系统性削弱
- 对比明代：明代六部尚书可直接下令督抚，清代则完全丧失此权。
- 确认清代六部不再作为统一机关运作，各长官可独立行动，部门整体性被破坏。

### 2. 分析满汉双设结构及其内耗效应
- 每部设满汉尚书各一、满汉侍郎各二，共六位长官。
- 各长官均可单独向皇帝上奏，互不知情，形成相互掣肘。
- 正职无法管辖副职，满汉官员之间信息隔绝，削弱协同能力。

### 3. 考察官员任用的“法术化”特征
- 高级官员由皇帝特简，绕过吏部与廷推程序。
- 低级官员虽经吏部铨叙，但必须经皇帝引见后方可上任。
- 引见制度表面宣示“用人之权在皇帝”，实质为满洲部族控制手段。

### 4. 识别配套言论限制措施
- 废除给事中封驳权，使其台谏合一，丧失监督功能。
- 禁止翰林院编修、检讨“专折言事”。
- 地方仅总督、巡抚、藩臬二司可直接上奏，府县以下无言事权。

### 5. 提炼核心逻辑
- 所有制度安排均服务于皇帝（即满洲部族）集权。
- 故意破坏行政机构的整体性，强化官员对皇帝的个人依附。
- 此非理性制度设计，而是维护部族统治的“法术”手段。

## 输出要求
最终应明确指出：清代行政体系的低效源于其非制度化的“法术化”设计，本质是满洲部族为维持统治而刻意削弱官僚系统自主性的结果。
