# 明代黄册制度操作指南

## 适用场景
- 明代户籍管理中需更新户级信息
- 赋役征派前需核实实际田亩数量
- 土地交易（买入、卖出、继承）后需在黄册中登记

## 前提条件
- 存在明确的“户”单位
- 发生田亩买卖或变动
- 地方政府具备造册能力

## 操作流程
1. **确认登记单位**：以“户”为基本单位，每十年重新编订一次黄册。共制作四本，分别上交户部、布政司、府、县。
2. **应用四柱式记录法**：
   - **旧管**：原登记田亩数量
   - **新收**：本期新增田亩（如购入）
   - **开除**：本期减少田亩（如售出）
   - **实在**：当前实际拥有田亩（计算公式：实在 = 旧管 + 新收 - 开除）
3. **处理跨都交易**：若买田者属于不同“都”，在其所属户图中设立“子户”，并逐项注明交易细节。
4. **赋役组织结构**：每110户为一“里”，推选丁粮最多之十户为“里长”，其余百户分为十甲；每年轮派里长一人、甲首一人，负责催办赋役。
5. **输出结果**：计算并输出该户当期“实在”田亩数，作为赋役征派的核心依据。

## 注意事项
- 不适用于无固定户籍人口
- 在土地兼并严重地区易失效
- 清代后期已废止，仅适用于明代相关情境
- 警惕大户通过篡改“新收”为“旧管”规避监管
