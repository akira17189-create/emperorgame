# 汉代内朝外朝冲突分析

## 何时使用
- 用户提及汉代皇权扩张、相权削弱现象
- 讨论大司马大将军、尚书台等内廷职位干预政务
- 分析霍光废立昌邑王等“皇帝家事”争议
- 探究汉代中央政府结构失衡或王莽篡汉的制度根源

## 执行步骤
1. **确认制度原意**：汉代初期明确区分皇室与政府，宰相为“副皇帝”，代表皇室统理全国政务。
2. **识别皇权越界行为**：判断是否存在皇帝绕过外朝宰相，直接通过内廷官员（如大司马、尚书）处理国家事务的情形。
3. **定位关键转折点**：重点考察汉武帝设立大司马大将军辅政、尚书台脱离御史系统等制度变更。
4. **分析冲突案例**：以霍光废昌邑王为例，指出其错误将皇位继承视为“皇帝家事”，忽视高后四年诏书及昭帝崩后“诏群臣议所立”的先例——证明继统属政府公议事项。
5. **评估制度后果**：说明内朝权重如何导致皇室凌驾政府之上，破坏权力制衡。
6. **提出修正方向**：建议重建制度性沟通机制（如汉宣帝恢复御史中丞统领尚书），防止皇室权力无约束扩张。

> 注意：本技能不适用于皇权虚位或现代宪政体制下的分析场景。
