# 唐代尚书省六部制度及其执行机制

## 何时使用
- 用户询问唐代政令如何具体执行
- 需确定某类政务（如人事、户籍、军事等）归属哪个部门管辖
- 分析五品以上与以下官员任用权限差异
- 探讨六部制度的历史沿革及其对后世影响

> **前提条件**：政事堂会议已作出政令决定，进入执行阶段。

## 执行步骤

### 1. 尚书省定位
- 作为唐代最大、最高的行政执行机关，仅负责执行，无决策权
- 接收经中书起草、门下审查、政事堂盖印的正式诏敕

### 2. 六部分工与职能
按标准排序（宋神宗后定型并沿袭）：
- **吏部**：主管官员任用；五品以下官全权任免，五品以上可提名但由宰相决定
- **户部**：掌管户口、赋税、民政事务
- **礼部**：主管礼仪、科举、宗教、教育
- **兵部**：掌管武官选授、军籍、军令（不直接统兵）
- **刑部**：主管司法审判、律令执行
- **工部**：主管土木工程、水利、工匠管理

### 3. 内部组织
- 每部下设四司，共二十四司
- 第一司以部名为名（如吏部司）
- 办公制度：上午在都堂集体办公便于协调，下午回本部分别处理事务

### 4. 历史沿革要点
- 唐初六部排序为：吏、礼、兵、民（户）、刑
- 唐太宗时增工部，排序调整
- 宋初顺序变动较大
- **宋神宗王安石变法后定型为：吏、户、礼、兵、刑、工**，成为后世定制

## 输出结果
政令通过六部系统依职能分派执行，确保国家政务专业化、制度化落实。该体制自唐延续至清末，构成中国帝制后期行政主干。
