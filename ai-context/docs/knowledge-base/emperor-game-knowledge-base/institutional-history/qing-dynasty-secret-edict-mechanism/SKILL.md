# 清代寄信上谕秘密政治机制分析

## 何时使用
- 用户探讨清代中央集权或皇权专制特征
- 需解释为何清代政令可绕过内阁与六部
- 比较唐宋明与清代诏令制度差异
- 分析“秘密政治”对传统公开政务原则的破坏

## 核心操作流程

### 1. 确认适用背景
- 仅适用于雍正朝以后的清代中央政府
- 不适用于设有宰相审核机制的唐宋等朝代
- 前提：军机处已设立，宰相制度已废除

### 2. 区分上谕类型
- **明发上谕**：用于常规事务（如官员例行升降、巡幸），经内阁拟稿、六部执行
- **寄信上谕**：用于紧要政务，由军机处直接拟稿，皇帝审阅后密封发出，六部无权知晓

### 3. 执行寄信上谕流程
1. 军机大臣草拟上谕内容（基于皇帝口谕或密旨）
2. 皇帝亲自审阅并批准
3. 加盖“办理军机处”专用印（象征皇帝亲办）
4. 交兵部尚书加封，直接寄送指定受命人（如巡抚、尚书）
5. 全程保密，其他官员不得拆阅或干预

### 4. 揭示政治后果
- 全国高级官员直接向皇帝个人负责，正式政府机构形同虚设
- 政务完全秘密化，违背中国传统“公开议政”原则
- 此机制本质为满洲部族私权工具，属“法术”而非制度

## 关键变量
- **寄信上谕**：清代特有的秘密命令文书类型
- **办理军机处印**：使上谕生效且具保密效力的专用官印
