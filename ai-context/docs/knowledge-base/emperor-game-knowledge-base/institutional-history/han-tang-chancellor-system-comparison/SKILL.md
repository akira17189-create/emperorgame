# 汉唐相权制度对比：领袖制 vs 委员制

## 适用场景
- 分析中国古代相权制度的演变趋势
- 比较汉代与唐代中央行政体制差异
- 探讨皇权与相权的互动关系
- 评估传统政府组织的稳定性与制衡机制

## 执行步骤

### 1. 识别相权模式
- 若为**汉代**：采用**领袖制**，由丞相总揽全国行政大权，御史大夫为副职分管监察。
- 若为**唐代**：采用**委员制**，相权分割于中书省（发令）、门下省（审核）、尚书省（执行）。

### 2. 分析决策机制
- **汉代**：重大政事通过皇帝、宰相及廷臣临时会议（廷议）决定，无常设集体决策机构。
- **唐代**：通过**政事堂常设会议**进行集体决策，实行“执笔”轮值主席制综合三省意见。

### 3. 评估相权完整性
- 判断是否包含**决策、审核、执行**全链条：
  - 汉代：完整集中于一人，但易被皇权或内朝架空。
  - 唐代：三省分工协作，合三省职权方等于汉代一相，制度化程度高。

### 4. 解读历史演变逻辑
- 春秋时期宰相为家臣 → 秦汉时期转化为国家执政官 → 东汉魏晋皇室私臣（如尚书）代行相权 → 唐代将私臣转为正式外朝官，重建政府权威。

### 5. 输出制度评价
- 唐代委员制更稳定，能有效防止个人专权，恢复被皇室侵夺的相权。
- 但仍保留变通空间（如皇帝可发“斜封墨敕”），体现中国传统法制“留有余地”的特点。

## 注意事项
- 仅适用于**汉代与唐代**中央政府层级，不适用于秦以前或宋以后制度。
- 不适用于地方行政或非相权相关议题。
