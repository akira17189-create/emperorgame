# 汉代田租与地主兼并机制

## 何时使用
- 用户研究汉代经济结构、赋税政策效果或土地制度演变
- 需解释“轻税为何未惠及农民”或“地主如何从低税率中获益”
- 分析董仲舒限田建议或王莽井田制改革失败的制度背景

## 执行步骤
1. 确认适用条件：仅适用于汉代（非井田制时代）、耕地且实行土地私有与自由买卖。
2. 核实田租政策：法定为十五税一（≈6.67%），实际执行多为三十税一（≈3.33%），文帝时期甚至长期免税。
3. 追踪土地流转：农民因经济困难出售土地，失去土地后成为佃农。
4. 对比租率差异：佃农向地主缴纳高达50%的地租，而地主仅按三十税一向政府纳税。
5. 推导机制结论：政府轻税政策的实际受益者是地主，而非农民，导致贫富差距扩大。
6. 关联制度根源：此现象源于井田制崩溃后，政府仅按田亩征税，不干预土地所有权分配。
7. 补充历史尝试：指出董仲舒“限田”与王莽“皇田”改革均未成功，此后无彻底土地改革。

> **注意**：本机制不适用于山林、池泽等非耕地，亦不适用于土地国有或禁止买卖的时期。
