# 唐代租庸调制与账籍管理制度

## 适用条件
仅在以下条件同时满足时适用：
- 国家有效推行均田制
- 户籍登记完整清晰
- 土地兼并不严重、社会秩序稳定

## 核心执行流程

### 1. 确认租庸调三要素
- **租**：均田农民按“四十税一”缴纳粮食
- **庸**：每年服劳役20天（可纳绢代役）
- **调**：缴纳本地土产，主要为丝麻织物

### 2. 验证制度前提
确保“为民制产”实现：每丁有田（可纳租）、有身（可服役）、有家（可纳调）。

### 3. 检查账籍管理制度
- **户籍（“籍”）**：记录人口生卒，按资产分九等，每三年重造一次
- **壮丁册（“账”）**：预定次年徭役人数，每年重造一次
- **保存要求**：州县保存15年（五比），户部保存9年（三比）

### 4. 识别制度失效信号
若出现以下情况，表明制度已难维持：
- 户籍未及时更新（如老人未除名、新丁未录入）
- 豪强阻扰均田实施
- 全国统计因交通、纸张、书写等技术限制难以精确

### 5. 关联历史定位
- 继承北魏均田制，终结井田—均田传统
- 体现计划经济思想，失败后被两税法取代
- 与孟子“粟米之征、力役之征、布帛之征”思想呼应
