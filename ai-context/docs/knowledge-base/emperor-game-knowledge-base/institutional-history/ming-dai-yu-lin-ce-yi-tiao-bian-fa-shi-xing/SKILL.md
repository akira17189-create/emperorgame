# 明代鱼鳞册与一条鞭法施行

## 适用场景
- 政府需简化赋役征收手续
- 存在土地买卖、产权变动或差役负担争议
- 分析明代中后期（宣德至万历）财政制度演变

## 执行步骤
1. **确认鱼鳞册基础**：核实是否存在以“土田”为单位、按县—乡—都层级编排的鱼鳞图册，图中应标注每块田地的属性（官/民、高/污、埂/瘠、山/荡）及业主姓名。
2. **执行年度注记**：若发生土地交易，须按“一年一注”原则更新册籍，确保“人户流动，田地不移”，维持“以田为母，以人为子”的确权逻辑。
3. **整合差役入田赋**：将里甲、均徭、杂泛等各类差役及附加杂项合并，统一按田亩数量折算为银两征收（即“计亩征银”）。
4. **评估区域可行性**：优先在长江下游等南方经济发达、银两流通成熟的地区推行；北方因经济结构差异大、银货不足，需谨慎评估阻力。
5. **识别舞弊风险**：检查是否存在“飞洒”（分散田产避税）或“诡寄”（假托他人名下）行为，防止业主花名混淆导致册籍失真。
6. **判断实效期限**：注意嘉靖以后鱼鳞册多已形同虚设，一条鞭法亦难以持久，需结合具体年代判断制度实际效力。

## 输出结果
生成一份基于鱼鳞册土地数据的统一按亩征银赋税方案，并附注其历史可行性与潜在局限。
