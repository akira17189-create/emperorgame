## 三、GitHub README 中文版（800字以内）

# 靖朝风云 - 架空历史AI策略游戏

![游戏截图](https://img.shields.io/badge/状态-开发中-yellow)
![TypeScript](https://img.shields.io/badge/语言-TypeScript-blue)
![Vite](https://img.shields.io/badge/构建-Vite-purple)
![AI叙事](https://img.shields.io/badge/特色-AI叙事-green)

**《靖朝风云》** 是一款基于AI叙事的架空历史策略游戏。玩家扮演靖朝皇帝，通过自然语言指令治理国家，在虚构的东方王朝中书写属于自己的历史。

## 🎮 游戏特色

### 🤖 **AI驱动叙事**
- 所有事件、对话、史书均由AI实时生成
- 文风参考《明朝那些事儿》，幽默而不失深度
- 每次游戏都是独一无二的历史剧本

### 📜 **自由旨意系统**
- 用自然语言下达任何命令：`减税惠民`、`兴建水利`、`整顿吏治`
- AI理解指令意图，转化为游戏内行动
- 支持复杂指令组合，体验真实帝王决策

### ⚖️ **多派系政治博弈**
- **清流派**：科举文官，重名节，轻实务
- **帝党派**：皇帝亲信，唯上是从，务实高效
- **宦官派**：内廷势力，权力寄生，利益至上
- 平衡三大势力，体验真实朝堂斗争

### 🏛️ **完整治理系统**
- **8大资源**：财政、粮食、军力、士气、商业、外患、宦官、党争
- **22种政策**：从传统"减税惠民"到穿越者专属"蒸汽奇器"
- **动态事件**：天灾、边境冲突、宫廷斗争、民间起义

### 🏁 **多结局系统**
- **8种王朝结局**：盛世中兴、仁寿归天、亡于内乱、山河破碎、阉祸亡朝、蒸汽纪元、丹鼎误国、守成之君
- 每个决定影响结局走向
- 史官评语为统治生涯盖棺定论

## 🚀 快速开始

### 在线体验
访问 [在线演示地址] 立即开始游戏

### 本地开发
```bash
# 克隆项目
git clone https://github.com/your-username/jing-dynasty-game.git

# 安装依赖
cd jing-dynasty-game
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

### 环境要求
- Node.js 18+
- 现代浏览器（Chrome 90+, Firefox 88+, Safari 14+）

## 🛠️ 技术栈

- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite
- **状态管理**：Zustand
- **AI集成**：OpenAI API / 本地LLM
- **样式方案**：Tailwind CSS
- **代码质量**：ESLint + Prettier

## 📁 项目结构

```
src/
├── engine/          # 游戏引擎核心
│   ├── policy-engine.ts    # 政策系统
│   ├── event-engine.ts     # 事件系统
│   ├── narrator.ts         # 叙事引擎
│   └── tick.ts             # Tick系统
├── ui/              # 用户界面
│   ├── CourtPage.tsx       # 主朝堂界面
│   ├── PolicyPanel.tsx     # 政策面板
│   └── ChroniclePage.tsx   # 史册页面
└── types/           # TypeScript类型定义
```
