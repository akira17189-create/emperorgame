# 皇帝游戏：world.factions 派系势力设计文档

## 设计概述

基于对靖朝架空历史背景（内阁制、宦官、党争、边患等元素）的分析，以及现有游戏代码（`types.ts`中的初始值定义）的兼容性考虑，本设计文档为 `world.factions` 派系系统提供完整的设计方案。系统包含5个派系：清流派、帝党、宦官党、军队派、务实派，每个派系值域为0~100，影响政策制定、NPC反应和游戏结局。

## 一、各派系初始值设计

### 当前初始值分析（`types.ts`中定义）：
```typescript
factions: {
  qingliu: 60,    // 清流派势力
  didang: 50,     // 帝党势力
  eunuch_faction: 30,  // 宦官党势力
  military: 50,      // 军队派势力
  pragmatists: 40    // 务实派势力
}
```

### 初始值合理性评估与调整建议：

1. **清流派 (qingliu: 60) - 基本合理**
   - 合理性：靖朝为内阁制，清流派作为传统文官集团，在朝堂中势力较强
   - 建议维持60，反映其作为主流派系的地位
   - 对应关系：与 `resources.faction`（党争值）正相关

2. **帝党 (didang: 50) - 需要调整**
   - 问题：新帝登基（年龄18），帝党势力应相对薄弱
   - 建议调整为：**35-40**
   - 理由：新帝尚未建立稳固的忠诚班底，需通过政策逐步培养帝党势力

3. **宦官党 (eunuch_faction: 30) - 合理**
   - 合理性：游戏开始时宦官势力不应过强，但与 `resources.eunuch`（宦官值30）联动
   - 建议维持30，保持与宦官资源值的同步

4. **军队派 (military: 50) - 偏高**
   - 问题：靖朝初期边患不重（外患20），军队派势力应适中
   - 建议调整为：**45**
   - 理由：反映军队在王朝中的常态地位，既不过强也不过弱

5. **务实派 (pragmatists: 40) - 偏低**
   - 问题：务实派代表改革和技术官僚，在架空历史中应有发展潜力
   - 建议调整为：**45**
   - 理由：为后续工业类政策预留增长空间

### 调整后初始值推荐：
```typescript
factions: {
  qingliu: 60,      // 清流派 - 维持
  didang: 38,       // 帝党 - 从50下调（新帝基础薄弱）
  eunuch_faction: 30, // 宦官党 - 维持
  military: 45,     // 军队派 - 从50微调
  pragmatists: 45   // 务实派 - 从40上调（预留增长空间）
}
```

### 2.2 事件对派系的影响规则

#### 常见事件影响：
1. **边关告急事件**
   - 军队派：+3~+5（危机时刻军队重要性上升）
   - 务实派：+1~+2（需要务实解决方案）
   - 清流派：-1~-2（文官应对不力）

2. **科举舞弊事件**
   - 清流派：-3~-5（声誉受损）
   - 帝党：-1~-2（监管不力）
   - 处理成功：清流派+2，帝党+1

3. **宦官干政事件**
   - 宦官党：+2~+4（权力扩张）
   - 清流派：-2~-3（反对宦官）
   - 帝党：-1~-2（皇权被侵蚀）

4. **天灾救济事件**
   - 务实派：+2~+3（有效赈灾）
   - 帝党：+1~+2（皇恩浩荡）
   - 处理失败：所有派系-1~-2

### 2.3 派系自动变化规则（每季度）

1. **帝党自然增长**：+0.5（每季度），上限受皇帝威望影响
2. **宦官党渗透**：当 `resources.eunuch > 50` 时，宦官党+1（每季度）
3. **清流派传承**：当 `resources.f 30` 时，清流派+1（每季度）
4. **军队派衰减**：和平时期（外30），军队派-0.3（每季度）
5. **务实派机遇**：当生效工业政策≥2时，务实派+1（每季度）

## 三、与 resources 字段的联动规则

### 3.1 `eunuch_faction` 与 `resources.eunuch` 联动
```
联动公式：eunuch_faction = resources.eunuch × 0.8 + 基础值10
```
- **单向影响**：`resources.eunuch` 变化时，`eunuch_faction` 自动调整
- **反向影响**：`eunuch_faction > 60` 时，每季度 `resources.eunuch` +1
- **阈值效果**：
  - 当 `eunuch_faction ≥ 70`：触发宦官专权事件
  - 当 `resources.eunuch ≥ 80` 且 `eunuch_faction ≥ 60`：触发 E5 结局条件

### 3.2 派系与 `resources.faction`（党争值）联动
```
党争值 = (|清流派 - 宦官党| ÷ 2) + (|清流派 - 务实派| ÷ 4) + 基础值20
```
- **派系差距越大，党争越激烈**
- **清流派 vs 宦官党**：主要矛盾，权重50%
- **清流派 vs 务实派**：次要矛盾，权重25%
- **其他派系差距**：补充影响

### 3.3 其他派系与资源字段联动

1. **军队派与军力资源**
   - 当 `military ≥ 70`：每季度 `resources.military` +50
   - 当 `resources.m 1000`：军队派-2（每季度）

2. **务实派与商业资源**
   - 当 `pragmatists ≥ 60`：商业增长效率+20%
   - 当 `resources.commerce > 800`：务实派+3

3. **清流派与民心资源**
   - 当 `qingliu ≥ 65`：`resources.morale` 衰减减缓 30%
   - 当 `resources.morale40`：清流派-3

## 四、派系间相互影响

### 4.1 清流派与宦官党关系：**此消彼长**
- 清流派+1 → 宦官党-0.5
- 宦官党+1 → 清流派-0.8（宦官扩张性更强）
- 当两者差距>40时，触发"朝堂对峙"事件

### 4.2 帝党与其他派系关系
1. **帝党 vs 清流派**：合作竞争
   - 帝党+1 → 清流派+0.3（需要文官支持）
   - 清流派>70 → 帝党-0.2（文官集团尾大不掉）

2. **帝党 vs 宦官党**：控制与反控制
   - 帝30 且 宦官党>50 → 触发"宦官干政"
   - 帝党>60 → 限制宦官党增长上限

3. **帝党 vs 军队派**：相互依存
   - 外患>50时，帝党+1 → 军队派+0.5
   - 军队派>70 → 帝党需维持>40以防军变

### 4.3 军队派与务实派关系：**有条件合作**
- 工业政策下：军队派+1 → 务实派+0.4
- 和平时期：两者关系中立
- 危机时期：合作加强（外患>60时，协同+2）

## 五、派系势力对游戏的影响

### 5.1 高/低派系势力触发事件

#### 高势力触发事件（≥75）：
1. **清流派≥75**："文官集团请愿"事件
   - 要求罢免宦官相关官员
   - 拒绝：清流派-5，党争+10
   - 同意：宦官党-8，帝党-3

2. **宦官党≥75**："司礼监干政"事件
   - 宦官要求批红权
   - 拒绝：触发宦官叛乱风险
   - 同意：皇权-10，帝党-8

3. **军队派≥75**："边将邀功"事件
   - 要求封赏，增加军费
   - 拒绝：军队派-8，军力-200
   - 同意：国库-300，外患-5

4. **务实派≥75**："技术官僚改革"事件
   - 提出全面改革方案
   - 实施：商业+100，清流派-10
   - 搁置：务实派-10，发展停滞

#### 低势力触发事件（≤25）：
1. **帝党≤25**："皇权旁落"事件
   - 触发权臣或宦官摄政
   - 需紧急提升帝党势力

2. **务实派≤25**："技术断层"事件
   - 工业发展停滞
   - 商业增长-30%

### 5.2 派系势力对结局触发的影响

#### 对现有结局的派系条件补充：

1. **E3 亡于内乱**（党争≥80）
   - **派系条件**：`|qingliu - eunuch_faction| ≥ 50` 时，党争+20
   - **提前预警**：当清流派≥70且宦官党≥60，每季度党争+5

2. **E5 阉祸滔天**（宦官≥80且党争≥60）
   - **派系条件**：`eunuch_faction ≥ 70` 时触发条件概率+50%
   - **帝党制约**：当 `didang ≥ 50` 时，触发概率-30%

3. **E6 工业革命开创者**（商业≥1000且火器营生效）
   - **派系条件**：`pragmatists ≥ 60` 时，商业增长+30%
   - **军队支持**：`military ≥ 55` 时，火器营效果+20%

4. **E7 仙道误国**（仙道政策≥4条）
   - **派系条件**：`qingliu ≤ 40` 时，更容易沉迷仙道
   - **务实抵制**：`pragmatists ≥ 50` 时，仙道政策效果-25%

## 七、示例场景说明

### 场景1：推行"火器营制"政策
**初始状态**：
- 军队派：45，务实派：45，帝党：38
- 资源：军力2000，商业300

**政策执行后**：
- 军队派：+5 → 50
- 务实派：+4 → 49  
- 帝党：+3 → 41
- 清流派：-2 → 58
- 资源：军力+70 → 2070，国库-200

**连锁反应**：
1. 军队派>48，触发"军工复合体"效果
2. 务实派增长，为后续工业政策铺路
3. 帝党提升，强化皇权基础

### 场景2：宦官党扩张至危机
**发展过程**：
1. 执行"召仙炼丹"：宦官党+5 → 35
2. 执行"丹药普赐"：宦官党+3 → 38
3. `resources.eunuch`增长至45 → 宦官党同步至46
4. 宦官党>45，触发"宦官干政"事件风险

**危机应对**：
- 选项A：打压宦官（帝党+3，宦官党-8）
- 选项B：妥协（宦官党+3，帝党-5，触发清流派抗议）

### 场景3：务实派推动工业革命
**积累阶段**：
1. 推广活字印刷：务实派+3 → 48
2. 开设钱庄：务实派+3 → 51
3. 开海通商：务实派+4 → 55
4. 务实派>50，商业增长+15%

**突破阶段**：
1. 蒸汽奇器：务实派+5 → 60
2. 火器营制：务实派+4 → 64
3. 务实派≥60，触发"技术官僚集团"事件
4. 商业加速增长，向E6结局迈进

## 八、实现建议

### 代码实现位置：
1. **派系更新逻辑**：在 `state-updater.ts` 的季度更新函数中
2. **政策派系影响**：在 `policy-engine.ts` 的政策效果定义中扩展
3. **派系事件触发**：在 `event-engine.ts` 中添加派系阈值检查
4. **联动公式**：在 `tick.ts` 的资源更新周期中实现

### 兼容性保证：
1. **保持现有`createEmptyGameState`函数结构**：只调整初始值
2. **扩展而非修改**：新增派系影响字段，不破坏现有逻辑
3. **渐进式实现**：先实现核心联动，再逐步添加高级特性

### 测试建议：
1. **单元测试**：验证各政策对派系的影响值
2. **集成测试**：模拟完整游戏周期，检查派系平衡性
3. **边界测试**：验证派系0-100边界行为
