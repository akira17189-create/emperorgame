# 皇帝游戏：发布演示剧本 - 工业革命之路（E6结局）

## 演示目标
展示如何在5-10分钟内快速达成E6结局"蒸汽纪元"，同时演示游戏核心系统：朝堂对话、政策颁布、资源管理、事件处理和结局触发。

## 预计时长
8-10分钟

## 前置条件
- 新游戏开始
- 游戏难度：简单模式（资源初始值较高）
- 初始状态：
  - 国库：3000
  - 商业：200  
  - 军力：1500
  - 民心：65
  - 外患：20

## 演示步骤

### 第1步：游戏初始化与界面熟悉
- **操作**：启动新游戏，进入主界面，熟悉各功能区：
  1. 左上角：年号显示（靖朝元朔元年）
  2. 中间：朝堂场景（皇帝、国师、大臣站位）
  3. 右侧：资源面板（国库、民心、军力等）
  4. 下方：行动按钮（政策、对话、查看史册）
- **预期**：界面加载正常，初始资源显示正确
- **验证**：检查资源数值与预设一致

### 第2步：与国师首次对话
- **操作**：点击国师头像，选择对话选项：
  1. "国师，当前朝政如何？"
  2. "朕该如何开始治国？"
  3. "听闻民间有新技术，国师可知？"
- **预期**：
  - 国师介绍当前朝堂局势
  - 解锁"穿越者知识"系统提示
  - 获得建议：优先发展商业和军备
- **验证**：对话记录出现在日志中，获得第一条穿越者知识

### 第3步：颁布第一个工业政策 - 推广活字印刷
- **操作**：
  1. 点击"政策"按钮
  2. 选择"工业"标签页
  3. 找到"推广活字印刷"政策
  4. 确认颁布（消耗：国库-100，时间：永久生效）
- **预期**：
  - 资源变化：商业+30，民心+1
  - 派系变化：务实派+3
  - 解锁成就："技术革新第一步"
- **验证**：资源面板显示商业增至230，政策生效状态显示为绿色

### 第4步：处理随机事件 - 科举舞弊案
- **操作**：
  1. 触发随机事件弹窗"科举舞弊风波"
  2. 选择处理方案：
    - 选项A：严查到底（清流派+3，民心+2，国库-50）
    - 选项B：从轻发落（宦官党+2，民心-1）  
    - **选择A：严查到底**
- **预期**：
  - 获得第二条穿越者知识："严格监管体系的重要性"
  - 清流派势力上升
  - 民心小幅提升
- **验证**：事件日志记录处理结果，集体记忆增加一条

### 第5步：颁布军事政策 - 火器营制
- **操作**：
  1. 再次进入政策界面
  2. 选择"军事"标签页（或工业标签下的火器相关）
  3. 找到"火器营制"政策
  4. 确认颁布（消耗：国库-200，时间：永久生效）
- **预期**：
  - 资源变化：军力+70，外患-6，民心+1
  - 派系变化：军队派+5，务实派+4
  - 解锁E6结局关键条件之一
- **验证**：军力增至1570，外患降至14，政策生效

### 第6步：与兵部尚书对话了解军情
- **操作**：
  1. 点击兵部尚书（或相关武将）头像
  2. 选择对话：
    - "当前边关形势如何？"
    - "火器营建设进展？"
    - "军费是否充足？"
- **预期**：
  - 了解边关安全状况
  - 获得火器营建设进度报告
  - 触发第三条穿越者知识："军事现代化的重要性"
- **验证**：集体记忆中科技相关记忆达到3条，满足E6条件之一

### 第7步：颁布经济政策 - 开海通商
- **操作**：
  1. 进入政策界面
  2. 选择"工业"或"经济"标签页
  3. 找到"开海通商"政策
  4. 确认颁布（消耗：无，时间：永久生效）
- **预期**：
  - 资源变化：商业+90，国库+100，外患+3，民心-1
  - 派系变化：务实派+4，清流派-3
  - 商业值大幅提升，接近E6触发条件
- **验证**：商业增至320+90=410，继续增长趋势

### 第8步：使用时间加速与查看史册
- **操作**：
  1. 点击"下一季度"按钮4次（加速4个季度，约1年游戏时间）
  2. 期间处理可能出现的随机事件（选择默认或最优选项）
  3. 点击"史册"按钮查看游戏记录
- **预期**：
  - 时间推进至靖朝元朔二年
  - 商业自然增长至约600-700
  - 史册中记录所有政策、事件和对话
- **验证**：时间显示正确，资源持续增长，史册功能正常

### 第9步：颁布关键政策 - 蒸汽奇器
- **操作**：
  1. 商业值达到800+时（通过自然增长和前序政策）
  2. 进入政策界面
  3. 颁布"蒸汽奇器"政策（消耗：国库-300，时间：持续5年）
- **预期**：
  - 资源变化：商业+120，民心+2，军力+20
  - 派系变化：务实派+5，清流派-4
  - 商业突破1000大关，满足E6核心条件
- **验证**：商业值达到1000+，所有E6触发条件就绪

### 第10步：触发E6结局 - 蒸汽纪元
- **操作**：
  1. 继续点击"下一季度"1-2次
  2. 等待系统自动检测结局条件
  3. 结局触发弹窗出现
- **预期**：
  - 弹出E6结局画面："蒸汽纪元"
  - 显示完整的叙事文本（含新增补丁内容）
  - 播放结局音乐和特效
  - 提供"查看详情"和"重新开始"选项
- **验证**：
  1. 触发条件验证：
    - 商业≥1000 ✓
    - 火器营政策生效 ✓  
    - 穿越者知识≥3条 ✓
  2. 结局叙事显示完整且包含精准触发条件提及
  3. 演示完成，返回主菜单

## 演示要点

### 核心系统展示：
1. **朝堂对话系统**：与国师、大臣的互动，获取建议和知识
2. **政策颁布系统**：工业、军事、经济三类政策的选择和执行
3. **资源管理系统**：国库、商业、军力、民心等资源的动态变化
4. **事件处理系统**：随机事件的触发和决策影响
5. **结局触发系统**：多条件复合触发机制

### 演示节奏控制：

**前3分钟**：系统熟悉和基础政策（步骤1-4）
- 快速建立游戏认知
- 完成第一个政策颁布
- 处理首个随机事件

**中3分钟**：核心政策推进（步骤5-7）
- 颁布关键军事政策
- 完成穿越者知识收集
- 大幅提升商业值

**后3分钟**：结局冲刺（步骤8-10）
- 时间加速观察效果
- 颁布最终政策
- 触发结局展示

### 应急预案：

1. **商业增长不足**：
   - 备选方案：额外颁布"开设钱庄"政策（商业+60）
   - 调整：在第8步多加速1-2个季度

2. **随机事件干扰**：
   - 如果出现负面事件影响商业，选择最优恢复选项
   - 演示时可提前说明"这是演示剧本，实际游戏事件随机"

3. **时间控制**：
   - 超时处理：在第9步后直接使用控制台命令触发结局
   - 压缩方案：合并步骤2-3，快速推进
