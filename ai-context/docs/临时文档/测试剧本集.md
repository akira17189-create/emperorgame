# 靖朝游戏测试剧本集（10个）

## 剧本设计原则

1. **覆盖全面**：涵盖游戏核心系统（政策、事件、结局、基础流程）
2. **目标明确**：每个剧本聚焦测试特定功能或结局
3. **操作具体**：提供可执行的玩家指令序列
4. **验收清晰**：明确验证要点和期望结果
5. **难度梯度**：从简单到复杂，逐步深入

---

## 一、正常游戏流程测试（2个）

### 剧本01：新手入门流程

**剧本名称**：靖朝元年新政  
**主要测试目标**：验证基础游戏循环、Tick系统、资源管理

**初始状态描述**：
- 游戏年份：靖朝元年
- 资源初始值：
  - 士气（morale）：60
  - 财政（fiscal）：3000
  - 军力（military）：500
  - 粮食（food）：2000
  - 外患（threat）：20
  - 商业（commerce）：100
  - 宦官（eunuch）：30
  - 党争（faction）：40
- 皇帝年龄：25岁

**玩家操作序列**（5步）：
1. `查看国库` - 了解当前财政状况
2. `减税惠民` - 颁布预设政策，提升民心
3. `等待一年` - 执行Tick，观察政策效果
4. `兴建水利` - 颁布农业政策，增加粮食产量
5. `查看史册` - 验证事件记录功能

**期望触发的事件**：
- 政策执行反馈（叙事栏显示政策效果）
- Tick执行后的资源变化
- 史册记录政策颁布

**验收要点**：
1. ✅ 政策面板能正确显示预设政策
2. ✅ 颁布政策后资源数值实时更新
3. ✅ Tick系统正常推进时间（年份+1）
4. ✅ 叙事栏显示政策执行描述
5. ✅ 史册页面记录政策颁布事件

## 四、结局触发测试（4个）

### 剧本07：盛世中兴结局

**剧本名称**：明君治世五十年  
**主要测试目标**：验证正面结局触发条件、长线游戏稳定性

**初始状态描述**：
- 游戏年份：靖朝元年
- 资源初始值：标准中等水平
- 特殊条件：无，从零开始

**玩家操作序列**（目标导向）：
1. 前10年：稳定发展期
   - `减税惠民`、`兴建水利`、`整顿吏治`
   - 保持士气>70，财政稳步增长
2. 中间20年：强国期
   - `推广活字印刷`、`开设钱庄`、`官道驿路`
   - 商业>500，外患<20
3. 最后20年：盛世期
   - 维持现有政策，避免高风险决策
   - 确保士气>80，财政>8000，外患<15
4. 第30年后：等待结局触发

**关键操作节点**：
- 第15年：`和亲邦交`降低外患
- 第25年：`裁汰冗兵`平衡财政
- 第35年：保持稳定，避免动荡

**期望触发的结局**：盛世中兴（E1）

**验收要点**：
1. ✅ 执政满30年自动检查结局条件
2. ✅ 满足条件时触发结局叙事
3. ✅ 结局叙事引用玩家实际经历的事件
4. ✅ 游戏正常结束，可查看完整史册
5. ✅ 结局评分系统正常工作

## 五、测试执行指南

1. 先执行剧本01-02（基础流程）
2. 再执行剧本03-04（政策系统）
