# 第一章：新帝登基·三方试探 - 史册文本示例

## 场景一：登基大典·暗流涌动

**永德元年正月，新帝登基。**

太和殿外，百官肃立。十八岁的皇帝身着十二章服，在司礼监王公公的引导下踏上丹陛。阳光透过云层，在龙椅投下斑驳光影。户部尚书李大人垂首而立，余光却扫向兵部尚书张将军——后者按剑的手指节发白。

**史册记录（156字）**
> 永德元年正月初一，帝践祚于太和殿。礼成，百官贺。户部李尚书奏曰："陛下初登大宝，当以休养为务。今国库仅存银六千两，粮千二百石，若兴兵事，民必困矣。"兵部张将军立驳："边关烽火三日一报，北狄骑兵已至雁门。此时言休养，无异开门揖盗！"帝默然片刻，曰："容朕思之。"王公公奉茶近前，低语："两位大人皆忠臣，然国库空虚亦是实情。"

## 场景二：御书房·第一道旨意

**三日后，御书房。**

皇帝展开边疆地图，墨迹未干的"整军诏"搁在案头。郑经奉茶而入，这位翰林院侍讲学士虽官职不高，却因边防策论深得帝心。

**史册记录（198字）**
> 初四，帝召翰林院侍讲学士郑经问策。郑经指图言："雁门关守军三千，然盔甲朽坏者过半。北狄轻骑来去如风，当以坚壁清野为要。"帝问所需，对曰："若拨银二千两修缮关隘，再调京营三千人协防，可保今岁无虞。"帝颔首，遂拟旨："着兵部即刻整饬雁门关防务，拨银二千两。"旨意传出，李尚书连夜求见："陛下！二千两已占国库三成，若全用于兵事，春耕赈济将无银可支！"张将军则于宫门外长揖："陛下圣明！"

## 场景三：朝会交锋·三方博弈

**五日后，例行朝会。**

皇帝端坐龙椅，下方三位重臣各怀心思。王公公侍立帝侧，目光在三人间流转。

**史册记录（217字）**
> 初九朝会，李尚书持户部账簿出列："臣核计边关用度，二千两足可支撑半年。然今春三省旱情已显，若不开仓放粮，恐生民变。"张将军怒曰："李大人只知算账，可知北狄破关是何后果？届时岂止民变，乃是国破！"郑经忽奏："臣有一策：二千两分作两份，一千五百两用于关防，五百两购粮种分与边民。令其春耕自足，既减朝廷运粮之耗，又固边防人心。"王公公适时进言："郑学士此议老成谋国。只是...五百两购粮种，至多惠及千户，于三省旱情不过杯水车薪。"帝沉吟良久，最终朱批："准郑经所奏，着即施行。"

## 场景四：旨意下达·余波荡漾

**旨意传出宫门，三方反应各异。**

李尚书回府后闭门半日，终是提笔写下一封"乞骸骨"奏疏。张将军虽不满款项削减，仍连夜点兵奔赴雁门。郑经则被王公公"请"至司礼监值房。

**史册记录（189字）**
> 旨意既下，朝野震动。李尚书幕僚劝曰："大人何必求去？陛下分明采纳了您的谏言。"李叹："非也。陛下用郑经之策，是知兵事不可废，民困不可忽。然五百两济旱实属无奈——这说明国库比我所奏更空虚。老夫掌户部十载，竟使国家至此，还有何颜面立于朝堂？"张将军离京前，特至翰林院寻郑经："小子听着，若你的粮种策无效，本将军回来第一个找你！"郑经躬身："下官愿以性命担保。"王公公于窗后目睹一切，轻笑："这局棋，才刚开始呢。"

## 史册文本格式参考

### 标准格式要素：
1. **时间标记**：精确到日（永德元年正月初一）
2. **场景描述**：简洁的环境描写（太和殿外，百官肃立）
3. **对话引用**：关键人物发言，体现立场与性格
4. **动作细节**：微妙的肢体语言（按剑的手指节发白）
5. **决策影响**：暗示后续发展（这局棋，才刚开始呢）

### 字数控制：
- 单一场景：150-200字
- 复杂交锋：200-250字  
- 关键转折：不超过300字

### 叙事节奏：
- 开场：环境+人物登场（20-30字）
- 发展：矛盾呈现+言语交锋（80-120字）
- 高潮：决策时刻+各方反应（40-60字）
- 结尾：余波+伏笔（20-30字）
